# T16 青训学院 - 详细实现方案

> 任务编号：T16
> 周期：W56-W58（3 周）
> 优先级：P0（C 轨经营系统收尾，依赖 T09 成长月结）
> 依据：V0.1 `08_青训_球探_历史新星系统.md` §二青训学院 + V0.2 `08_球员成长_训练_伤病细化算法.md`
> 铁律：青训产出 7 因子公式 + 8 种青训风格 + 6 种青训状态 + 8 种青训操作 + 6 种青训事件 + 异常保护防"小妖全部满潜"
> 定位：长程经营核心闭环，决定玩家是否能"自给自足"地培养球星，与 T15 历史新星池互补（自产 vs 外挖）

---

## 一、业务目标与范围

### 1. 业务目标

- 让玩家通过投资青训获得长期回报（5-8 年成才周期）
- 与历史新星池形成"自产 + 外挖"双轨球员来源
- 青训风格影响产出位置分布，体现俱乐部差异化打法
- 异常保护：避免每个青训球员都达到满潜，让兑现率合理分布

### 2. V1 范围

**V1 做**：
- 9 项青训配置 + 8 种青训风格 + 7 因子产出质量公式
- U18 / U21 两级青训梯队（V0.1 03 文档规定的页面结构）
- 8 种青训操作（签青年合同 / 签职业合同 / 重点培养 / 训练方向 / 导师安排 / 提拔一线队 / 外租培养 / 放弃培养）
- 6 种青训事件（黄金一代 / 豪门挖角 / 教练推荐 / 态度下降 / 国青入选 / 要求职业合同）
- 月度青训球员生成（按俱乐部青训产出质量）
- 青训球员成长复用 T0Y 月度成长公式（青训球员走年龄档 14-19 岁高成长区）
- 投资升级（青训等级 / 训练设施 / 招募范围 / 教练质量）
- 青训球员提拔至一线队 / 外租 / 放弃的全流程

**V1 不做**（推迟 V2）：
- U17 / U19 / U23 三级梯队细分（V1 用 U18 / U21 两级）
- 青训球员详细个人故事 / 性格事件链
- 青训学院自建训练基地可视化
- 青训教练雇佣系统（V1 走 club 配置，不走 staff 表）
- 青训球探与一线球探分离（V1 共用 T14 球探系统）

### 3. 关键设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 青训球员存哪 | save.db `youth_player` 表 | 青训球员是存档动态生成，不属于 history 库静态数据 |
| 青训球员何时转一线队 | 满足 18 岁 + 签职业合同 + 提拔操作 | 符合现实足球规则 |
| 月度生成还是赛季生成 | 月度概率生成 | 与 T07 每日推进月结任务对齐 |
| 青训球员成长复用 T0Y | 是 | 复用月度成长 10 因子公式 + 异常保护 |
| 青训风格是否可切换 | 可切换但有冷却（3 个月） | 避免玩家反复切换刷位置 |
| 历史新星是否走青训流程 | 否 | 历史新星走 T15 独立路径，与青训解耦 |

## 四、核心类设计

### 1. YouthAcademyManager（青训学院管理 + 投资升级）

```kotlin
// academy/YouthAcademyManager.kt
class YouthAcademyManager(
    private val saveDb: SaveDatabase,
    private val config: YouthAcademyConfig,
    private val clubService: ClubService                // T01/T03
) {

    /**
     * V0.1 08 §二.1 青训配置
     * 初始化俱乐部青训配置（开档时调用）
     * 根据俱乐部所在国家设置 nation_talent_pool_bonus
     */
    suspend fun initializeAcademy(saveId: Int, clubId: Int): YouthAcademyEntity {
        val club = clubService.getClub(saveId, clubId)
        val nationBonus = config.getNationTalentPoolBonus(club.countryCode)

        val academy = YouthAcademyEntity(
            saveId = saveId,
            clubId = clubId,
            youthLevel = club.youthLevel,                 // 来自 club 表初始值
            trainingFacility = club.trainingLevel,
            recruitmentRange = RecruitmentRange.LOCAL,
            academyReputation = config.getInitialReputation(club.reputation),
            academyStyle = config.getDefaultStyle(club.countryCode),
            monthlyBudget = config.getDefaultBudget(club.reputation),
            u18CoachQuality = 50,
            u21CoachQuality = 50,
            nationTalentPoolBonus = nationBonus
        )

        val id = saveDb.youthAcademyDao().upsert(academy)
        return academy.copy(academyId = id.toInt())
    }

    /**
     * 切换青训风格
     * @return true 切换成功，false 在冷却期
     */
    suspend fun changeStyle(saveId: Int, clubId: Int, newStyle: AcademyStyle): Boolean {
        val academy = saveDb.youthAcademyDao().getByClub(saveId, clubId) ?: return false

        // 冷却期检查
        if (academy.styleChangeCooldown > 0) return false

        saveDb.youthAcademyDao().upsert(
            academy.copy(
                academyStyle = newStyle,
                styleChangeCooldown = config.styleChangeCooldownMonths    // 默认 3 个月
            )
        )
        return true
    }

    /**
     * V0.1 03 §二.2 投资升级
     * 升级成本随等级递增：base_cost × (current_level / 10)^1.5
     */
    suspend fun invest(
        saveId: Int,
        clubId: Int,
        field: InvestmentField,
        currentDate: String
    ): InvestmentResult {
        val academy = saveDb.youthAcademyDao().getByClub(saveId, clubId)
            ?: return InvestmentResult.failed("青训学院未初始化")

        val (currentLevel, maxLevel) = when (field) {
            InvestmentField.YOUTH_LEVEL -> academy.youthLevel to 100
            InvestmentField.TRAINING_FACILITY -> academy.trainingFacility to 100
            InvestmentField.RECRUITMENT_RANGE -> academy.recruitmentRange.ordinal to 3
            InvestmentField.U18_COACH -> academy.u18CoachQuality to 100
            InvestmentField.U21_COACH -> academy.u21CoachQuality to 100
        }

        if (currentLevel >= maxLevel) {
            return InvestmentResult.failed("已达最高等级")
        }

        val cost = calculateInvestmentCost(field, currentLevel)
        val clubFinance = clubService.getFinance(saveId, clubId)

        if (clubFinance.balance < cost) {
            return InvestmentResult.failed("资金不足，需要 ${formatMoney(cost)}")
        }

        // 扣款 + 升级
        clubService.deductFunds(saveId, clubId, cost, "青训投资：${field.displayName}")
        val updated = applyInvestment(academy, field, currentLevel + 1)
        saveDb.youthAcademyDao().upsert(updated)

        // 记录投资历史
        saveDb.youthAcademyInvestmentDao().insert(
            YouthAcademyInvestmentEntity(
                saveId = saveId,
                clubId = clubId,
                investField = field,
                levelBefore = currentLevel,
                levelAfter = currentLevel + 1,
                cost = cost,
                investDate = currentDate
            )
        )

        return InvestmentResult.success(cost, currentLevel + 1)
    }

    /**
     * 投资成本公式：base × (current_level / 10)^1.5
     * 招募范围升级是阶梯式（LOCAL→REGIONAL→NATIONAL→INTERNATIONAL）
     */
    private fun calculateInvestmentCost(field: InvestmentField, currentLevel: Int): Int {
        val baseCost = config.investmentBaseCost[field] ?: 500_000
        return if (field == InvestmentField.RECRUITMENT_RANGE) {
            // 阶梯式：50万 / 200万 / 500万 / 1000万
            config.recruitmentRangeCosts[currentLevel]
        } else {
            (baseCost * Math.pow(currentLevel / 10.0, 1.5)).toInt()
        }
    }

    private fun applyInvestment(
        academy: YouthAcademyEntity,
        field: InvestmentField,
        newLevel: Int
    ): YouthAcademyEntity {
        return when (field) {
            InvestmentField.YOUTH_LEVEL -> academy.copy(youthLevel = newLevel)
            InvestmentField.TRAINING_FACILITY -> academy.copy(trainingFacility = newLevel)
            InvestmentField.RECRUITMENT_RANGE -> academy.copy(
                recruitmentRange = RecruitmentRange.values()[newLevel]
            )
            InvestmentField.U18_COACH -> academy.copy(u18CoachQuality = newLevel)
            InvestmentField.U21_COACH -> academy.copy(u21CoachQuality = newLevel)
        }
    }

    /**
     * 每月扣除青训预算 + 风格冷却递减（由 T07 月结调用）
     */
    suspend fun processMonthly(saveId: Int, clubId: Int) {
        val academy = saveDb.youthAcademyDao().getByClub(saveId, clubId) ?: return

        // 扣月度预算
        clubService.deductFunds(saveId, clubId, academy.monthlyBudget, "青训月度运营")

        // 冷却递减
        if (academy.styleChangeCooldown > 0) {
            saveDb.youthAcademyDao().updateStyleCooldown(
                academy.academyId, academy.styleChangeCooldown - 1
            )
        }
    }
```

```kotlin
// generator/YouthPlayerGenerator.kt
class YouthPlayerGenerator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val productionCalculator: ProductionQualityCalculator,
    private val positionWeightTable: PositionWeightTable,
    private val nameGenerator: PlayerNameGenerator,        // 复用 T01
    private val config: YouthAcademyConfig
) {

    /**
     * V0.1 08 §二.3 青训产出公式
     * 月度生成入口（由 T07 月结调用）
     * 生成数量受梯队容量限制
     */
    suspend fun generateMonthly(
        saveId: Int,
        clubId: Int,
        currentDate: LocalDate
    ): List<YouthPlayerEntity> {
        val academy = saveDb.youthAcademyDao().getByClub(saveId, clubId)
            ?: return emptyList()

        // 1. 检查是否本月已生成
        val monthKey = currentDate.format(YYYY_MONTH)
        if (academy.lastProductionDate == monthKey) return emptyList()

        // 2. 检查梯队容量
        val u18Count = saveDb.youthPlayerDao().countByTier(saveId, clubId, YouthTier.U18)
        val u21Count = saveDb.youthPlayerDao().countByTier(saveId, clubId, YouthTier.U21)
        if (u18Count >= config.maxU18Capacity && u21Count >= config.maxU21Capacity) {
            return emptyList()
        }

        // 3. 计算产出质量分数
        val productionScore = productionCalculator.calculate(academy, saveId, clubId)

        // 4. 决定是否本月生成（概率性，避免每月都生成）
        val generationProbability = calculateGenerationProbability(productionScore, academy)
        if (Random.nextDouble() > generationProbability) {
            markProductionDate(academy, monthKey)
            return emptyList()
        }

        // 5. 生成 1-2 名球员（高分学院有几率生成多名）
        val generateCount = calculateGenerationCount(productionScore, u18Count, u21Count)
        val newPlayers = mutableListOf<YouthPlayerEntity>()

        repeat(generateCount) {
            val player = generateSinglePlayer(
                saveId, clubId, academy, productionScore, currentDate
            )
            if (player != null) {
                val id = saveDb.youthPlayerDao().upsert(player)
                newPlayers.add(player.copy(youthPlayerId = id.toInt()))

                // 同时在 save_player_state 表插入对应记录（用于比赛/统计）
                syncToSavePlayerState(player)
            }
        }

        markProductionDate(academy, monthKey)
        return newPlayers
    }

    /**
     * V0.1 08 §二.3 青训产出公式（7 因子）
     * production_score =
     *   youth_level × 0.25
     * + training_facility × 0.20
     * + academy_manager × 0.15        # V1 用 u21_coach_quality 兜底
     * + recruitment_range × 0.15
     * + club_reputation × 0.10
     * + nation_talent_pool × 0.10
     * + random_genius_probability × 0.05
     */
    private fun calculateGenerationProbability(score: Double, academy: YouthAcademyEntity): Double {
        // score 范围 0-100，转化为概率 0.0-0.8
        // 高分学院每月生成概率高，但有上限
        val baseProbability = (score / 100.0) * 0.7
        return baseProbability.coerceIn(0.05, 0.8)
    }

    private fun calculateGenerationCount(score: Double, u18Count: Int, u21Count: Int): Int {
        // 高分学院有几率生成 2 名
        return when {
            score >= 85 && u18Count + u21Count < 5 && Random.nextDouble() < 0.3 -> 2
            else -> 1
        }
    }

    /**
     * 生成单个青训球员
     */
    private suspend fun generateSinglePlayer(
        saveId: Int,
        clubId: Int,
        academy: YouthAcademyEntity,
        productionScore: Double,
        currentDate: LocalDate
    ): YouthPlayerEntity? {
        // 1. 决定梯队（年龄）
        val age = pickAge(academy)
        val tier = if (age <= 17) YouthTier.U18 else YouthTier.U21

        // 2. 决定位置（按风格权重）
        val position = positionWeightTable.pickPosition(academy.academyStyle)
        val altPositions = positionWeightTable.pickAlternativePositions(position)

        // 3. 决定国籍（受招募范围影响）
        val nationality = pickNationality(academy)

        // 4. 生成姓名
        val playerName = nameGenerator.generate(nationality)

        // 5. 计算 PA（受 production_score 影响，但有异常保护）
        val (initialPa, isGenius) = generatePotentialPa(productionScore, age, academy)
        val currentCa = generateInitialCa(age, initialPa, productionScore)

        // 6. 生成隐藏属性
        val professionalism = rollAttribute(isGenius, weightedBias = 60)
        val ambition = rollAttribute(isGenius, weightedBias = 55)
        val injuryProneness = rollAttribute(isGenius, weightedBias = 30, reverse = true)
        val adaptability = rollAttribute(isGenius, weightedBias = 50)

        // 7. 生成隐藏标签
        val hiddenTags = generateHiddenTags(isGenius, professionalism, age)

        // 8. 生成属性 JSON
        val attributes = generateAttributesByPosition(position, currentCa, academy.academyStyle)

        return YouthPlayerEntity(
            saveId = saveId,
            academyId = academy.academyId,
            clubId = clubId,
            playerId = 0,                    // 待 sync 后回填
            playerName = playerName,
            nationality = nationality,
            birthDate = currentDate.minusYears(age).toString(),
            primaryPosition = position,
            alternativePositions = altPositions,
            tier = tier,
            status = if (age < 17) YouthPlayerStatus.YOUTH_CONTRACT else YouthPlayerStatus.YOUTH_CONTRACT,
            currentCa = currentCa,
            potentialPa = initialPa,
            initialPa = initialPa,
            professionalism = professionalism,
            ambition = ambition,
            injuryProneness = injuryProneness,
            adaptability = adaptability,
            generatedDate = currentDate.toString(),
            attributesJson = attributes,
            hiddenTags = hiddenTags
        )
    }

    /**
     * V0.2 §十五 成长异常保护
     * 防止"小妖全部满潜"问题
     */
    private suspend fun generatePotentialPa(
        productionScore: Double,
        age: Int,
        academy: YouthAcademyEntity
    ): Pair<Int, Boolean> {
        // 1. 基础 PA 范围：30-80，受 production_score 影响
        val basePa = 30 + (productionScore / 100.0 * 50).toInt()    // 30-80

        // 2. 天才概率（5% 因子）
        val geniusProbability = config.geniusProbability * (1 + productionScore / 200.0)
        val isGenius = Random.nextDouble() < geniusProbability

        // 3. 天才球员 PA 可达 90-99
        val pa = if (isGenius) {
            Random.nextInt(90, 100)
        } else {
            // 普通球员：30-80，正态分布
            normalRandomInt(basePa - 10, basePa + 10, mean = basePa, stdDev = 8)
                .coerceIn(30, 85)
        }

        // 4. 异常保护：单俱乐部每月最多 1 名天才
        if (isGenius) {
            val recentGeniusCount = saveDb.youthPlayerDao()
                .countGeniusThisMonth(academy.saveId, academy.clubId)
            if (recentGeniusCount >= config.maxGeniusPerMonth) {
                return Pair(pa.coerceAtMost(85), false)
            }
        }

        return Pair(pa, isGenius)
    }

    /**
     * 初始 CA：年龄越小 CA 越低
     * 14 岁：20-35
     * 15 岁：25-40
     * 16 岁：30-45
     * 17 岁：35-55
     * 18-19 岁：40-60
     */
    private fun generateInitialCa(age: Int, pa: Int, productionScore: Double): Int {
        val ageBase = when (age) {
            14 -> Random.nextInt(20, 35)
            15 -> Random.nextInt(25, 40)
            16 -> Random.nextInt(30, 45)
            17 -> Random.nextInt(35, 55)
            18 -> Random.nextInt(40, 60)
            else -> Random.nextInt(45, 65)
        }
        // CA 不能超过 PA
        return ageBase.coerceAtMost(pa - 10)
    }

    private fun pickAge(academy: YouthAcademyEntity): Int {
        // 主要生成 14-17 岁（U18）
        // 偶尔生成 18-19 岁（U21）
        val roll = Random.nextDouble()
        return when {
            roll < 0.30 -> 14
            roll < 0.55 -> 15
            roll < 0.80 -> 16
            roll < 0.92 -> 17
            else -> 18
        }
    }

    private suspend fun pickNationality(academy: YouthAcademyEntity): String {
        // 招募范围影响国籍分布
        // LOCAL：90% 本国 + 10% 邻国
        // REGIONAL：70% 本国 + 30% 区域
        // NATIONAL：80% 本国 + 20% 全国分布
        // INTERNATIONAL：50% 本国 + 50% 全球
        val clubCountry = clubService.getClubCountry(academy.clubId)
        val localRatio = when (academy.recruitmentRange) {
            RecruitmentRange.LOCAL -> 0.90
            RecruitmentRange.REGIONAL -> 0.70
            RecruitmentRange.NATIONAL -> 0.80
            RecruitmentRange.INTERNATIONAL -> 0.50
        }
        return if (Random.nextDouble() < localRatio) {
            clubCountry
        } else {
            config.getForeignNationalityPool(academy.recruitmentRange, clubCountry)
                .random()
        }
    }

    private fun rollAttribute(
        isGenius: Boolean,
        weightedBias: Int,
        reverse: Boolean = false
    ): Int {
        val base = if (isGenius) {
            // 天才球员职业态度偏高（避免天才态度差导致浪费）
            if (!reverse) normalRandomInt(60, 90, mean = 75, stdDev = 10)
            else normalRandomInt(20, 60, mean = 40, stdDev = 12)
        } else {
            // 普通球员正态分布
            if (!reverse) normalRandomInt(30, 90, mean = weightedBias, stdDev = 15)
            else normalRandomInt(10, 80, mean = weightedBias, stdDev = 15)
        }
        return base.coerceIn(1, 100)
    }

    private fun generateHiddenTags(
        isGenius: Boolean,
        professionalism: Int,
        age: Int
    ): List<String> {
        val tags = mutableListOf<String>()
        if (isGenius) tags.add("GENIUS")
        if (professionalism >= 80) tags.add("HIGH_PROFESSIONALISM")
        if (professionalism < 30) tags.add("LAZY")
        if (age <= 15 && isGenius) tags.add("EARLY_BLOOMER")
        if (age >= 18 && !isGenius) tags.add("LATE_BLOOMER")
        return tags
    }
}
```

```kotlin
// generator/ProductionQualityCalculator.kt
class ProductionQualityCalculator(
    private val clubService: ClubService,
    private val config: YouthAcademyConfig
) {

    /**
     * V0.1 08 §二.3 青训产出公式（7 因子）
     * production_score =
     *   youth_level × 0.25
     * + training_facility × 0.20
     * + academy_manager × 0.15        # V1 用 (u18_coach + u21_coach) / 2 兜底
     * + recruitment_range × 0.15
     * + club_reputation × 0.10
     * + nation_talent_pool × 0.10
     * + random_genius_probability × 0.05
     *
     * @return 0-100 分数
     */
    suspend fun calculate(
        academy: YouthAcademyEntity,
        saveId: Int,
        clubId: Int
    ): Double {
        val club = clubService.getClub(saveId, clubId)

        // 1. 青训等级 (0-100)
        val youthLevel = academy.youthLevel.toDouble()

        // 2. 训练设施 (0-100)
        val trainingFacility = academy.trainingFacility.toDouble()

        // 3. 青训主管（V1 用 U18 + U21 教练质量均值兜底）
        val academyManager = (academy.u18CoachQuality + academy.u21CoachQuality) / 2.0

        // 4. 招募范围（LOCAL=25, REGIONAL=50, NATIONAL=75, INTERNATIONAL=100）
        val recruitmentRange = when (academy.recruitmentRange) {
            RecruitmentRange.LOCAL -> 25.0
            RecruitmentRange.REGIONAL -> 50.0
            RecruitmentRange.NATIONAL -> 75.0
            RecruitmentRange.INTERNATIONAL -> 100.0
        }

        // 5. 俱乐部声望 (0-100)
        val clubReputation = club.reputation.toDouble()

        // 6. 国家人才池加成 (0-100)
        val nationTalentPool = academy.nationTalentPoolBonus.toDouble()

        // 7. 随机天才概率（0-1，乘以 100 转分）
        val randomGenius = Random.nextDouble() * 100.0

        val w = config.productionWeights
        val score = (
            youthLevel * w.youthLevel +
            trainingFacility * w.trainingFacility +
            academyManager * w.academyManager +
            recruitmentRange * w.recruitmentRange +
            clubReputation * w.clubReputation +
            nationTalentPool * w.nationTalentPool +
            randomGenius * w.randomGenius
        )

        return score.coerceIn(0.0, 100.0)
    }
}
```

```kotlin
class PositionWeightTable(private val config: YouthAcademyConfig) {

    /**
     * 按风格权重随机抽取位置
     */
    fun pickPosition(style: AcademyStyle): String {
        val weights = style.positionWeights
        val totalWeight = weights.values.sum()
        var roll = Random.nextDouble() * totalWeight

        for ((pos, weight) in weights) {
            roll -= weight
            if (roll <= 0) return pos
        }
        return weights.keys.first()
    }

    /**
     * 选择副位置（同位置族）
     */
    fun pickAlternativePositions(primary: String): List<String> {
        val group = when (primary) {
            "ST", "CF" -> listOf("ST", "CF", "LW", "RW")
            "LW", "RW" -> listOf("LW", "RW", "AM", "ST")
            "AM", "CM" -> listOf("AM", "CM", "DM", "LW", "RW")
            "DM" -> listOf("DM", "CM", "CB")
            "CB" -> listOf("CB", "DM")
            "LB", "RB" -> listOf("LB", "RB", "LWB", "RWB")
            "GK" -> listOf("GK")
            else -> listOf(primary)
        }
        return group.filter { it != primary }.shuffled().take(2)
    }
}
```

```kotlin
class YouthDevelopmentCalculator(
    private val monthlyGrowthCalculator: MonthlyGrowthCalculator,    // T0Y
    private val mentorHandler: MentorInfluenceHandler,               // T0Y

     * 性能要求：全队青训球员 ≤500ms
     */
    suspend fun processMonthlyGrowth(
        saveId: Int,
        clubId: Int,
        currentDate: LocalDate
    ): List<YouthGrowthResult> {
        val academy = saveDb.youthAcademyDao().getByClub(saveId, clubId)
            ?: return emptyList()

        val youthPlayers = saveDb.youthPlayerDao().getByClub(saveId, clubId)
        val results = mutableListOf<YouthGrowthResult>()

        // 计算青训训练质量（用于 T0Y training_quality_factor）
        val trainingQuality = calculateYouthTrainingQuality(academy)
        val u18CoachLevel = academy.u18CoachQuality
        val u21CoachLevel = academy.u21CoachQuality

        for (player in youthPlayers) {
            // 跳过已提拔 / 外租 / 离队的球员
            if (player.status == YouthPlayerStatus.FIRST_TEAM ||
                player.status == YouthPlayerStatus.LOANED_OUT ||
                player.status == YouthPlayerStatus.LEAVING) continue

            val age = calculateAge(player.birthDate)
            val coachLevel = if (player.tier == YouthTier.U18) u18CoachLevel else u21CoachLevel

            // 1. 构造 T0Y 需要的输入
            val monthlyStats = getYouthMonthlyStats(player)
            val tacticalFit = 0.7    // 青训战术适配默认 0.7
            val leagueIntensity = 0.5 // 青训不踢职业联赛

            // 2. 调用 T0Y 月度成长公式
            val growthResult = monthlyGrowthCalculator.calculate(
                player = player.toGrowthInput(),
                monthlyStats = monthlyStats,
                clubTrainingQuality = trainingQuality,
                clubCoachLevel = coachLevel,
                leagueIntensity = leagueIntensity,
                tacticalFitScore = tacticalFit
            )

            // 3. 应用训练专项
            val specializationChanges = specializationHandler.applySpecialization(
                player.toGrowthInput(),
                TrainingProgram(TrainingFocus.valueOf(player.trainingFocus), intensity = 5),
                trainingQuality
            )

            // 4. 应用导师影响
            val mentorEffect = player.mentorPlayerId?.let { mentorId ->
                val mentor = saveDb.savePlayerStateDao().getById(saveId, mentorId)
                mentor?.let { mentorHandler.calculateMentorEffect(player.toGrowthInput(), it) }
            }

            // 5. V0.2 §十五 异常保护（青训版）
            val protectedDelta = applyYouthAnomalyProtection(
                growthResult, player, age, mentorEffect
            )

            // 6. 应用成长
            val newCa = (player.currentCa + protectedDelta.caDelta).coerceIn(1, player.potentialPa)
            saveDb.youthPlayerDao().updateCa(player.youthPlayerId, newCa)

            // 7. 更新属性
            applyAttributeChanges(player.youthPlayerId, protectedDelta.attributeChanges)

            // 8. 自动升级梯队（17 岁 → U21）
            if (age >= 18 && player.tier == YouthTier.U18) {
                saveDb.youthPlayerDao().update(
                    player.copy(tier = YouthTier.U21)
                )
            }

            // 9. 记录成长日志
            saveGrowthLog(player, protectedDelta, currentDate)

            results.add(YouthGrowthResult(
                youthPlayerId = player.youthPlayerId,
                playerName = player.playerName,
                caBefore = player.currentCa,
                caAfter = newCa,
                caDelta = protectedDelta.caDelta,
                realizationScore = growthResult.realizationScore
            ))
        }

        return results
    }

    /**
     * V0.2 §十五 异常保护（青训版）
     * 防止"小妖全部满潜"
     */
    private fun applyYouthAnomalyProtection(
        result: GrowthResult,
        player: YouthPlayerEntity,
        age: Int,
        mentorEffect: MentorEffect?
    ): GrowthResult {
        var caDelta = result.caDelta

        // 1. 单月成长上限（青训球员 +1 至 +3）
        val maxMonthlyGrowth = when (age) {
            in 14..15 -> 2
            in 16..17 -> 3
            else -> 3
        }
        if (caDelta > maxMonthlyGrowth) caDelta = maxMonthlyGrowth

        // 2. 接近 PA 时成长放缓
        val gapToPotential = player.potentialPa - player.currentCa
        caDelta = when {
            gapToPotential <= 0 -> 0
            gapToPotential <= 3 -> caDelta / 3
            gapToPotential <= 5 -> caDelta / 2
            else -> caDelta
        }

        // 3. 低职业态度降低成长（异常保护）
        if (player.professionalism < 30) {
            caDelta = (caDelta * 0.5).toInt()
        }

        // 4. 导师加成（仅正向）
        if (mentorEffect != null && mentorEffect.isPositive) {
            caDelta = (caDelta * (1 + mentorEffect.growthBonus)).toInt()
        }

        return result.copy(caDelta = caDelta.coerceAtLeast(0))
    }

    /**
     * 青训训练质量（用于 T0Y training_quality_factor）
     * training_quality = 0.70 + (training_facility / 100) * 0.40
     * 范围 0.70 - 1.10
     * 青训设施对 18 岁以下额外加成
     */
    private fun calculateYouthTrainingQuality(academy: YouthAcademyEntity): Double {
        val base = 0.70 + (academy.trainingFacility / 100.0) * 0.40
        return base.coerceIn(0.70, 1.10)
    }

    /**
     * 青训球员出场时间统计（U18/U21 不踢正式比赛，默认 0）
     * 若被外租则统计外租出场时间
     */
    private fun getYouthMonthlyStats(player: YouthPlayerEntity): MonthlyPlayerStats {
        return if (player.status == YouthPlayerStatus.LOANED_OUT) {
            saveDb.loanStatsDao().getMonthlyStats(player.youthPlayerId)
        } else {
            MonthlyPlayerStats(minutesPlayed = 0, maxPossibleMinutes = 0)
        }
    }
}
```

```kotlin
class YouthAcademyService(
    private val academyManager: YouthAcademyManager,
    private val playerGenerator: YouthPlayerGenerator,
    private val developmentCalculator: YouthDevelopmentCalculator,
    private val operationHandler: YouthOperationHandler,
    private val eventTrigger: YouthEventTrigger,
    private val saveDb: SaveDatabase
) {

    /**
     * 由 T07 MonthlyTaskExecutor 月结调用
     * 性能要求：全队青训球员 ≤500ms
     */
    suspend fun processMonthly(ctx: AdvanceContext): YouthMonthlyResult {
        val results = mutableListOf<YouthGrowthResult>()
        val events = mutableListOf<YouthEventEntity>()

        // 1. 月度预算扣除 + 风格冷却递减
        academyManager.processMonthly(ctx.saveId, ctx.clubId)

        // 2. 青训球员生成（月度概率性）
        val newPlayers = playerGenerator.generateMonthly(
            ctx.saveId, ctx.clubId, ctx.currentDate
        )

        // 3. 青训球员月度成长（复用 T0Y）
        results.addAll(
            developmentCalculator.processMonthlyGrowth(
                ctx.saveId, ctx.clubId, ctx.currentDate
            )
        )

        // 4. 青训事件触发
        events.addAll(
            eventTrigger.checkMonthlyEvents(
                ctx.saveId, ctx.clubId, ctx.currentDate
            )
        )

        return YouthMonthlyResult(
            newPlayers = newPlayers,
            growthResults = results,
            events = events
        )
    }

    /**
     * 由 T19 赛季归档调用：青训球员赛季末处理
     * - 自动升级 U18→U21（18 岁+）
     * - 自动签青年合同（14-17 岁未签的）
     * - 清理离队球员
     */
    suspend fun processSeasonEnd(saveId: Int, clubId: Int, currentDate: LocalDate) {
        val players = saveDb.youthPlayerDao().getByClub(saveId, clubId)

        for (player in players) {
            val age = calculateAge(player.birthDate)

            // 1. 18 岁自动升级 U21
            if (age >= 18 && player.tier == YouthTier.U18) {
                saveDb.youthPlayerDao().update(player.copy(tier = YouthTier.U21))
            }
```

```text
function calculate_production_score(academy, club):
    score =
        academy.youth_level × 0.25
      + academy.training_facility × 0.20
      + ((academy.u18_coach_quality + academy.u21_coach_quality) / 2) × 0.15
      + map_recruitment_range(academy.recruitment_range) × 0.15
      + club.reputation × 0.10
      + academy.nation_talent_pool_bonus × 0.10
      + random(0, 100) × 0.05
    return clamp(score, 0, 100)
```

```text
function generate_youth_player(academy, production_score, current_date):
    age = pick_age_weighted()        # 14-18 岁，14-17 占 92%
```

### 3. 月度成长（青训版，复用 T0Y）

```text
function process_monthly_growth(academy, youth_players, current_date):
    training_quality = 0.70 + (academy.training_facility / 100) × 0.40
    u18_coach = academy.u18_coach_quality
    u21_coach = academy.u21_coach_quality

    for player in youth_players:
        if player.status in [FIRST_TEAM, LOANED_OUT, LEAVING]:
            continue

        age = calculate_age(player.birth_date)
        coach_level = u18_coach if player.tier == U18 else u21_coach
        monthly_stats = get_youth_monthly_stats(player)
        tactical_fit = 0.7
        league_intensity = 0.5

        # 调用 T0Y 10 因子公式
        growth_result = monthly_growth_calculator.calculate(
            player, monthly_stats, training_quality, coach_level,
            league_intensity, tactical_fit
        )

        # 应用训练专项
        specialization_changes = specialization_handler.apply_specialization(
            player, training_program(player.training_focus, intensity=5), training_quality
        )

        # 应用导师影响
        mentor_effect = mentor_handler.calculate_mentor_effect(player, mentor) if player.mentor_id else null

        # 异常保护（青训版）
        ca_delta = growth_result.ca_delta
        ca_delta = min(ca_delta, max_monthly_growth_by_age(age))    # 14-15: 2, 16+: 3
        gap = player.potential_pa - player.current_ca
        if gap <= 0: ca_delta = 0
        elif gap <= 3: ca_delta = ca_delta / 3
        elif gap <= 5: ca_delta = ca_delta / 2

        if player.professionalism < 30:
            ca_delta = ca_delta × 0.5

        if mentor_effect and mentor_effect.is_positive:
            ca_delta = ca_delta × (1 + mentor_effect.growth_bonus)

        ca_delta = max(0, ca_delta)
        new_ca = clamp(player.current_ca + ca_delta, 1, player.potential_pa)

        # 更新数据库
        update_ca(player.id, new_ca)
        apply_attribute_changes(player.id, specialization_changes)

        # 自动升级 U18 → U21
        if age >= 18 and player.tier == U18:
            update_tier(player.id, U21)

        save_growth_log(player, growth_result, current_date)
```

```text
function check_monthly_events(save_id, club_id, current_date):
    events = []

    # 事件 1：黄金一代（5% 概率 + ≥3 名 PA≥80 + 5 年冷却）
    high_pa_players = get_high_potential_players(club_id, min_pa=80)
    if high_pa_players.size >= 3 and not exists_in_recent_years(5):
        if random() < 0.05:
            events.add(golden_generation_event(high_pa_players))

    # 事件 2：被豪门挖角（PA≥85 的球员，每月 5% 概率）
    for player in get_high_potential_players(club_id, min_pa=85):
        if random() < 0.05:
            giant_club = find_giant_club(youth_preference>=70)
            if giant_club:
                events.add(poach_event(player, giant_club))

    # 事件 3：教练强烈推荐（教练质量≥70，10% 概率）
    if academy.u18_coach_quality >= 70 or academy.u21_coach_quality >= 70:
        if random() < 0.10:
            candidates = get_non_key_prospects(club_id, min_pa=75)
            if candidates:
                events.add(coach_recommendation_event(candidates.random()))

    # 事件 4：训练态度下降（职业态度<50，15% 概率）
    for player in get_low_professionalism_players(club_id, threshold=50):
        if random() < 0.15:
            events.add(attitude_drop_event(player))

    # 事件 5：国青入选（CA≥60 + 年龄≤19，8% 概率）
    for player in get_young_high_ca_players(club_id, max_age=19, min_ca=60):
        if random() < 0.08:
            events.add(youth_national_team_event(player))

    # 事件 6：要求职业合同（17 岁+ + CA≥55 + 当前是青年合同，20% 概率）
    for player in get_eligible_pro_contract_request_players(club_id):
        if random() < 0.20:
            events.add(pro_contract_request_event(player))

    for event in events:
        save_event(event)

    return events
```

### 2. 性能优化策略

- **批量读取**：U18 + U21 球员一次性 `getByClub` 读取，避免 N+1 查询
- **索引优化**：`youth_player` 表对 `(save_id, club_id, tier)` 和 `(save_id, status)` 建索引
- **异步任务**：月度成长放到 IO Dispatcher，避免阻塞主线程
- **延迟加载**：球员详情页的属性 JSON 按需解析
- **缓存策略**：青训学院配置读取后缓存至内存，避免重复查询
- **事件去重**：事件触发前先检查本月是否已触发同类型事件

### 3. 数据库查询优化

```kotlin
// 批量获取青训球员（避免 N+1）
@Query("""
    SELECT * FROM youth_player
    WHERE save_id = :saveId AND club_id = :clubId AND tier IN (:tiers)
    ORDER BY tier, potential_pa DESC
""")
suspend fun getByClubAndTiers(saveId: Int, clubId: Int, tiers: List<YouthTier>): List<YouthPlayerEntity>

// 高潜力球员批量获取（事件触发用）
@Query("""
    SELECT * FROM youth_player
    WHERE save_id = :saveId AND club_id = :clubId AND potential_pa >= :minPa
""")
suspend fun getHighPotentialByClub(saveId: Int, clubId: Int, minPa: Int): List<YouthPlayerEntity>
```

## 十四、关键风险与应对

### 1. 小妖全部满潜

- **表现**：青训球员普遍达到 PA 上限，破坏游戏平衡
- **应对**：
  - 单月成长上限（14-15 岁 +2，16+ 岁 +3）
  - 单俱乐部每月最多 1 名天才
  - 低职业态度（<30）成长减半
  - 接近 PA 时成长放缓（gap≤3 时 /3，gap≤5 时 /2）
  - 满潜后强制不再成长
- **红线**：单月成长不超过 3

### 2. 青训球员堆积

- **表现**：青训球员数量爆炸，列表过长
- **应对**：
  - U18 容量上限 25 人
  - U21 容量上限 15 人
  - 21 岁强制决策（提拔 / 外租 / 释放）
  - 月度生成有概率性，非每月必生成

### 3. 风格刷位置

- **表现**：玩家反复切换风格刷特定位置球员
- **应对**：
  - 风格切换 3 个月冷却
  - 切换后不影响已生成球员

### 4. 黄金一代滥用

- **表现**：黄金一代频繁触发，声望暴涨
- **应对**：
  - 5 年冷却期
  - 触发概率仅 5%
  - 公开选项会提升挖角风险（双刃剑）

### 5. 性能不达标

- **表现**：月度成长 >500ms
- **应对**：
  - 批量读取青训球员（避免 N+1）
  - 异步处理（IO Dispatcher）
  - 索引优化（save_id + club_id + tier）

### 6. 与 T0Y 集成偏差

- **表现**：青训球员成长曲线与一线队不一致
- **应对**：
  - 严格复用 MonthlyGrowthCalculator.calculate()，不重写公式
  - 青训特化点仅在于 training_quality / coach_level / league_intensity 输入参数
  - 异常保护叠加在 T0Y 结果之上，不修改 T0Y 内部逻辑
