# 07_经济通胀、身价与工资模型

## 一、设计目标

游戏跨越 1992-2080，如果没有经济模型，会出现：

- 2002 年工资套用到 2020 年不合理。
- 早期球员被现代价格碾压。
- AI 豪门无限囤人。
- 中小球队财政崩盘或异常暴富。

经济系统要解决：

1. 转会费随年代增长。
2. 工资随年代增长。
3. 联赛商业收入随年代增长。
4. 球员身价由能力、年龄、表现、合同、市场环境动态决定。
5. 不同国家、联赛、俱乐部财力差异长期存在。

## 二、时代通胀系数

定义 football_economy_index。

以 2002 年为 1.00。

| 年份 | 经济系数 |
|---:|---:|
| 1992 | 0.35 |
| 1995 | 0.45 |
| 1998 | 0.65 |
| 2002 | 1.00 |
| 2006 | 1.25 |
| 2010 | 1.60 |
| 2014 | 2.20 |
| 2017 | 3.20 |
| 2020 | 3.60 |
| 2024 | 4.20 |
| 2030 | 5.00 |

2030 后进入架空增长，默认年增长 2%-4%，可受全球经济事件影响。

## 三、联赛商业系数

每个联赛有 league_economy_multiplier。

示例：

| 联赛 | 2002 系数 | 特点 |
|---|---:|---|
| 英超 | 1.10 | 后续增长最快 |
| 西甲 | 1.00 | 豪门集中 |
| 意甲 | 1.05 | 2000 前后强，后续可能下滑 |
| 德甲 | 0.85 | 稳定、工资更克制 |
| 法甲 | 0.70 | 后续资本事件可提升 |
| 荷甲 | 0.45 | 青训出口 |
| 葡超 | 0.40 | 黑店体系 |
| 巴甲 | 0.35 | 输出天才 |
| 阿甲 | 0.30 | 输出天才 |

## 四、俱乐部财力计算

```text
club_financial_power =
  club_reputation * 0.25
+ league_economy * 0.25
+ stadium_income * 0.15
+ commercial_income * 0.15
+ owner_investment * 0.15
+ recent_success * 0.05
```

## 五、球员基础身价

```text
base_value =
  ability_value_curve(CA)
* potential_multiplier(PA, age)
* age_multiplier(age)
* position_multiplier(position)
* reputation_multiplier(reputation)
* league_visibility_multiplier(league)
* economy_index(year)
```

### CA 价值曲线

能力越高，价值非线性上升。

```text
CA < 60：低价值
60-70：普通职业球员
70-80：主力级
80-88：明星级
88-94：世界级
95+：历史级，指数增长
```

建议使用指数曲线：

```text
ability_value_curve = base_amount * pow(1.075, CA - 50)
```

## 八、表现修正

```text
performance_multiplier =
  1.0
+ recent_form_bonus
+ season_stats_bonus
+ award_bonus
+ international_bonus
- injury_penalty
- poor_form_penalty
```

限制范围：

```text
0.65 <= performance_multiplier <= 1.60
```

## 十、财政安全线

俱乐部工资/收入比：

| 比例 | 状态 |
|---:|---|
| < 55% | 健康 |
| 55%-70% | 可接受 |
| 70%-85% | 风险 |
| > 85% | 高危 |

AI 俱乐部超过 85% 后：

- 降低买人欲望。
- 提高卖人概率。
- 优先清理高薪替补。
- 董事会可能限制预算。

## 十一、经济事件

经济可被历史事件改变：

- 资本入主：owner_investment 提升。
- 新球场：短期债务上升，长期收入上升。
- 欧冠资格：收入提升。
- 降级：收入下降。
- 财政危机：预算冻结。
- 联赛转播上涨：league_economy 增长。

## 十二、调参指标

每个赛季统计：

- 最高转会费。
- 前 10 转会均价。
- 平均工资。
- 豪门工资/收入比。
- 中游队工资/收入比。
- 自由球员数量。
- 破产风险俱乐部数量。
- 20 岁以下高潜球员均价。

目标：长期模拟后价格上涨，但不无限失控。
