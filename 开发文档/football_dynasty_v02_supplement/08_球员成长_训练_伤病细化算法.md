# 08_球员成长、训练、伤病细化算法

## 一、设计目标

球员成长系统要让玩家感到：

1. 天才需要被发现。
2. 天才需要正确培养。
3. 出场时间重要。
4. 伤病会改变命运。
5. 自律和职业态度影响长期上限。
6. 潜力不是保证，而是机会。

## 二、核心变量

每名球员拥有：

- CA：当前能力
- PA：理论潜力
- growth_rate：成长速度
- professionalism：职业态度
- ambition：野心
- consistency：稳定性
- pressure：抗压
- injury_proneness：伤病倾向
- adaptability：适应力
- peak_age_start：巅峰起点
- peak_age_end：巅峰终点
- decline_rate：下滑速度

## 三、月度成长计算

每月结算一次成长。

```text
monthly_growth =
  base_growth_by_age
* potential_gap_factor
* professionalism_factor
* playing_time_factor
* training_quality_factor
* coach_factor
* league_strength_factor
* morale_factor
* tactical_fit_factor
* injury_penalty
* randomness
```

## 四、年龄基础成长

| 年龄 | 基础成长 |
|---:|---:|
| 14-16 | 0.20 - 0.60 |
| 17-19 | 0.40 - 1.20 |
| 20-23 | 0.25 - 0.90 |
| 24-27 | 0.05 - 0.35 |
| 28-30 | -0.05 - 0.15 |
| 31-33 | -0.10 - -0.35 |
| 34+ | -0.20 - -0.80 |

## 五、潜力差距因子

```text
potential_gap = PA - CA
```

```text
potential_gap_factor = clamp(potential_gap / 30, 0.10, 1.40)
```

潜力越大，成长空间越大。接近 PA 后成长变慢。

## 六、出场时间因子

按年龄不同，出场要求不同。

### 17-23 岁

```text
playing_time_factor =
  0.60 if minutes_ratio < 0.15
  0.85 if 0.15 <= minutes_ratio < 0.35
  1.10 if 0.35 <= minutes_ratio < 0.65
  1.20 if 0.65 <= minutes_ratio < 0.85
  1.05 if minutes_ratio >= 0.85 and fatigue_high
```

年轻球员需要比赛，但过度使用会增加伤病和疲劳。

### 24 岁以上

出场时间主要影响状态维持，而不是大幅成长。

## 七、训练质量因子

```text
training_quality_factor =
  0.70 + (club_training_level / 100) * 0.40
```

范围：0.70 - 1.10。

青训设施对 18 岁以下额外加成。

## 八、职业态度因子

```text
professionalism_factor = 0.70 + professionalism / 100 * 0.60
```

范围：0.70 - 1.30。

高职业态度让球员更容易兑现潜力。

## 九、伤病惩罚

```text
injury_penalty =
  1.0 if healthy
  0.5 if minor_injury_recent
  0.2 if long_injury_recent
  -0.3 if major_injury_recovery
```

重大伤病可能导致：

- CA 下降。
- PA 下降。
- 身体属性永久损失。
- 伤病倾向上升。
- 巅峰期提前结束。

## 十、重大伤病影响

```text
if injury_type == ACL:
    pace -= random(1,4)
    acceleration -= random(1,4)
    injury_proneness += random(3,8)
    PA -= random(0,5) if age < 24
```

医疗水平高可降低损失。

## 十一、训练专项

训练专项不直接提高总 CA，而是影响属性分配。

例如射门专项：

- 射门成长权重提高。
- 远射小幅提高。
- 体能成长降低。
- 过高强度增加肌肉伤概率。

## 十二、导师影响

导师每月对年轻球员产生一次隐藏属性影响。

```text
mentor_effect =
  mentor_leadership * 0.30
+ mentor_professionalism * 0.30
+ relationship_quality * 0.20
+ shared_position_bonus * 0.20
```

可能提升：

- 职业态度
- 稳定性
- 大赛属性
- 领导力
- 位置熟练度

负面导师可能降低自律或引发派系问题。

## 十三、位置改造

位置改造需要时间。

```text
position_training_progress +=
  training_focus * adaptability * coach_quality * playing_time_at_new_position
```

达到阈值后：

- 新位置从“不熟悉”变“可用”。
- 再提升到“熟练”。
- 最后可成为“主位置”。

## 十四、潜力兑现率

每名球员有动态 potential_realization_score。

```text
realization_score =
  professionalism * 0.25
+ playing_time_quality * 0.20
+ training_quality * 0.15
+ coach_quality * 0.10
+ injury_history_score * 0.15
+ morale_stability * 0.05
+ tactical_fit * 0.05
+ pressure_handling * 0.05
```

兑现率影响是否能接近 PA。

## 十五、成长异常保护

为了避免小妖全部满潜：

1. PA 只是理论上限，不是必达。
2. 低职业态度会降低兑现率。
3. 长期替补会浪费成长窗口。
4. 重大伤病会降低身体上限。
5. 过早高压使用会增加心理和伤病风险。
6. 联赛太弱会限制后期成长。

## 十六、测试指标

模拟 20 年后检查：

- 历史级球员数量是否过多。
- 高 PA 球员兑现率是否合理。
- 低级联赛球员是否异常成长为满级。
- 重大伤病是否有足够影响。
- 青训投入是否有可见收益。
