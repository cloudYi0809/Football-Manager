# 04_比赛模拟算法详细设计

## 一、设计目标

比赛模拟必须做到：

1. 结果可信。
2. 比分分布合理。
3. 战术选择有影响。
4. 球星能改变比赛，但不能无敌。
5. 弱队有爆冷空间。
6. 参数可调，方便平衡。
7. 能快速模拟大量 AI 比赛。

## 二、比赛模拟分层

比赛引擎分四层：

1. 赛前评分层：计算双方基础强弱。
2. xG 生成层：把强弱映射为预期进球。
3. 事件生成层：每 5 分钟生成比赛事件。
4. 结果校准层：限制极端比分，输出统计。

## 四、战术克制关系

战术不是简单加分，而是互相克制。

| 我方战术 | 对方战术 | 修正 |
|---|---|---:|
| 高位压迫 | 后场短传控球 | 我方抢断事件 +8%，体能消耗 +10% |
| 快速反击 | 高防线压迫 | 我方单刀事件 +10% |
| 防守反击 | 控球组织 | 对方控球增加，我方反击事件 +6% |
| 边路传中 | 窄阵防守 | 传中事件 +10% |
| 中路渗透 | 双后腰防守 | 中路机会 -8% |
| 长传冲吊 | 矮个中卫 | 争顶成功 +8% |
| 巨星自由发挥 | 严密盯防 | 核心球员被限制概率 +10% |

## 五、xG 生成模型

### 1. 基础 xG

```text
home_base_xg = league_avg_home_xg
away_base_xg = league_avg_away_xg
```

默认值：

```text
league_avg_home_xg = 1.45
league_avg_away_xg = 1.15
```

### 2. 强弱修正

```text
attack_defense_ratio = team_attack / opponent_defense
control_ratio = team_control / opponent_control
```

```text
xg = base_xg
  * clamp(attack_defense_ratio, 0.55, 1.75)
  * clamp(0.85 + (control_ratio - 1.0) * 0.25, 0.80, 1.20)
  * tactical_modifier
  * morale_modifier
  * condition_modifier
  * match_importance_modifier
```

### 3. 上下限

常规比赛单队 xG 限制：

```text
min_xg = 0.15
max_xg = 4.20
```

极端强弱差可突破，但需触发 rare_extreme_match 标记。

## 六、进球生成

V1 采用泊松分布生成进球数，再由事件系统分配进球时刻。

```text
goals = poisson(xg)
```

为防止极端比分泛滥：

```text
if goals >= 5:
    apply_extreme_score_dampening()
```

极端比分抑制规则：

- 单队 5 球以上概率乘 0.65。
- 单队 6 球以上概率乘 0.40。
- 单队 7 球以上概率乘 0.20。
- 若对手红牌、门将受伤、实力差巨大，可减弱抑制。

## 七、事件生成

每 5 分钟一个 tick，共 18 个常规 tick + 补时 tick。

每个 tick 计算：

1. 控球方。
2. 事件类型。
3. 参与球员。
4. 事件结果。
5. 是否转化为射门或进球。

### 事件概率基础表

| 事件 | 基础概率 |
|---|---:|
| 普通控球推进 | 35% |
| 普通进攻 | 22% |
| 危险进攻 | 10% |
| 射门 | 8% |
| 角球 | 5% |
| 任意球 | 4% |
| 反击 | 4% |
| 越位 | 3% |
| 犯规 | 5% |
| 无明显事件 | 4% |

实际概率由战术、比分、体能、球员能力修正。

## 八、射门质量

射门质量分为：

- 低质量：0.03 - 0.08 xG
- 普通机会：0.08 - 0.18 xG
- 好机会：0.18 - 0.35 xG
- 单刀/空门：0.35 - 0.70 xG

射门质量受：

- 射门球员能力
- 传球者创造力
- 对方防守压力
- 战术类型
- 球星模板
- 体能
- 心理压力

## 十、红黄牌与伤病

### 黄牌概率

受以下因素影响：

- 犯规强度
- 防守球员侵略性
- 裁判严厉程度
- 比赛重要性
- 体能下降

### 红牌概率

基础极低，只在以下情况下提升：

- 已有黄牌继续高强度防守。
- 球员侵略性高、冷静低。
- 德比战。
- 落后急躁。

### 伤病概率

受以下因素影响：

- 伤病倾向
- 体能
- 比赛强度
- 天气
- 场地质量
- 近期出场密度
- 是否强行复出

## 十一、赛后评分

```text
match_rating = 6.5
+ goals * position_goal_weight
+ assists * 0.6
+ key_passes * 0.08
+ tackles_success * 0.05
+ interceptions * 0.05
+ saves * 0.10
- errors_leading_to_goal * 0.8
- red_card * 1.5
- own_goal * 1.0
+ team_result_bonus
```

评分限制：

```text
min = 4.0
max = 10.0
```

## 十二、平衡性测试

每次调整比赛引擎必须跑：

1. 10000 场随机强弱对战。
2. 1000 个完整联赛赛季。
3. 顶级强队 vs 弱队 1000 场。
4. 同级别球队 1000 场。
5. 不同战术互克矩阵测试。
