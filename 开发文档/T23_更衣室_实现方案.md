# T23 更衣室 - 详细实现方案

> 任务编号：T23
> 周期：W86-W90（5 周）
> 优先级：P1（依赖 T22 董事会 + T07 每日推进 + T02 比赛引擎）
> 依据：V0.1 `02_功能总表_35系统需求.md §14 更衣室系统` + `03_页面结构与交互说明.md §6 更衣室页` + V0.2 `05_AI俱乐部决策模型.md`（dressing_room_unrest 入参）
> 铁律：士气多因子加权 / 派系形成有上限 / 谈话效果双向 / 更衣室兵变必须有缓冲 / 士气计算频率分批 / 不影响每日推进 P95 ≤3 秒
> 定位：D 轨俱乐部管理系统的核心任务之一，更衣室稳定直接影响球队表现与董事会满意度，是 Gate 3 长程模拟"世界像足球"的关键体验层

## 一、业务目标与范围

### 1. V1 做

- **更衣室页（二级页面）**：队长 / 核心球员 / 派系关系 / 不满球员 / 士气趋势 / 谈话记录六大模块
- **球员士气系统**：12 类影响因素（上场时间 / 比赛结果 / 个人表现 / 合同状况 / 战术适配 / 队友关系 / 教练关系 / 球队氛围 / 俱乐部成就 / 个人荣誉 / 伤病状况 / 家庭生活），5 档士气等级（极高 / 高 / 中 / 低 / 极低）
- **派系系统**：6 类派系（国籍 / 语言 / 老将 vs 年轻 / 青训 vs 外购 / 核心 / 边缘），派系影响传播
- **更衣室谈话**：6 种谈话类型（单独谈话 / 公开支持 / 公开批评 / 承诺出场时间 / 承诺战术调整 / 警告），效果双向
- **队长系统**：队长影响力 / 领导力属性 / 与派系关系 / 更换队长风险
- **更衣室事件**：8 类事件（球员冲突 / 派系对立 / 上场时间不满 / 要求离队 / 更衣室兵变 / 队长公开表态 / 老将告别 / 新援融入困难）
- **士气对比赛表现的影响**：与 T02 比赛引擎衔接，士气 → 表现系数
- **派系/士气数据持久化**：Room Entity 落 save.db，支持赛季归档

### 2. V1 不做（推迟到 M7+）

- 球员个性深度模拟（仅用 leadership/professionalism/temperament 三属性简化驱动）
- 派系动态分裂与合并（V1 派系形成后只做成员增减，不做派系演化）
- 多语言派系与国籍派系的复合叠加（V1 二选一归属）
- 球员私人生活事件链（仅做随机单点事件）
- 球员与主帅的私人恩怨剧情（仅做关系值变化）
- 更衣室"小团体"可视化图谱（V1 用列表展示，不做力导向图）
- 女足 / 青年队的独立更衣室（仅一线队）
- 球员工会 / 球员代表的协商机制
- 队长竞选机制（V1 由经理直接任命）

## 三、数据模型（Room Entity + DAO）

### 1. 三库归属

| 表名 | 所属库 | 理由 |
|------|--------|------|
| player_morale | save.db | 每存档独立，每日推进高频写 |
| faction | save.db | 每存档独立，赛季级低频写 |
| faction_member | save.db | 每存档独立，转会窗中频写 |
| conversation | save.db | 每存档独立，玩家触发 |
| conversation_log | save.db | 每存档独立，仅追加 |
| captaincy | save.db | 每存档独立，低频写 |
| dressing_room_event | save.db | 每存档独立，事件流 |
| dressing_room_event_archive | history.db | 赛季归档后迁入（T19 调用） |
| faction_template | cache.db | 派系类型配置缓存，全局共享 |

> **三库分离铁律**：T23 不在 history.db 写入运行时数据；cache.db 仅缓存派系模板与计算中间结果；所有运行时状态落 save.db。

```kotlin
// persistence/dao/PlayerMoraleDao.kt
@Dao
interface PlayerMoraleDao {

    @Query("SELECT * FROM player_morale WHERE save_id = :saveId AND player_id = :playerId LIMIT 1")
    suspend fun getByPlayer(saveId: Int, playerId: Int): PlayerMoraleEntity?

    @Query("SELECT * FROM player_morale WHERE save_id = :saveId AND club_id = :clubId")
    suspend fun getByClub(saveId: Int, clubId: Int): List<PlayerMoraleEntity>

    @Query("""
        SELECT * FROM player_morale
        WHERE save_id = :saveId AND club_id = :clubId
        AND morale_level IN ('LOW','EXTREME_LOW')
        ORDER BY morale ASC
    """)
    suspend fun getUnhappyPlayers(saveId: Int, clubId: Int): List<PlayerMoraleEntity>

    @Query("""
        SELECT AVG(morale) FROM player_morale
        WHERE save_id = :saveId AND club_id = :clubId
    """)
    suspend fun getAverageMorale(saveId: Int, clubId: Int): Double?

    @Query("""
        SELECT * FROM player_morale
        WHERE save_id = :saveId AND club_id = :clubId
        AND unrest_accumulator >= :threshold
        AND pending_conversation = 0
    """)
    suspend fun getPlayersNeedingTalk(saveId: Int, clubId: Int, threshold: Int): List<PlayerMoraleEntity>

    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun upsert(entity: PlayerMoraleEntity): Long

    @Query("""
        UPDATE player_morale
        SET morale = morale + :delta,
            morale_level = :newLevel,
            last_updated_date = :date
        WHERE save_id = :saveId AND player_id = :playerId
    """)
    suspend fun applyMoraleDelta(
        saveId: Int, playerId: Int, delta: Int, newLevel: String, date: String
    )

    @Query("""
        UPDATE player_morale
        SET unrest_accumulator = unrest_accumulator + :delta,
            pending_conversation = :pending
        WHERE save_id = :saveId AND player_id = :playerId
    """)
    suspend fun adjustUnrest(saveId: Int, playerId: Int, delta: Int, pending: Boolean)

    @Query("DELETE FROM player_morale WHERE save_id = :saveId AND club_id = :clubId")
    suspend fun deleteByClub(saveId: Int, clubId: Int)
}
```

```kotlin
// api/DressingRoomManager.kt
class DressingRoomManager(
    private val moraleCalculator: MoraleCalculator,
    private val factionManager: FactionManager,
    private val captainManager: CaptainManager,
    private val conversationManager: ConversationManager,
    private val eventTrigger: DressingRoomEventTrigger,
    private val repository: DressingRoomRepository,
    private val config: DressingRoomConfig,
    private val eventBus: DressingRoomEventBus
) {

    /**
     * 入口：生成更衣室快照（UI 消费）
     * 一次性聚合：队长 / 核心球员 / 派系 / 不满球员 / 士气趋势 / 谈话记录
     */
    suspend fun getSnapshot(saveId: Int, clubId: Int): DressingRoomSnapshot {
        val captain = captainManager.getActiveCaptain(saveId, clubId)
        val corePlayers = factionManager.getCorePlayers(saveId, clubId)
        val factions = factionManager.getActiveFactions(saveId, clubId)
        val unhappyPlayers = repository.getUnhappyPlayers(saveId, clubId)
        val moraleTrend = repository.getMoraleTrend(saveId, clubId, days = 30)
        val recentConversations = repository.getRecentConversations(saveId, clubId, limit = 20)
        val teamMorale = repository.getAverageMorale(saveId, clubId)

        return DressingRoomSnapshot(
            captain = captain,
            corePlayers = corePlayers,
            factions = factions,
            unhappyPlayers = unhappyPlayers,
            moraleTrend = moraleTrend,
            recentConversations = recentConversations,
            teamMorale = teamMorale.toInt(),
            stabilityIndex = calculateStabilityIndex(teamMorale, factions, unhappyPlayers.size)
        )
    }

    /**
     * 每日推进调用：士气和派系日度更新（轻量）
     * 由 T07 DailyTaskExecutor.MoraleTask 调用
     * 性能预算：玩家俱乐部 50ms，活跃范围每队 10ms
     */
    suspend fun onDailyAdvance(date: LocalDate, saveId: Int, clubId: Int, isMatchDay: Boolean) {
        // 1. 士气日度衰减 / 微调（不重算全因子）
        moraleCalculator.applyDailyDrift(date, saveId, clubId)

        // 2. 比赛日赛后处理（由 MatchEngineMoraleBridge 触发，此处不重复）
        if (!isMatchDay) {
            // 3. 检查未兑现承诺（非比赛日检查）
            conversationManager.checkExpiredPromises(date, saveId, clubId)
        }

        // 4. 不满累积检查（低频，每周一）
        if (date.dayOfWeek == DayOfWeek.MONDAY) {
            eventTrigger.checkUnrestAccumulation(date, saveId, clubId)
        }
    }

    /**
     * 每周推进调用：派系士气传染 + 派系成员更新
     * 由 T07 每周任务"4. 更衣室状态"调用
     */
    suspend fun onWeeklyAdvance(date: LocalDate, saveId: Int, clubId: Int) {
        factionManager.updateFactionMembership(saveId, clubId)
        factionManager.propagateFactionMorale(saveId, clubId)
    }

    /**
     * 每月推进调用：完整重算士气全因子 + 队长影响 + 派系凝聚力
     * 由 T07 每月任务调用
     */
    suspend fun onMonthlyAdvance(date: LocalDate, saveId: Int, clubId: Int) {
        moraleCalculator.recalculateAllFactors(date, saveId, clubId)
        captainManager.applyCaptainInfluence(saveId, clubId)
        factionManager.recalculateFactionCohesion(saveId, clubId)
    }

    /**
     * 玩家操作：发起谈话
     */
    suspend fun initiateConversation(
        saveId: Int, clubId: Int, playerId: Int,
        type: ConversationType, tone: ConversationTone,
        playtimePromiseMinutes: Int? = null,
        promiseDeadlineDate: LocalDate? = null
    ): ConversationOutcome {
        val outcome = conversationManager.execute(
            saveId, clubId, playerId, type, tone,
            playtimePromiseMinutes, promiseDeadlineDate
        )
        // 谈话可能触发事件（如警告导致球员要求离队）
        if (outcome.triggeredTransferRequest) {
            eventTrigger.triggerTransferRequest(
                saveId, clubId, playerId,
                reason = "警告后球员不满"
            )
        }
        eventBus.emit(ConversationEvent(saveId, clubId, playerId, outcome))
        return outcome
    }

    /**
     * 玩家操作：调整队长
     */
    suspend fun changeCaptain(
        saveId: Int, clubId: Int, newCaptainPlayerId: Int, date: LocalDate
    ): CaptainChangeResult {
        val result = captainManager.appointCaptain(saveId, clubId, newCaptainPlayerId, date)
        // 更换队长可能引起老队长派系不满
        if (result.previousCaptainId != null) {
            eventTrigger.onCaptainChanged(saveId, clubId, result.previousCaptainId, newCaptainPlayerId, date)
        }
        eventBus.emit(CaptainChangeEvent(saveId, clubId, result))
        return result
    }

    /**
     * 赛季归档时调用（T19 集成）
     */
    suspend fun onSeasonArchive(saveId: Int, clubId: Int, season: Int) {
        repository.archiveEventsToHistoryDb(saveId, clubId, season)
        repository.resetUnrestAccumulator(saveId, clubId)
    }

    /**
     * 稳定指数：综合球队士气 / 派系冲突数 / 不满球员数
     * 0-100，越高越稳定，提供给 T22 董事会
     */
    private fun calculateStabilityIndex(
        teamMorale: Double, factions: List<Faction>, unhappyCount: Int
    ): Int {
        val moraleScore = teamMorale  // 0-100
        val factionConflictPenalty = factions.count { it.factionMorale < 40 } * 5
        val unhappyPenalty = (unhappyCount * 3).coerceAtMost(30)
        return (moraleScore - factionConflictPenalty - unhappyPenalty).toInt().coerceIn(0, 100)
    }
}
```

### 2. MoraleCalculator（士气计算器）

```kotlin
// morale/MoraleCalculator.kt
class MoraleCalculator(
    private val repository: DressingRoomRepository,
    private val config: DressingRoomConfig,
    private val matchBridge: MatchEngineMoraleBridge,
    private val contractBridge: ContractMoraleBridge,
    private val transferBridge: TransferMoraleBridge
) {

    /**
     * 12 因子士气计算（V0.1 §14 + 任务描述）
     * morale =
     *   playing_time       * 0.20
     * + match_result       * 0.15
     * + personal_perf      * 0.12
     * + contract_status    * 0.10
     * + tactical_fit       * 0.08
     * + teammate_relation  * 0.08
     * + coach_relation     * 0.07
     * + team_atmosphere    * 0.07
     * + club_achievement   * 0.05
     * + personal_honor     * 0.04
     * + injury_status      * 0.03
     * + life_event         * 0.01
     *
     * 输出 0-100，5 档分级
     */
    suspend fun calculateFactorScores(
        date: LocalDate, saveId: Int, clubId: Int, playerId: Int
    ): MoraleFactorScores {
        val playerState = repository.getPlayerState(saveId, playerId)
        val seasonStats = repository.getPlayerSeasonStats(saveId, playerId, date.year)

        val playingTime = calculatePlayingTimeFactor(playerState, seasonStats)
        val matchResult = calculateMatchResultFactor(saveId, clubId, playerId)
        val personalPerf = calculatePersonalPerformanceFactor(seasonStats)
        val contract = contractBridge.calculateContractFactor(saveId, playerId, date)
        val tacticalFit = calculateTacticalFitFactor(playerState)
        val teammateRelation = calculateTeammateRelationFactor(saveId, clubId, playerId)
        val coachRelation = calculateCoachRelationFactor(saveId, playerId)
        val teamAtmosphere = calculateTeamAtmosphereFactor(saveId, clubId)
        val clubAchievement = calculateClubAchievementFactor(saveId, clubId)
        val personalHonor = calculatePersonalHonorFactor(saveId, playerId)
        val injury = calculateInjuryFactor(playerState, date)
        val lifeEvent = calculateLifeEventFactor(playerState, date)

        return MoraleFactorScores(
            playingTime = playingTime,
            matchResult = matchResult,
            personalPerformance = personalPerf,
            contract = contract,
            tacticalFit = tacticalFit,
            teammateRelation = teammateRelation,
            coachRelation = coachRelation,
            teamAtmosphere = teamAtmosphere,
            clubAchievement = clubAchievement,
            personalHonor = personalHonor,
            injury = injury,
            lifeEvent = lifeEvent
        )
    }

    suspend fun recalculateAllFactors(date: LocalDate, saveId: Int, clubId: Int) {
        val players = repository.getPlayersByClub(saveId, clubId)
        for (player in players) {
            val scores = calculateFactorScores(date, saveId, clubId, player.playerId)
            val morale = applyWeights(scores)
            val level = MoraleLevel.fromScore(morale)

            repository.upsertMorale(
                PlayerMoraleEntity(
                    saveId = saveId,
                    playerId = player.playerId,
                    clubId = clubId,
                    savePlayerStateId = player.id,
                    morale = morale,
                    moraleLevel = level.name,
                    playingTimeFactor = scores.playingTime,
                    matchResultFactor = scores.matchResult,
                    personalPerformanceFactor = scores.personalPerformance,
                    contractFactor = scores.contract,
                    tacticalFitFactor = scores.tacticalFit,
                    teammateRelationFactor = scores.teammateRelation,
                    coachRelationFactor = scores.coachRelation,
                    teamAtmosphereFactor = scores.teamAtmosphere,
                    clubAchievementFactor = scores.clubAchievement,
                    personalHonorFactor = scores.personalHonor,
                    injuryFactor = scores.injury,
                    lifeEventFactor = scores.lifeEvent,
                    lastUpdatedDate = date.toString(),
                    lastUpdatedSeason = repository.getCurrentSeason(saveId)
                )
            )
        }
    }

    /**
     * 日度衰减：士气向均值回归 + 微小随机扰动
     * 避免士气长期卡在极值
     */
    suspend fun applyDailyDrift(date: LocalDate, saveId: Int, clubId: Int) {
        val players = repository.getPlayersByClub(saveId, clubId)
        val regressionRate = config.dailyDrift.regressionRate  // 默认 0.02
        val noiseRange = config.dailyDrift.noiseRange          // 默认 1

        for (player in players) {
            val morale = repository.getMorale(saveId, player.playerId) ?: continue
            // 向 50（中位）回归
            val regression = (50 - morale.morale) * regressionRate
            val noise = (-noiseRange..noiseRange).random()
            val newMorale = (morale.morale + regression + noise).toInt().coerceIn(0, 100)
            val newLevel = MoraleLevel.fromScore(newMorale)
            repository.updateMorale(saveId, player.playerId, newMorale, newLevel, date.toString())
        }
    }

    /**
     * 比赛结束后由 MatchEngineMoraleBridge 调用
     * 根据上场时间 / 比赛结果 / 个人表现即时更新
     */
    suspend fun applyMatchImpact(
        date: LocalDate, saveId: Int, clubId: Int,
        matchResult: MatchResult, playerMatchStats: List<PlayerMatchStat>
    ) {
        for (stat in playerMatchStats) {
            val playingTimeDelta = calculatePlayingTimeDelta(stat)
            val matchResultDelta = calculateMatchResultDelta(matchResult, stat)
            val personalPerfDelta = calculatePersonalPerfDelta(stat)
            val totalDelta = playingTimeDelta + matchResultDelta + personalPerfDelta
            // 单场士气变化上限，避免剧烈波动
            val clampedDelta = totalDelta.coerceIn(-config.matchImpact.maxDelta, config.matchImpact.maxDelta)

            val current = repository.getMorale(saveId, stat.playerId) ?: continue
            val newMorale = (current.morale + clampedDelta).coerceIn(0, 100)
            val newLevel = MoraleLevel.fromScore(newMorale)
            repository.updateMorale(saveId, stat.playerId, newMorale, newLevel, date.toString())

            // 不满累积：长期未上场的球员累积不满
            if (stat.minutesPlayed == 0 && matchResult.isImportant) {
                repository.adjustUnrest(saveId, stat.playerId, delta = 1, pending = true)
            }
        }
    }

    // ===== 各因子计算 =====

    /**
     * 上场时间因子（任务描述：首发高 / 替补低 / 长期替补很低）
     * 基于 minutes_ratio（本月出场分钟 / 可能出场分钟）
     */
    private fun calculatePlayingTimeFactor(player: SavePlayerState, stats: PlayerSeasonStats): Double {
        val minutesRatio = stats.recentMinutesRatio  // 0-1
        return when {
            minutesRatio >= 0.75 -> 0.95   // 首发：高
            minutesRatio >= 0.40 -> 0.65   // 替补：中
            minutesRatio >= 0.10 -> 0.35   // 偶尔出场：低
            else -> 0.15                   // 长期替补：很低
        }
    }

    /** 比赛结果因子（连胜高 / 连败低） */
    private fun calculateMatchResultFactor(saveId: Int, clubId: Int, playerId: Int): Double {
        val recentForm = repository.getRecentForm(saveId, clubId, matches = 5)
        val winRate = recentForm.count { it.isWin } / recentForm.size.coerceAtLeast(1)
        val streakBonus = calculateStreakBonus(recentForm)
        return (winRate * 0.7 + streakBonus * 0.3).coerceIn(0.0, 1.0)
    }

    /** 个人表现因子（进球/助攻/低分） */
    private fun calculatePersonalPerformanceFactor(stats: PlayerSeasonStats): Double {
        val baseScore = (stats.averageRating - 5.0) / 5.0  // 评分 5-10 映射到 0-1
        val goalBonus = (stats.goals * 0.05).coerceAtMost(0.2)
        val assistBonus = (stats.assists * 0.03).coerceAtMost(0.15)
        return (baseScore + goalBonus + assistBonus).coerceIn(0.0, 1.0)
    }

    /** 战术适配因子（在熟悉位置高 / 客串低） */
    private fun calculateTacticalFitFactor(player: SavePlayerState): Double {
        val isPrimaryPosition = player.lastPlayedPosition == player.primaryPosition
        val isFamiliar = player.familiarPositions.contains(player.lastPlayedPosition)
        return when {
            isPrimaryPosition -> 1.0
            isFamiliar -> 0.7
            else -> 0.3  // 客串
        }
    }

    /** 队友关系因子（基于派系归属 + 同派系数量） */
    private fun calculateTeammateRelationFactor(saveId: Int, clubId: Int, playerId: Int): Double {
        val factionMembership = repository.getFactionMembership(saveId, playerId)
        if (factionMembership == null) return 0.5  // 不属于任何派系
        val sameFactionMembers = repository.countFactionMembers(saveId, factionMembership.factionId)
        val factionMorale = repository.getFactionMorale(saveId, factionMembership.factionId)
        val baseRelation = (sameFactionMembers - 1) / 10.0  // 同派系越多关系越好
        return (baseRelation * 0.5 + factionMorale / 100.0 * 0.5).coerceIn(0.0, 1.0)
    }

    /** 教练关系因子（V1 简化：基于球员忠诚度 + 队长关系） */
    private fun calculateCoachRelationFactor(saveId: Int, playerId: Int): Double {
        val loyalty = repository.getPlayerLoyalty(saveId, playerId)
        val captainRelation = repository.getCaptainRelation(saveId, playerId)
        return (loyalty / 100.0 * 0.6 + captainRelation / 100.0 * 0.4)
    }

    /** 球队氛围因子（基于球队平均士气 + 派系冲突数） */
    private fun calculateTeamAtmosphereFactor(saveId: Int, clubId: Int): Double {
        val avgMorale = repository.getAverageMorale(saveId, clubId) ?: 50.0
        val factionConflictCount = repository.countFactionConflicts(saveId, clubId)
        val conflictPenalty = (factionConflictCount * 0.1).coerceAtMost(0.3)
        return (avgMorale / 100.0 - conflictPenalty).coerceIn(0.0, 1.0)
    }

    /** 俱乐部成就因子（基于近 2 赛季奖杯数 + 联赛排名） */
    private fun calculateClubAchievementFactor(saveId: Int, clubId: Int): Double {
        val recentTrophies = repository.countRecentTrophies(saveId, clubId, seasons = 2)
        val recentLeaguePosition = repository.getRecentLeaguePosition(saveId, clubId)
        val trophyBonus = (recentTrophies * 0.2).coerceAtMost(0.4)
        val positionFactor = when {
            recentLeaguePosition <= 3 -> 0.6
            recentLeaguePosition <= 6 -> 0.4
            recentLeaguePosition <= 10 -> 0.2
            else -> 0.0
        }
        return (trophyBonus + positionFactor).coerceIn(0.0, 1.0)
    }

    /** 个人荣誉因子（基于个人奖项） */
    private fun calculatePersonalHonorFactor(saveId: Int, playerId: Int): Double {
        val recentAwards = repository.countRecentPersonalAwards(saveId, playerId, seasons = 2)
        return (recentAwards * 0.3).coerceIn(0.0, 1.0)
    }

    /** 伤病状况因子（受伤 / 长期伤 / 反复伤低） */
    private fun calculateInjuryFactor(player: SavePlayerState, date: LocalDate): Double {
        return when {
            player.injuryStatus == "healthy" -> 0.9
            player.injuryStatus in listOf("minor") -> 0.5
            player.injuryStatus in listOf("moderate") -> 0.3
            player.injuryStatus in listOf("severe", "career_threatening") -> 0.1
            else -> 0.7
        }
    }

    /** 家庭/生活因子（随机事件，5% 概率触发） */
    private fun calculateLifeEventFactor(player: SavePlayerState, date: LocalDate): Double {
        val lifeEvent = repository.getActiveLifeEvent(player.playerId, date)
        return when {
            lifeEvent == null -> 0.7  // 默认中性
            lifeEvent.isPositive -> 0.95
            else -> 0.2
        }
    }

    /**
     * 加权求和
     */
    private fun applyWeights(scores: MoraleFactorScores): Int {
        val w = config.moraleWeights
        val morale = (
            scores.playingTime * w.playingTime +
            scores.matchResult * w.matchResult +
            scores.personalPerformance * w.personalPerformance +
            scores.contract * w.contract +
            scores.tacticalFit * w.tacticalFit +
            scores.teammateRelation * w.teammateRelation +
            scores.coachRelation * w.coachRelation +
            scores.teamAtmosphere * w.teamAtmosphere +
            scores.clubAchievement * w.clubAchievement +
            scores.personalHonor * w.personalHonor +
            scores.injury * w.injury +
            scores.lifeEvent * w.lifeEvent
        ) * 100
        return morale.toInt().coerceIn(0, 100)
    }
}
```

### 3. FactionManager（派系管理）

```kotlin
// faction/FactionManager.kt
class FactionManager(
    private val repository: DressingRoomRepository,
    private val propagator: FactionInfluencePropagator,
    private val config: DressingRoomConfig
) {

    /**
     * 派系形成算法（6 类派系）
     * 触发：每月推进 / 转会窗关闭后 / 新援融入时
     *
     * 规则：
     * - 同国籍 ≥3 人 → 形成国籍派系
     * - 同语言 ≥3 人 → 形成语言派系（与国籍派系互斥，二选一）
     * - ≥30 岁球员 ≥3 人 + ≤21 岁球员 ≥3 人 → 老将 vs 年轻派系
     * - 青训球员 ≥3 人 + 外购球员 ≥3 人 → 青训 vs 外购派系
     * - 主力（squadRole=starter/key_player）≥3 人 → 核心派系
     * - 替补/边缘（squadRole=backup/fringe）≥3 人 → 边缘派系
     *
     * 限制：单俱乐部活跃派系数 ≤ config.maxFactionsPerClub（默认 6）
     */
    suspend fun updateFactionMembership(saveId: Int, clubId: Int) {
        val players = repository.getPlayersByClub(saveId, clubId)
        val existingFactions = repository.getActiveFactions(saveId, clubId)

        // 1. 国籍派系（同 nationality ≥3）
        val byNationality = players.groupBy { it.nationality }
            .filter { it.value.size >= config.factionFormation.minMembersForNationality }
        for ((nation, members) in byNationality) {
            val factionKey = "NATIONALITY:$nation"
            ensureFactionExists(
                saveId, clubId, FactionType.NATIONALITY, factionKey,
                name = "${nation}帮",
                members = members,
                existingFactions = existingFactions
            )
        }

        // 2. 语言派系（同语言 ≥3，与国籍派系互斥）
        // 简化：若该俱乐部已有 ≥1 个国籍派系，则不再生成语言派系，避免叠加
        val hasNationalityFaction = existingFactions.any { it.factionType == FactionType.NATIONALITY.name }
        if (!hasNationalityFaction || config.factionFormation.allowLanguageFactionCoexist) {
            val byLanguage = players.groupBy { it.language }
                .filter { it.value.size >= config.factionFormation.minMembersForLanguage }
            for ((lang, members) in byLanguage) {
                val factionKey = "LANGUAGE:$lang"
                ensureFactionExists(
                    saveId, clubId, FactionType.LANGUAGE, factionKey,
                    name = "${lang}语系",
                    members = members,
                    existingFactions = existingFactions
                )
            }
        }

        // 3. 老将 vs 年轻派系
        val veterans = players.filter { it.age >= config.factionFormation.veteranAgeThreshold }
        val youngsters = players.filter { it.age <= config.factionFormation.youngAgeThreshold }
        if (veterans.size >= 3 && youngsters.size >= 3) {
            ensureFactionExists(saveId, clubId, FactionType.VETERAN_YOUTH, "VETERAN_YOUTH:VETERAN",
                "老将派系", veterans, existingFactions)
            ensureFactionExists(saveId, clubId, FactionType.VETERAN_YOUTH, "VETERAN_YOUTH:YOUTH",
                "年轻球员派系", youngsters, existingFactions)
        }

        // 4. 青训 vs 外购派系
        val academy = players.filter { it.isAcademyProduct }
        val bought = players.filter { !it.isAcademyProduct }
        if (academy.size >= 3 && bought.size >= 3) {
            ensureFactionExists(saveId, clubId, FactionType.ACADEMY_BOUGHT, "ACADEMY_BOUGHT:ACADEMY",
                "青训派系", academy, existingFactions)
            ensureFactionExists(saveId, clubId, FactionType.ACADEMY_BOUGHT, "ACADEMY_BOUGHT:BOUGHT",
                "外购派系", bought, existingFactions)
        }

        // 5. 核心派系
        val core = players.filter { it.squadRole in listOf("starter", "key_player") }
        if (core.size >= 3) {
            ensureFactionExists(saveId, clubId, FactionType.CORE_FRINGE, "CORE_FRINGE:CORE",
                "核心球员派系", core, existingFactions)
        }

        // 6. 边缘派系
        val fringe = players.filter { it.squadRole in listOf("backup", "fringe") }
        if (fringe.size >= 3) {
            ensureFactionExists(saveId, clubId, FactionType.CORE_FRINGE, "CORE_FRINGE:FRINGE",
                "边缘球员派系", fringe, existingFactions)
        }

        // 7. 解散成员 <2 的派系
        for (faction in existingFactions) {
            val memberCount = repository.countFactionMembers(saveId, faction.id)
            if (memberCount < config.factionFormation.minMembersToSurvive) {
                repository.deactivateFaction(faction.id)
            }
        }

        // 8. 上限保护：活跃派系 >maxFactionsPerClub 时，按凝聚力从低到高解散
        enforceMaxFactionsLimit(saveId, clubId)
    }

    private suspend fun ensureFactionExists(
        saveId: Int, clubId: Int, type: FactionType, key: String,
        name: String, members: List<SavePlayerState>,
        existingFactions: List<FactionEntity>
    ) {
        val existing = existingFactions.find { it.factionKey == key }
        if (existing != null) {
            // 更新成员
            syncFactionMembers(saveId, existing.id, members)
        } else {
            val leader = selectFactionLeader(members)
            val factionId = repository.insertFaction(FactionEntity(
                saveId = saveId, clubId = clubId,
                factionType = type.name, factionKey = key, factionName = name,
                leaderPlayerId = leader.playerId,
                memberCount = members.size,
                factionMorale = 50,
                factionCohesion = 0.5,
                isActive = true,
                formedDate = currentDate(),
                formedSeason = repository.getCurrentSeason(saveId)
            ))
            syncFactionMembers(saveId, factionId, members)
        }
    }

    /**
     * 派系领袖选择：leadership 属性最高 + 主力优先
     */
    private suspend fun selectFactionLeader(members: List<SavePlayerState>): SavePlayerState {
        return members.maxWithOrNull(
            compareBy(
                { it.squadRole == "key_player" || it.squadRole == "starter" },
                { repository.getLeadership(it.playerId) }
            )
        ) ?: members.first()
    }

    /**
     * 派系影响传播：派系成员士气相互影响
     * 规则：
     * - 派系领袖的士气变化 → 影响所有成员（衰减 0.5）
     * - 派系平均士气与成员士气差距 → 拉近 5%
     * - 派系不满传染：>50% 成员 morale<40 → 全员 -3
     */
    suspend fun propagateFactionMorale(saveId: Int, clubId: Int) {
        propagator.propagate(saveId, clubId)
    }

    /**
     * 派系凝聚力重算（每月）
     * cohesion = 同派系成员平均士气一致性 + 领袖影响力
     */
    suspend fun recalculateFactionCohesion(saveId: Int, clubId: Int) {
        val factions = repository.getActiveFactions(saveId, clubId)
        for (faction in factions) {
            val members = repository.getFactionMembers(saveId, faction.id)
            if (members.isEmpty()) continue
            val morales = members.map { repository.getMorale(saveId, it.playerId)?.morale ?: 50 }
            val avg = morales.average()
            val stdDev = morales.map { (it - avg).pow(2) }.average().sqrt()
            val consistency = (1.0 - stdDev / 50.0).coerceIn(0.0, 1.0)
            val leaderInfluence = repository.getLeadership(faction.leaderPlayerId) / 100.0
            val cohesion = (consistency * 0.6 + leaderInfluence * 0.4).coerceIn(0.0, 1.0)
            val newFactionMorale = avg.toInt()
            repository.updateFactionMorale(faction.id, newFactionMorale, cohesion)
        }
    }

    suspend fun getActiveFactions(saveId: Int, clubId: Int): List<Faction> =
        repository.getActiveFactions(saveId, clubId).map { it.toDomain() }

    suspend fun getCorePlayers(saveId: Int, clubId: Int): List<Int> =
        repository.getPlayersByClub(saveId, clubId)
            .filter { it.squadRole in listOf("starter", "key_player") }
            .map { it.playerId }

    private suspend fun enforceMaxFactionsLimit(saveId: Int, clubId: Int) {
        val active = repository.getActiveFactions(saveId, clubId)
        if (active.size > config.maxFactionsPerClub) {
            val sorted = active.sortedBy { it.factionCohesion }
            val toDisband = sorted.take(active.size - config.maxFactionsPerClub)
            for (faction in toDisband) {
                repository.deactivateFaction(faction.id)
            }
        }
    }
}

class FactionInfluencePropagator(
    private val repository: DressingRoomRepository,
    private val config: DressingRoomConfig
) {

    /**
     * 派系士气传染算法
     */
    suspend fun propagate(saveId: Int, clubId: Int) {
        val factions = repository.getActiveFactions(saveId, clubId)

        for (faction in factions) {
            val members = repository.getFactionMembers(saveId, faction.id)
            if (members.size < 2) continue

            val morales = members.map { member ->
                member.playerId to (repository.getMorale(saveId, member.playerId)?.morale ?: 50)
            }
            val avg = morales.map { it.second }.average()

            // 1. 领袖影响：领袖士气变化 → 衰减 0.5 影响成员
            val leaderMorale = repository.getMorale(saveId, faction.leaderPlayerId)?.morale ?: 50
            val leaderDelta = leaderMorale - 50

            for ((playerId, morale) in morales) {
                if (playerId == faction.leaderPlayerId) continue

                // 拉近派系均值 5%
                val pullToAvg = (avg - morale) * 0.05
                // 领袖影响衰减
                val leaderInfluence = leaderDelta * 0.5 * config.factionInfluence.leaderPropagationRate
                val newMorale = (morale + pullToAvg + leaderInfluence).toInt().coerceIn(0, 100)
                repository.updateMorale(saveId, playerId, newMorale, MoraleLevel.fromScore(newMorale).name, currentDate())
            }
        }

        // 2. 派系不满传染：>50% 成员 morale<40 → 全员 -3
        for (faction in factions) {
            val members = repository.getFactionMembers(saveId, faction.id)
            val unhappyCount = members.count {
                (repository.getMorale(saveId, it.playerId)?.morale ?: 50) < 40
            }
            if (unhappyCount > members.size / 2) {
                for (member in members) {
                    val current = repository.getMorale(saveId, member.playerId)?.morale ?: continue
                    val newMorale = (current - config.factionInfluence.unhappinessContagionPenalty).coerceIn(0, 100)
                    repository.updateMorale(saveId, member.playerId, newMorale, MoraleLevel.fromScore(newMorale).name, currentDate())
                }
            }
        }
    }
}
```

```kotlin
// captain/CaptainManager.kt
class CaptainManager(
    private val repository: DressingRoomRepository,
    private val config: DressingRoomConfig
) {

    /**
     * 队长任命
     * 规则：
     * - 撤销原队长 → 原队长士气 -15，原队长所在派系士气 -5
     * - 新队长士气 +10，所在派系士气 +3
     * - 若新队长 leadership <50 → 任命效果减半
     */
    suspend fun appointCaptain(
        saveId: Int, clubId: Int, newCaptainPlayerId: Int, date: LocalDate
    ): CaptainChangeResult {
        val previous = repository.getActiveCaptain(saveId, clubId)
        val newCaptainLeadership = repository.getLeadership(newCaptainPlayerId)

        // 1. 撤销原队长
        if (previous != null) {
            repository.revokeCaptain(previous.id, date.toString(), "MANAGER_DECISION")
            // 原队长士气 -15
            adjustMorale(saveId, previous.playerId, -config.captainChange.previousCaptainPenalty, date)
            // 原队长所在派系士气 -5
            val prevFaction = repository.getFactionMembership(saveId, previous.playerId)
            if (prevFaction != null) {
                repository.adjustFactionMorale(prevFaction.factionId, -config.captainChange.previousFactionPenalty)
            }
        }

        // 2. 任命新队长
        val captaincy = CaptaincyEntity(
            saveId = saveId, clubId = clubId, playerId = newCaptainPlayerId,
            appointedDate = date.toString(),
            appointedSeason = repository.getCurrentSeason(saveId),
            appointedBy = "MANAGER",
            leadership = newCaptainLeadership,
            influence = calculateCaptainInfluence(newCaptainLeadership),
            status = "ACTIVE"
        )
        val captaincyId = repository.insertCaptaincy(captaincy)

        // 3. 新队长士气 +10
        adjustMorale(saveId, newCaptainPlayerId, config.captainChange.newCaptainBonus, date)

        // 4. 新队长所在派系士气 +3
        val newFaction = repository.getFactionMembership(saveId, newCaptainPlayerId)
        if (newFaction != null) {
            repository.adjustFactionMorale(newFaction.factionId, config.captainChange.newFactionBonus)
        }

        return CaptainChangeResult(
            captaincyId = captaincyId,
            previousCaptainId = previous?.playerId,
            newCaptainPlayerId = newCaptainPlayerId,
            newCaptainInfluence = captaincy.influence
        )
    }

    /**
     * 队长影响力计算
     * influence = leadership * 0.6 + reputation_in_squad * 0.2 + age_factor * 0.2
     * V1 简化：leadership 是球员隐藏属性，由历史数据 + personality 推断
     */
    fun calculateCaptainInfluence(leadership: Int): Int {
        val base = leadership * 0.6
        val capped = base.coerceAtMost(80.0)
        return capped.toInt()
    }

    /**
     * 队长影响全队士气（每月推进调用）
     * 规则：
     * - 队长 influence >60 → 全队士气 +3
     * - 队长 influence 40-60 → 全队士气 +1
     * - 队长 influence <40 → 无影响
     * - 队长 morale <30 → 全队士气 -5（队长带头崩盘）
     */
    suspend fun applyCaptainInfluence(saveId: Int, clubId: Int) {
        val captain = repository.getActiveCaptain(saveId, clubId) ?: return
        val captainMorale = repository.getMorale(saveId, captain.playerId)?.morale ?: 50
        val players = repository.getPlayersByClub(saveId, clubId)

        val teamDelta = when {
            captainMorale < config.captainInfluence.captainLowMoraleThreshold ->
                -config.captainInfluence.captainCollapsePenalty
            captain.influence > config.captainInfluence.highInfluenceThreshold ->
                config.captainInfluence.highInfluenceBonus
            captain.influence > config.captainInfluence.midInfluenceThreshold ->
                config.captainInfluence.midInfluenceBonus
            else -> 0
        }

        if (teamDelta != 0) {
            for (player in players) {
                if (player.playerId == captain.playerId) continue
                val current = repository.getMorale(saveId, player.playerId)?.morale ?: continue
                val newMorale = (current + teamDelta).coerceIn(0, 100)
                repository.updateMorale(saveId, player.playerId, newMorale, MoraleLevel.fromScore(newMorale).name, currentDate())
            }
        }
    }

    suspend fun getActiveCaptain(saveId: Int, clubId: Int): Captaincy? =
        repository.getActiveCaptain(saveId, clubId)?.toDomain()

    private suspend fun adjustMorale(saveId: Int, playerId: Int, delta: Int, date: LocalDate) {
        val current = repository.getMorale(saveId, playerId)?.morale ?: 50
        val newMorale = (current + delta).coerceIn(0, 100)
        repository.updateMorale(saveId, playerId, newMorale, MoraleLevel.fromScore(newMorale).name, date.toString())
    }
}
```

```kotlin
// conversation/ConversationManager.kt
class ConversationManager(
    private val repository: DressingRoomRepository,
    private val outcomeCalculator: ConversationOutcomeCalculator,
    private val config: DressingRoomConfig
) {

    /**
     * 执行谈话
     * 6 种谈话类型，每种有不同的基础效果和风险
     */
    suspend fun execute(
        saveId: Int, clubId: Int, playerId: Int,
        type: ConversationType, tone: ConversationTone,
        playtimePromiseMinutes: Int? = null,
        promiseDeadlineDate: LocalDate? = null
    ): ConversationOutcome {
        // 1. 检查谈话冷却（避免同一球员一天被谈多次）
        val cooldown = repository.getLastConversationDate(saveId, playerId)
        val today = currentDate()
        if (cooldown != null && Duration.between(cooldown, today).toDays() < config.conversation.cooldownDays) {
            return ConversationOutcome(
                moraleDelta = 0, loyaltyDelta = 0,
                outcome = ConversationOutcomeType.NEUTRAL,
                playerReaction = "cooldown",
                triggeredTransferRequest = false,
                conversationId = 0
            )
        }

        // 2. 计算谈话效果
        val outcome = outcomeCalculator.calculate(
            saveId, clubId, playerId, type, tone
        )

        // 3. 应用士气变化
        val current = repository.getMorale(saveId, playerId)?.morale ?: 50
        val newMorale = (current + outcome.moraleDelta).coerceIn(0, 100)
        repository.updateMorale(saveId, playerId, newMorale, MoraleLevel.fromScore(newMorale).name, today.toString())

        // 4. 应用忠诚度变化
        repository.adjustLoyalty(saveId, playerId, outcome.loyaltyDelta)

        // 5. 公开谈话影响派系
        if (type.isPublic) {
            applyPublicConversationImpact(saveId, clubId, playerId, type, outcome)
        }

        // 6. 处理承诺
        val promiseId = if (playtimePromiseMinutes != null && promiseDeadlineDate != null) {
            createPlaytimePromise(saveId, clubId, playerId, playtimePromiseMinutes, promiseDeadlineDate, type, tone)
        } else null

        // 7. 清除 pending_conversation 标记
        repository.setPendingConversation(saveId, playerId, false)

        // 8. 记录谈话日志
        val conversationId = repository.insertConversation(
            saveId, clubId, playerId, type, tone, today,
            playtimePromiseMinutes, promiseDeadlineDate
        )
        repository.insertConversationLog(
            saveId, clubId, playerId, conversationId,
            outcome.moraleDelta, outcome.loyaltyDelta,
            outcome.outcome.name, outcome.playerReaction,
            outcome.triggeredTransferRequest
        )

        return outcome.copy(conversationId = conversationId)
    }

    /**
     * 检查未兑现承诺
     * 规则：
     * - 未在截止日前兑现 → 球员士气 -15，忠诚度 -10，unrest +2
     * - 兑现 → 士气 +5
     */
    suspend fun checkExpiredPromises(currentDate: LocalDate, saveId: Int, clubId: Int) {
        val expired = repository.getExpiredPromises(currentDate.toString())
        for (promise in expired) {
            val actualMinutes = repository.getMinutesPlayedSince(
                saveId, promise.playerId, promise.conversationDate, currentDate.toString()
            )
            val promised = promise.playtimePromiseMinutes ?: continue
            val isFulfilled = actualMinutes >= promised * 0.8  // 80% 兑现即视为兑现

            repository.markPromiseFulfilled(promise.id, isFulfilled)

            if (isFulfilled) {
                adjustMorale(saveId, promise.playerId, config.conversation.promiseFulfilledBonus, currentDate)
            } else {
                adjustMorale(saveId, promise.playerId, -config.conversation.promiseBrokenPenalty, currentDate)
                repository.adjustLoyalty(saveId, promise.playerId, -config.conversation.promiseBrokenLoyaltyPenalty)
                repository.adjustUnrest(saveId, promise.playerId, 2, true)
            }
        }
    }

    /**
     * 公开谈话对派系的影响
     * - 公开支持 → 该球员所在派系成员士气 +2
     * - 公开批评 → 该球员所在派系成员士气 -3（除非派系领袖认同）
     */
    private suspend fun applyPublicConversationImpact(
        saveId: Int, clubId: Int, playerId: Int,
        type: ConversationType, outcome: ConversationOutcome
    ) {
        val factionMembership = repository.getFactionMembership(saveId, playerId) ?: return
        val members = repository.getFactionMembers(saveId, factionMembership.factionId)

        val factionDelta = when (type) {
            ConversationType.PUBLIC_SUPPORT -> config.conversation.publicSupportFactionBonus
            ConversationType.PUBLIC_CRITICISM -> -config.conversation.publicCriticismFactionPenalty
            else -> 0
        }

        if (factionDelta != 0) {
            for (member in members) {
                if (member.playerId == playerId) continue
                val current = repository.getMorale(saveId, member.playerId)?.morale ?: continue
                val newMorale = (current + factionDelta).coerceIn(0, 100)
                repository.updateMorale(saveId, member.playerId, newMorale, MoraleLevel.fromScore(newMorale).name, currentDate().toString())
            }
        }
    }

    private suspend fun createPlaytimePromise(
        saveId: Int, clubId: Int, playerId: Int,
        minutes: Int, deadline: LocalDate, type: ConversationType, tone: ConversationTone
    ): Long {
        return repository.insertConversation(
            saveId, clubId, playerId, type, tone, currentDate(),
            playtimePromiseMinutes = minutes,
            promiseDeadlineDate = deadline
        )
    }

    private suspend fun adjustMorale(saveId: Int, playerId: Int, delta: Int, date: LocalDate) {

        val newMorale = (current + delta).coerceIn(0, 100)
        repository.updateMorale(saveId, playerId, newMorale, MoraleLevel.fromScore(newMorale).name, date.toString())

enum class ConversationType(val isPublic: Boolean, val baseMoraleDelta: Int) {
    PRIVATE_TALK(false, 0),         // 单独谈话（私人）
    PUBLIC_SUPPORT(true, 8),        // 公开支持：力挺球员

    WARNING(false, -3)              // 警告：最后机会
}

enum class ConversationTone { SUPPORTIVE, NEUTRAL, CRITICAL }

// conversation/ConversationOutcomeCalculator.kt
class ConversationOutcomeCalculator(
    private val repository: DressingRoomRepository,
    private val config: DressingRoomConfig
) {

    /**
     * 谈话效果计算（双向：可能好可能坏）
     * 影响因素：
     * - 球员性格（professionalism / temperament）
     * - 球员当前士气
     * - 谈话类型与球员状况的匹配度
     * - 队长关系
     */
    suspend fun calculate(
        saveId: Int, clubId: Int, playerId: Int,
        type: ConversationType, tone: ConversationTone
    ): ConversationOutcome {
        val player = repository.getPlayerState(saveId, playerId)
        val currentMorale = repository.getMorale(saveId, playerId)?.morale ?: 50
        val professionalism = repository.getProfessionalism(playerId)
        val temperament = repository.getTemperament(playerId)
        val isUnhappy = currentMorale < 40

        // 基础效果
        var moraleDelta = type.baseMoraleDelta

        // 性格修正
        val proMod = (professionalism - 50) / 100.0  // -0.5 ~ 0.5
        val tempMod = (temperament - 50) / 100.0     // -0.5 ~ 0.5

        when (type) {
            ConversationType.PUBLIC_CRITICISM, ConversationType.WARNING -> {
                // 高 professionalism 的球员接受批评 → 减少负效果
                moraleDelta = (moraleDelta * (1 + tempMod * 2)).toInt()
                // 低 temperament 的球员被批评会反弹 → 加重负效果
                if (temperament < 30) moraleDelta -= config.conversation.temperamentBackfirePenalty
            }
            ConversationType.PUBLIC_SUPPORT, ConversationType.PLAYTIME_PROMISE -> {
                // 不满球员得到支持效果加倍
                if (isUnhappy) moraleDelta += config.conversation.unhappySupportBonus
            }
            else -> {}
        }

        // 语气匹配度
        val toneMatch = when {
            tone == ConversationTone.SUPPORTIVE && currentMorale < 30 -> 1.2
            tone == ConversationTone.CRITICAL && currentMorale > 70 -> 0.8  // 高士气时批评反弹
            else -> 1.0
        }
        moraleDelta = (moraleDelta * toneMatch).toInt()

        // 计算忠诚度变化
        val loyaltyDelta = when (type) {
            ConversationType.PUBLIC_SUPPORT -> config.conversation.supportLoyaltyBonus
            ConversationType.PUBLIC_CRITICISM -> -config.conversation.criticismLoyaltyPenalty
            ConversationType.PLAYTIME_PROMISE -> config.conversation.promiseLoyaltyBonus
            ConversationType.WARNING -> -config.conversation.warningLoyaltyPenalty
            else -> 0
        }

        // 判断结果类型
        val outcomeType = when {
            moraleDelta > 3 -> ConversationOutcomeType.SUCCESS
            moraleDelta < -3 -> ConversationOutcomeType.BACKFIRE
            else -> ConversationOutcomeType.NEUTRAL
        }

        // 反弹风险：低 professionalism + 低 morale + WARNING → 可能触发转会要求
        val triggeredTransferRequest = type == ConversationType.WARNING &&
            professionalism < 40 && currentMorale < 30 &&
            Math.random() < config.conversation.transferRequestProbability

        val playerReaction = generateReactionKey(type, outcomeType, temperament)

        return ConversationOutcome(
            moraleDelta = moraleDelta,
            loyaltyDelta = loyaltyDelta,
            outcome = outcomeType,
            playerReaction = playerReaction,
            triggeredTransferRequest = triggeredTransferRequest,
            conversationId = 0
        )
    }

    private fun generateReactionKey(
        type: ConversationType, outcome: ConversationOutcomeType, temperament: Int
    ): String {
        val prefix = "conversation.${type.name.lowercase()}.${outcome.name.lowercase()}"
        val suffix = if (temperament < 30) ".volatile" else ".calm"
        return prefix + suffix
    }
}
```

```kotlin
// event/DressingRoomEventTrigger.kt
class DressingRoomEventTrigger(
    private val repository: DressingRoomRepository,
    private val mutinyGuard: MutinyGuard,
    private val config: DressingRoomConfig

    /**
     * 检查不满累积 → 触发上场时间不满事件
     * 每周一调用
     */
    suspend fun checkUnrestAccumulation(date: LocalDate, saveId: Int, clubId: Int) {
        val season = repository.getCurrentSeason(saveId)
        val players = repository.getPlayersNeedingTalk(saveId, clubId, config.unrest.unrestThreshold)

        for (player in players) {
            // 兵变缓冲检查：本队本赛季是否已有过兵变
            if (mutinyGuard.shouldSuppressEvent(saveId, clubId, season, "PLAYTIME_UNHAPPY")) {
                continue
            }

            val event = DressingRoomEventEntity(
                saveId = saveId, clubId = clubId,
                eventDate = date.toString(), eventSeason = season,
                eventType = "PLAYTIME_UNHAPPY",
                severity = if (player.unrestAccumulator >= config.unrest.severeThreshold) "MAJOR" else "MODERATE",
                involvedPlayerIds = player.playerId.toString(),
                involvedFactionIds = "",
                description = "球员因长期未获出场机会表达不满",
                moraleImpact = -2
            )
            val eventId = repository.insertEvent(event)
            // 重置累积器
            repository.adjustUnrest(saveId, player.playerId, -player.unrestAccumulator, true)

            // 触发全队小幅士气影响
            applyTeamMoraleImpact(saveId, clubId, -2, date)
        }
    }

    /**
     * 球员冲突事件（训练场 / 夜店）
     * 触发条件：两名球员士气均 <40 + 不同派系 + 随机概率
     */
    suspend fun checkPlayerConflict(date: LocalDate, saveId: Int, clubId: Int) {
        val season = repository.getCurrentSeason(saveId)
        if (mutinyGuard.shouldSuppressEvent(saveId, clubId, season, "PLAYER_CONFLICT")) return

        val unhappyPlayers = repository.getUnhappyPlayers(saveId, clubId)
        if (unhappyPlayers.size < 2) return

        // 找不同派系的两名球员
        val pairs = mutableListOf<Pair<Int, Int>>()
        for (i in unhappyPlayers.indices) {
            for (j in i + 1 until unhappyPlayers.size) {
                val f1 = repository.getFactionMembership(saveId, unhappyPlayers[i].playerId)
                val f2 = repository.getFactionMembership(saveId, unhappyPlayers[j].playerId)
                if (f1 != null && f2 != null && f1.factionId != f2.factionId) {
                    pairs.add(unhappyPlayers[i].playerId to unhappyPlayers[j].playerId)
                }
            }
        }
        if (pairs.isEmpty()) return

        // 随机概率触发
        if (Math.random() < config.eventTrigger.playerConflictProbability) {
            val (p1, p2) = pairs.random()
            val event = DressingRoomEventEntity(
                saveId = saveId, clubId = clubId,
                eventDate = date.toString(), eventSeason = season,
                eventType = "PLAYER_CONFLICT",
                severity = "MODERATE",
                involvedPlayerIds = "$p1,$p2",
                description = "训练场球员冲突",
                moraleImpact = -3
            )
            repository.insertEvent(event)
            applyTeamMoraleImpact(saveId, clubId, -3, date)
        }
    }

    /**
     * 更衣室兵变（集体不满）
     * 触发条件：
     * - 球队平均士气 <30
     * - 派系冲突数 ≥2
     * - 最近 5 场连败
     * - 兵变缓冲：本赛季已有过兵变则不再触发
     * - 队长缓冲：队长 leadership >70 时降低触发概率
     */
    suspend fun checkMutiny(date: LocalDate, saveId: Int, clubId: Int) {
        val season = repository.getCurrentSeason(saveId)

        // 1. 缓冲检查
        if (mutinyGuard.shouldSuppressEvent(saveId, clubId, season, "MUTINY")) return

        val avgMorale = repository.getAverageMorale(saveId, clubId) ?: 50.0
        if (avgMorale > config.mutiny.moraleThreshold) return  // 默认 30

        val factionConflicts = repository.countFactionConflicts(saveId, clubId)
        if (factionConflicts < config.mutiny.minFactionConflicts) return  // 默认 2

        val recentForm = repository.getRecentForm(saveId, clubId, 5)
        val consecutiveLosses = countConsecutiveLosses(recentForm)
        if (consecutiveLosses < config.mutiny.minConsecutiveLosses) return  // 默认 5

        // 2. 队长缓冲
        val captain = repository.getActiveCaptain(saveId, clubId)
        val captainBuffer = if (captain != null && captain.leadership > 70) {
            config.mutiny.captainBufferProbability  // 默认 0.5
        } else 0.0

        // 3. 触发概率
        val triggerProbability = config.mutiny.baseProbability - captainBuffer
        if (Math.random() > triggerProbability) return

        // 4. 触发兵变
        val event = DressingRoomEventEntity(
            saveId = saveId, clubId = clubId,
            eventDate = date.toString(), eventSeason = season,
            eventType = "MUTINY",
            severity = "CRITICAL",
            involvedPlayerIds = unhappyPlayers(saveId, clubId).joinToString(","),
            involvedFactionIds = repository.getActiveFactions(saveId, clubId).joinToString(",") { it.id.toString() },
            description = "更衣室兵变：球员集体对主帅表达不满",
            moraleImpact = -10
        )
        repository.insertEvent(event)
        applyTeamMoraleImpact(saveId, clubId, -10, date)

        // 5. 标记本赛季已发生兵变，防止再次触发
        mutinyGuard.markMutinyOccurred(saveId, clubId, season)
    }

    /**
     * 队长公开支持 / 批评经理
     * 触发条件：队长 morale <30 或 队长与主帅关系 <30
     */
    suspend fun checkCaptainStatement(date: LocalDate, saveId: Int, clubId: Int) {
        val captain = repository.getActiveCaptain(saveId, clubId) ?: return
        val captainMorale = repository.getMorale(saveId, captain.playerId)?.morale ?: 50

        val season = repository.getCurrentSeason(saveId)

            severity = if (isSupport) "MINOR" else "MAJOR",
            involvedPlayerIds = captain.playerId.toString(),
            description = if (isSupport) "队长公开支持主帅" else "队长公开批评主帅",
            moraleImpact = if (isSupport) 5 else -8
        )
        repository.insertEvent(event)
        applyTeamMoraleImpact(saveId, clubId, event.moraleImpact, date)
    }

    /**

    private suspend fun applyTeamMoraleImpact(
        saveId: Int, clubId: Int, delta: Int, date: LocalDate
    ) {
        val players = repository.getPlayersByClub(saveId, clubId)
        for (player in players) {
            val current = repository.getMorale(saveId, player.playerId)?.morale ?: continue
            val newMorale = (current + delta).coerceIn(0, 100)
            repository.updateMorale(saveId, player.playerId, newMorale, MoraleLevel.fromScore(newMorale).name, date.toString())
        }
    }

            if (match.isLoss) count++ else break

/**
 * 更衣室兵变缓冲机制
 * 防止集体崩盘 + 防止同赛季多次严重事件

class MutinyGuard(
    private val repository: DressingRoomRepository,
    private val config: DressingRoomConfig
) {

    /**
     * 判断是否应该抑制某类事件
     * 规则：
     * - MUTINY：每赛季最多 1 次
     * - CAPTAIN_STATEMENT：每赛季最多 2 次
     * - PLAYER_CONFLICT：每赛季最多 3 次
     * - PLAYTIME_UNHAPPY：每赛季最多 5 次
     */
    suspend fun shouldSuppressEvent(
        saveId: Int, clubId: Int, season: Int, eventType: String
    ): Boolean {
        val countThisSeason = repository.countEventsThisSeason(saveId, clubId, season, eventType)
        val limit = config.eventLimits[eventType] ?: return false
        return countThisSeason >= limit
    }

    suspend fun markMutinyOccurred(saveId: Int, clubId: Int, season: Int) {
        repository.markMutinyOccurred(saveId, clubId, season)
    }
}
```

### 1. 士气计算（多因子加权）

```
function calculateMorale(player, saveId, clubId, date):
    # 12 因子评分（每个 0-1）
    factors = {
        playing_time:        calculatePlayingTimeFactor(player, seasonStats)
        match_result:        calculateMatchResultFactor(saveId, clubId, playerId)
        personal_performance: calculatePersonalPerformanceFactor(seasonStats)
        contract_status:     contractBridge.calculateContractFactor(saveId, playerId, date)
        tactical_fit:        calculateTacticalFitFactor(player)
        teammate_relation:   calculateTeammateRelationFactor(saveId, clubId, playerId)
        coach_relation:      calculateCoachRelationFactor(saveId, playerId)
        team_atmosphere:     calculateTeamAtmosphereFactor(saveId, clubId)
        club_achievement:    calculateClubAchievementFactor(saveId, clubId)
        personal_honor:      calculatePersonalHonorFactor(saveId, playerId)
        injury_status:       calculateInjuryFactor(player, date)
        life_event:          calculateLifeEventFactor(player, date)
    }

    # 权重（任务描述 12 因子，总和 1.0）
    weights = config.morale_weights  # {playing_time:0.20, match_result:0.15, ...}

    # 加权求和
    morale_score = sum(factors[k] * weights[k] for k in factors) * 100

    # 钳制 0-100
    return clamp(morale_score, 0, 100)

function moraleLevel(morale):
    if morale >= 90: return EXTREME_HIGH   # 超常发挥 +10%
    if morale >= 75: return HIGH           # 正常发挥
    if morale >= 50: return MID            # 略低发挥 -5%
    if morale >= 30: return LOW            # 发挥下降 -15%
    return EXTREME_LOW                     # 严重影响 -30%，可能要求离队
```

### 2. 派系形成算法

```
function updateFactionMembership(saveId, clubId):
    players = getPlayersByClub(saveId, clubId)
    existing = getActiveFactions(saveId, clubId)

    # 1. 国籍派系
    byNationality = groupBy(players, p -> p.nationality)
    for (nation, members) in byNationality:
        if members.size >= 3:
            ensureFaction(NATIONALITY, "NATIONALITY:" + nation, members)

    # 2. 语言派系（与国籍互斥）
    if no Nationality faction exists OR allowCoexist:
        byLanguage = groupBy(players, p -> p.language)
        for (lang, members) in byLanguage:
            if members.size >= 3:
                ensureFaction(LANGUAGE, "LANGUAGE:" + lang, members)

    # 3. 老将 vs 年轻派系（成对生成）
    veterans = filter(players, p -> p.age >= 30)
    youngsters = filter(players, p -> p.age <= 21)
    if veterans.size >= 3 AND youngsters.size >= 3:
        ensureFaction(VETERAN_YOUTH, "VETERAN_YOUTH:VETERAN", veterans)
        ensureFaction(VETERAN_YOUTH, "VETERAN_YOUTH:YOUTH", youngsters)

    # 4. 青训 vs 外购派系
    academy = filter(players, p -> p.isAcademyProduct)
    bought = filter(players, p -> not p.isAcademyProduct)
    if academy.size >= 3 AND bought.size >= 3:
        ensureFaction(ACADEMY_BOUGHT, "ACADEMY_BOUGHT:ACADEMY", academy)
        ensureFaction(ACADEMY_BOUGHT, "ACADEMY_BOUGHT:BOUGHT", bought)

    # 5. 核心派系
    core = filter(players, p -> p.squadRole in ["starter", "key_player"])
    if core.size >= 3:
        ensureFaction(CORE_FRINGE, "CORE_FRINGE:CORE", core)

    # 6. 边缘派系
    fringe = filter(players, p -> p.squadRole in ["backup", "fringe"])
    if fringe.size >= 3:
        ensureFaction(CORE_FRINGE, "CORE_FRINGE:FRINGE", fringe)

    # 7. 解散成员 <2 的派系
    for faction in existing:
        if countMembers(faction.id) < 2:
            deactivate(faction.id)

    # 8. 上限保护
    enforceMaxFactionsLimit(saveId, clubId, max=6)

function ensureFaction(type, key, members):
    existing = findByKey(key)
    if existing != null:
        syncMembers(existing.id, members)  # 增量更新
    else:
        leader = selectLeader(members)  # leadership 最高 + 主力优先
        factionId = insertFaction(type, key, leader)
        for member in members:
            insertMember(factionId, member, role=LEADER if member==leader else MEMBER)
```

### 3. 派系影响传播

```
function propagateFactionMorale(saveId, clubId):
    factions = getActiveFactions(saveId, clubId)

    for faction in factions:
        members = getMembers(faction.id)
        if members.size < 2: continue

        morales = [(m.playerId, getMorale(m.playerId)) for m in members]
        avg = mean([m.morale for m in morales])
        leaderMorale = getMorale(faction.leaderPlayerId)
        leaderDelta = leaderMorale - 50  # 领袖偏离均值

        # 1. 领袖影响传播（衰减 0.5）
        for (playerId, morale) in morales:
            if playerId == faction.leaderPlayerId: continue
            pullToAvg = (avg - morale) * 0.05            # 拉近派系均值 5%
            leaderInfluence = leaderDelta * 0.5 * 0.5     # 衰减系数
            newMorale = clamp(morale + pullToAvg + leaderInfluence, 0, 100)
            updateMorale(playerId, newMorale)

        # 2. 不满传染：>50% 成员 morale<40 → 全员 -3
        unhappyCount = count(morales, m -> m.morale < 40)
        if unhappyCount > members.size / 2:
            for (playerId, morale) in morales:
                newMorale = clamp(morale - 3, 0, 100)
                updateMorale(playerId, newMorale)
```

### 4. 谈话效果计算

```
function calculateConversationOutcome(player, type, tone):
    currentMorale = getMorale(player.id)
    professionalism = getProfessionalism(player.id)
    temperament = getTemperament(player.id)

    # 基础士气变化
    moraleDelta = type.baseMoraleDelta
    # PRIVATE_TALK:0, PUBLIC_SUPPORT:+8, PUBLIC_CRITICISM:-6,
    # PLAYTIME_PROMISE:+5, TACTICAL_PROMISE:+3, WARNING:-3

    # 性格修正
    proMod = (professionalism - 50) / 100  # -0.5 ~ 0.5
    tempMod = (temperament - 50) / 100

    if type in [PUBLIC_CRITICISM, WARNING]:
        # 高 professionalism 减少负效果
        moraleDelta += proMod * 5
        # 低 temperament 加重反弹
        if temperament < 30:
            moraleDelta -= config.temperamentBackfirePenalty  # -3

    if type in [PUBLIC_SUPPORT, PLAYTIME_PROMISE] AND currentMorale < 40:
        # 不满球员得到支持效果加倍
        moraleDelta += config.unhappySupportBonus  # +3

    # 语气匹配度
    if tone == SUPPORTIVE AND currentMorale < 30:
        moraleDelta *= 1.2
    if tone == CRITICAL AND currentMorale > 70:
        moraleDelta *= 0.8  # 高士气时批评反弹

    moraleDelta = int(moraleDelta)

    # 忠诚度变化
    loyaltyDelta = {
        PUBLIC_SUPPORT:    +2,
        PUBLIC_CRITICISM:  -3,
        PLAYTIME_PROMISE:  +1,
        WARNING:           -5
    }[type] or 0

    # 反弹风险：低 pro + 低 morale + WARNING → 触发转会要求
    triggeredTransferRequest = (
        type == WARNING AND
        professionalism < 40 AND
        currentMorale < 30 AND
        random() < config.transferRequestProbability  # 0.2
    )

    # 结果分类
    if moraleDelta > 3: outcome = SUCCESS
    elif moraleDelta < -3: outcome = BACKFIRE
    else: outcome = NEUTRAL

    return {moraleDelta, loyaltyDelta, outcome, triggeredTransferRequest}

function executeConversation(player, type, tone, promise):
    # 冷却检查（默认 3 天）
    if lastConversationDate(player) within 3 days:
        return NEUTRAL (cooldown)

    outcome = calculateConversationOutcome(player, type, tone)

    # 应用士气变化
    newMorale = clamp(getMorale(player) + outcome.moraleDelta, 0, 100)
    updateMorale(player, newMorale)
    adjustLoyalty(player, outcome.loyaltyDelta)

    # 公开谈话影响派系
    if type.isPublic:
        faction = getFactionMembership(player)
        if faction != null:
            factionDelta = {
                PUBLIC_SUPPORT: +2,
                PUBLIC_CRITICISM: -3
            }[type] or 0
            for member in getFactionMembers(faction):
                if member != player:
                    adjustMorale(member, factionDelta)

    # 创建承诺
    if type == PLAYTIME_PROMISE AND promise != null:
        createPromise(player, promise.minutes, promise.deadline)

    # 触发转会要求
    if outcome.triggeredTransferRequest:
        triggerTransferRequest(player, reason="警告后球员不满")

    # 记录日志
    insertConversationLog(player, type, outcome)
```

```
function checkMutiny(date, saveId, clubId):
    season = getCurrentSeason()

    # 1. 缓冲检查：本赛季已发生过兵变则不再触发
    if countMutinyThisSeason(saveId, clubId, season) >= 1:
        return

    # 2. 触发条件检查
    avgMorale = getAverageMorale(saveId, clubId)
    if avgMorale > 30: return  # 球队士气必须 <30

    factionConflicts = countFactionConflicts(saveId, clubId)
    if factionConflicts < 2: return  # 派系冲突必须 ≥2

    recentForm = getRecentForm(saveId, clubId, matches=5)
    consecutiveLosses = countConsecutiveLosses(recentForm)
    if consecutiveLosses < 5: return  # 必须连败 5 场以上

    # 3. 队长缓冲
    captain = getActiveCaptain(saveId, clubId)
    captainBuffer = 0
    if captain != null AND captain.leadership > 70:
        captainBuffer = 0.5  # 队长缓冲概率 50%

    # 4. 触发概率
    triggerProbability = config.mutiny.baseProbability - captainBuffer  # 默认 0.7 - 0.5 = 0.2
    if random() > triggerProbability: return

    # 5. 触发兵变事件
    insertEvent(MUTINY, severity=CRITICAL, moraleImpact=-10)
    applyTeamMoraleImpact(saveId, clubId, -10)
    markMutinyOccurred(saveId, clubId, season)  # 防止再次触发

function checkUnrestAccumulation(date, saveId, clubId):
    # 每周一调用
    playersNeedingTalk = getPlayersNeedingTalk(saveId, clubId, threshold=3)
    for player in playersNeedingTalk:
        if eventLimitReached(saveId, clubId, "PLAYTIME_UNHAPPY", limit=5):
            continue
        severity = "MAJOR" if player.unrestAccumulator >= 5 else "MODERATE"
        insertEvent(PLAYTIME_UNHAPPY, severity, moraleImpact=-2)
        resetUnrest(player)
        applyTeamMoraleImpact(saveId, clubId, -2)
```

## 七、交互流程

### 1. 玩家发起谈话流程

```
1. 玩家进入更衣室页 → 不满球员 Tab
2. 看到一名 morale=25 的球员（极低），点击 "单独谈话" 按钮
3. 弹出 ConversationDialog
4. 选择类型：PRIVATE_TALK
5. 选择语气：SUPPORTIVE
6. 系统预测：基于该球员 professionalism=60, temperament=70
   预期士气 +3 ~ +8（提示："该球员职业态度良好，单独支持性谈话应有效"）
7. 玩家确认 → 调用 DressingRoomManager.initiateConversation
8. ConversationManager.execute 执行
   - 检查冷却（最近 3 天无谈话）
   - 计算 moraleDelta = 0 * 1.2（SUPPORTIVE + 低士气）= 0... 但 PRIVATE_TALK 基础 0
   - 调整为 +5（professionalism 修正）
   - updateMorale(player, 30)
   - insertConversationLog
9. 返回 ConversationOutcome（moraleDelta=+5, outcome=SUCCESS）
10. UI 显示球员反应："谢谢教练，我会努力训练。"
11. 玩家返回列表，看到该球员士气变为 30（LOW）
```

### 2. 更换队长流程

```
1. 玩家进入更衣室页 → 队长 & 核心球员 Tab
2. 看到当前队长 morale=35（LOW），影响力仅 45
3. 点击 "调整队长" 按钮
4. 弹出 ChangeCaptainDialog
5. 列出候选球员，按 leadership 排序：
   - Player A: leadership=85, age=28, starter
   - Player B: leadership=72, age=26, key_player
   - Player C: leadership=68, age=31, starter
6. 玩家选择 Player A
7. 显示风险提示：
   - 原队长士气 -15
   - 原队长所在派系士气 -5
   - 新队长士气 +10
8. 玩家确认 → 调用 DressingRoomManager.changeCaptain
9. CaptainManager.appointCaptain 执行
   - revoke 原队长
   - 原队长 morale -15（35 → 20）
   - 原队长派系 morale -5
   - insertCaptaincy（新队长）
   - 新队长 morale +10
   - 新队长派系 morale +3
10. EventTrigger.onCaptainChanged 触发 FACTION_CLASH 事件
11. UI 显示："Player A 已被任命为新队长。原队长派系表达不满。"
```

### 3. 更衣室兵变流程

```
1. 玩家球队连败 6 场，球队士气降至 25
2. 派系冲突数 = 3（巴西帮 vs 阿根廷帮，老将 vs 年轻，青训 vs 外购）
3. 周一推进时，EventTrigger.checkMutiny 触发
4. 检查缓冲：
   - 本赛季未发生过兵变 ✓
   - avgMorale=25 < 30 ✓
   - factionConflicts=3 ≥ 2 ✓
   - consecutiveLosses=6 ≥ 5 ✓
5. 队长缓冲：队长 leadership=65，未 >70，无缓冲
6. 触发概率：0.7 - 0 = 0.7
7. random() = 0.4 < 0.7 → 触发兵变
8. 插入 MUTINY 事件，severity=CRITICAL，moraleImpact=-10
9. 全队士气 -10（25 → 15）
10. markMutinyOccurred 防止本赛季再次触发
11. UI 显示红色横幅："⚠️ 更衣室兵变！球员集体对主帅表达不满"
12. 玩家必须解决：发起谈话 / 更换队长 / 出售带头者
13. T22 董事会收到通知：dressingRoomUnrest 上升，sack_score 受影响
```

### 5. 比赛日士气影响流程（与 T02 集成）

```
    → moraleCalculator.applyMatchImpact(date, saveId, clubId, matchResult, playerStats)
      for each player in playerStats:
        # 上场时间影响
        if minutesPlayed >= 75: delta += +3
        elif minutesPlayed >= 30: delta += +1
        elif minutesPlayed > 0: delta += -1
        else: delta += -3  # 未上场

        # 比赛结果影响
        if isWin: delta += +4
        elif isDraw: delta += +1
        else: delta += -3

        # 个人表现影响
        if rating >= 8.0: delta += +3
        elif rating >= 7.0: delta += +1
        elif rating < 5.5: delta += -2

        # 钳制单场变化上限（避免剧烈波动）
        delta = clamp(delta, -8, +8)
        applyMoraleDelta(player, delta)

        # 长期未上场 + 重要比赛 → 不满累积
        if minutesPlayed == 0 AND match.isImportant:
          adjustUnrest(player, +1)
```

## 九、性能要求

### 1. 性能预算（接入 T07 每日推进 P95 ≤3 秒）

| 操作 | 触发频率 | 玩家俱乐部耗时 | 活跃范围每队耗时 | 备注 |
|------|----------|----------------|------------------|------|
| 日度士气衰减 | 每日 | ≤50ms | ≤10ms | 仅回归 + 噪声，简单 SQL UPDATE |
| 日度承诺检查 | 每日 | ≤10ms | 跳过 | 仅玩家俱乐部 |
| 周一不满累积 | 每周一 | ≤20ms | 跳过 | 仅玩家俱乐部 |
| 周度派系传染 | 每周一 | ≤30ms | ≤15ms | 派系成员查询 + 批量 UPDATE |
| 月度全因子重算 | 每月 1 日 | ≤200ms | ≤80ms | 12 因子 × 25 球员 = 300 次计算 |
| 队长影响 | 每月 1 日 | ≤10ms | ≤5ms | 单次全队 UPDATE |
| 事件触发检查 | 每周一 | ≤20ms | ≤10ms | 简单条件判断 |

**总耗时占比**：玩家俱乐部每日 ≤120ms，活跃范围 19 队每日 ≤570ms，总计 ≤700ms，占 T07 总预算 3000ms 的 ~23%。

### 2. 数据库优化

- **索引**：所有 Entity 已在 `save_id + club_id` 上建索引，确保按俱乐部查询高效
- **批量更新**：日度衰减使用单条 SQL `UPDATE ... WHERE save_id = ? AND club_id = ?`，避免逐球员循环
- **派系缓存**：派系数据每周才变，可在 `cache.db` 缓存一周
- **士气趋势**：30 天滑动窗口数据量小（25 球员 × 30 天 = 750 行），无需特殊优化
- **事件归档**：赛季结束后由 T19 将 `dressing_room_event` 迁入 `history.db`，保持 save.db 表小

### 3. 内存优化

- `DressingRoomSnapshot` 仅在 UI 进入更衣室页时生成，离开后立即释放
- 派系成员列表懒加载，仅展开派系卡片时查询
- 12 因子雷达图数据按需计算，不预加载

## 十四、风险与应对

### 1. 士气剧烈波动

- **表现**：球员士气单日涨跌 >20，导致比赛表现极不稳定
- **应对**：
  - 单场士气变化钳制在 ±8 以内（`match_impact.max_delta_per_match`）
  - 日度衰减回归率仅 0.02，向 50 缓慢回归
  - 月度全因子重算结果与当前士气差距过大时分步过渡（V2 优化）
- **红线**：不直接钳制月度重算结果，避免士气"卡死"

### 2. 派系传染过强

- **表现**：一名球员士气低 → 全派系跟随低 → 全队崩盘
- **应对**：
  - 派系领袖影响衰减系数 0.5（`leader_propagation_rate`）
  - 派系均值拉近仅 5%（`pull_to_avg_rate`）
  - 不满传染需要 >50% 成员低于 40 才触发
  - 单俱乐部活跃派系 ≤6，避免传染路径过多
- **红线**：派系影响每周只传播 1 次，不每日传染

### 3. 兵变频发导致玩家体验差

- **表现**：玩家稍连败就触发兵变，无法翻身
- **应对**：
  - 兵变需 4 个条件同时满足（士气 <30 + 派系冲突 ≥2 + 连败 ≥5 + 概率）
  - 队长 leadership >70 提供 50% 缓冲概率
  - 同赛季最多 1 次兵变（`MutinyGuard` 强制限制）
  - 兵变后给玩家 3 个解决路径（谈话 / 换队长 / 出售带头者）
- **红线**：不连续触发兵变，不让玩家陷入"死循环"

### 4. 谈话效果单一化

- **表现**：所有球员对所有谈话类型反应相同，缺乏差异
- **应对**：
  - professionalism / temperament 三档修正（高 / 中 / 低）
  - 语气匹配度（SUPPORTIVE 在低士气时加倍，CRITICAL 在高士气时反弹）
  - 警告 + 低职业态度 + 低士气 → 触发转会要求（20% 概率）
- **红线**：谈话效果必须双向，不能只有正面

### 5. 性能影响每日推进

- **表现**：T23 接入后 T07 每日推进 P95 >3 秒
- **应对**：
  - 日度仅做衰减（SQL 批量 UPDATE），不重算因子
  - 月度全因子重算分批处理（`monthly_recalc_max_clubs_per_tick = 5`）
  - 活跃范围 19 队日度预算 ≤10ms/队
  - 派系数据缓存在 cache.db，每周才更新
- **红线**：T23 总耗时 ≤T07 总预算的 25%（即 ≤750ms）

### 6. 长程模拟更衣室持续崩盘

- **表现**：某俱乐部 20 年中连续 5+ 赛季平均士气 <30
- **应对**：
  - 日度衰减向 50 回归，避免长期卡在极低
  - 兵变后全队 -10，但下一赛季初重置 unrest_accumulator
  - AI 俱乐部（T18）的换帅逻辑会响应 dressingRoomUnrest，自动调整
  - Gate 3 长程测试用例校验"无球队连续 3 赛季士气 <30"
- **红线**：不强制重置士气，通过机制自然恢复

### 7. 与 T22 董事会集成耦合过紧

- **表现**：T23 士气波动 → T22 sack_score 频繁触发换帅
- **应对**：
  - `stabilityIndex` 是综合指标（士气 + 派系 + 不满数），不直接用单一士气
  - T22 的 `dressingRoomUnrest` 权重仅 0.15（在 sack_score 中）
  - T22 自身有连败触发阈值和耐心调整，不会因 T23 单次波动就换帅
- **红线**：T23 不直接调用 T22 接口，仅通过 `BoardMoraleBridge` 提供数据

### 8. 承诺系统被玩家滥用

- **表现**：玩家反复承诺出场时间但不兑现，导致士气反复下降
- **应对**：
  - 谈话冷却 3 天（`conversation.cooldown_days`）
  - 承诺未兑现 → 士气 -15 + 忠诚度 -10 + unrest +2（多重惩罚）
  - 多次违约 → unrest 累积触发 PLAYTIME_UNHAPPY 事件
- **红线**：不限制玩家承诺次数，但通过惩罚机制自然约束
