# T22 董事会 - 详细实现方案

> 任务编号：T22
> 周期：W82-W85（4 周）
> 优先级：P1（依赖 T06 赛程积分榜 + T07 每日推进 + T17 经济模型 + T18 AI 俱乐部画像 + T19 赛季归档）
> 依据：V0.2 `05_AI俱乐部决策模型.md` §九 换帅逻辑 + `11_赛季推进_AI俱乐部_世界模拟.md` §四 董事会 AI + `02_功能总表_35系统需求.md` §13 董事会目标系统
> 铁律：8 因子满意度 + 5 类赛季目标 + 6 类预算申请 + 8 类董事会事件 + 4 档解雇缓冲 + 与 T18 俱乐部画像协调（董事会野心 = 俱乐部画像野心）
> 定位：D 轨俱乐部管理系统的核心任务之一，连接成绩评估、财政管控、青训发展、解雇机制，是 20 年长程模拟中"经理人职业生涯"的承载层

## 一、业务目标与范围

### 1. 业务目标

- 让玩家感受到"董事会"作为雇主的存在感：目标设定 → 月度评估 → 赛季总结 → 奖惩反馈
- 让解雇机制有缓冲（警告 → 最后通牒 → 解雇），避免突兀 game over
- 让预算申请与俱乐部财政实力、满意度联动，避免无脑刷钱
- 与 T18 俱乐部画像协调：董事会野心 = 俱乐部画像野心（`ClubAiProfile.ambition`），豪门董事会要求前 4，保级队董事会要求保级
- 与 T17 经济模型协调：财政目标 = `wageToIncomeRatio` ≤ 70%，预算审批基于 `ClubFinancialState`

### 2. V1 做（W82-W85 范围内）

- 董事会页（二级页面）6 大模块：赛季目标 / 长期目标 / 财政目标 / 青训目标 / 满意度 / 预算申请
- 4 类操作：请求增加预算 / 请求改善设施 / 请求降低目标 / 查看评价
- 5 类赛季目标：联赛排名 / 杯赛 / 欧战 / 财政 / 青训
- 5 类长期目标：3 年/5 年规划 / 声望提升 / 球场扩建 / 青训设施升级 / 商业收入增长
- 8 因子满意度计算
- 6 类预算申请：转会预算 / 工资预算 / 青训设施 / 训练设施 / 医疗设施 / 球场扩建
- 8 类董事会事件：赛季初目标设定会议 / 赛季中评估 / 赛季末总结 / 目标未达成警告 / 连续未达成解雇威胁 / 董事会换届 / 老板注资 / 被收购传闻
- 4 档解雇机制：警告 → 最后通牒 → 解雇 + 紧急解雇（更衣室叛乱 / 财政危机 / 球迷抗议）
- 玩家俱乐部董事会全流程；AI 俱乐部董事会简化版（仅赛季末评估 + 解雇）

### 3. V1 不做（推迟到 M6+ / M7）

- 董事会成员个性化（仅做"董事会整体"实体，不做个别董事）
- 董事会换届的详细成员变动剧情
- 球迷股东大会详细模拟
- 俱乐部股票上市
- FFP（财政公平法案）详细审计流程
- 董事会媒体公开发言（T24 媒体系统独立做）
- 玩家自定目标（V1 仅由董事会设定，玩家不可改）
- 多人董事会投票机制

### 1. BoardroomManager（入口）

```kotlin
// api/BoardroomService.kt
class BoardroomService(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val cacheDb: CacheDatabase,
    private val goalManager: GoalManager,
    private val satisfactionCalculator: SatisfactionCalculator,
    private val budgetRequestHandler: BudgetRequestHandler,
    private val eventTrigger: BoardEventTrigger,
    private val dismissalManager: DismissalManager,
    private val aiBoardroomSimulator: AiBoardroomSimulator,
    private val config: BoardroomConfig,
    private val clubProfileService: ClubProfileService,    // T18
    private val economyService: EconomyService              // T17
) {

    /**
     * 赛季初：为指定俱乐部生成赛季目标
     * 由 T07 SeasonStartExecutor 调用
     */
    suspend fun generateSeasonTargets(saveId: Int, clubId: Int, seasonId: Int, currentDate: LocalDate) {
        goalManager.generateSeasonTargets(saveId, clubId, seasonId, currentDate)
    }

    /**
     * 月度：计算满意度快照
     * 由 T07 每月推进调用（月初）
     */
    suspend fun processMonthlyReview(saveId: Int, clubId: Int, currentDate: LocalDate, isPlayerClub: Boolean) {
        satisfactionCalculator.calculateAndSnapshot(saveId, clubId, currentDate)
        if (isPlayerClub) {
            eventTrigger.checkMidSeasonReviewTrigger(saveId, clubId, currentDate)
        }
    }

    /**
     * 赛季末：评估目标达成 + 解雇判定
     * 由 T19 SeasonArchiveExecutor 调用
     */
    suspend fun processSeasonEnd(saveId: Int, clubId: Int, seasonId: Int, currentDate: LocalDate): SeasonEndResult {
        val evaluation = goalManager.evaluateSeasonTargets(saveId, clubId, seasonId, currentDate)
        val dismissalDecision = dismissalManager.evaluate(saveId, clubId, seasonId, evaluation, currentDate)
        eventTrigger.triggerSeasonEndSummary(saveId, clubId, seasonId, evaluation, dismissalDecision)
        return SeasonEndResult(evaluation, dismissalDecision)
    }

    /**
     * 玩家：提交预算申请
     */
    suspend fun submitBudgetRequest(
        saveId: Int, clubId: Int, currentDate: LocalDate,
        requestType: BudgetRequestType, amount: Int, justification: String
    ): BudgetRequestResult {
        return budgetRequestHandler.submitAndReview(saveId, clubId, currentDate, requestType, amount, justification)
    }

    /**
     * 玩家：请求降低目标
     */
    suspend fun requestTargetLowering(saveId: Int, clubId: Int, seasonId: Int, currentDate: LocalDate): TargetLoweringResult {
        return goalManager.requestTargetLowering(saveId, clubId, seasonId, currentDate)
    }

    /**
     * AI 俱乐部董事会月度模拟（简化版）
     * 由 T18 AiScheduler 调用
     */
    suspend fun simulateAiClubMonthly(saveId: Int, clubId: Int, currentDate: LocalDate) {
        aiBoardroomSimulator.simulateMonthly(saveId, clubId, currentDate)
    }
}

data class SeasonEndResult(
    val evaluation: SeasonTargetEvaluation,
    val dismissalDecision: DismissalDecision
)
```

### 2. GoalManager（目标管理）

```kotlin
// goal/GoalManager.kt
class GoalManager(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val seasonTargetDao: SeasonTargetDao,
    private val longTermGoalDao: LongTermGoalDao,
    private val clubProfileService: ClubProfileService,    // T18
    private val economyService: EconomyService,            // T17
    private val config: BoardroomConfig
) {

    /**
     * 赛季初生成赛季目标
     * 规则：
     * - 联赛排名目标 = 基于 ClubAiProfile.ambition + 上赛季排名
     * - 杯赛目标 = 基于野心（野心 >70 → 夺冠，>50 → 半决赛，否则 8 强）
     * - 欧战目标 = 是否有欧战资格 + 野心
     * - 财政目标 = wageToIncomeRatio ≤ 70%（豪门）/ 80%（中游）/ 90%（保级）
     * - 青训目标 = boardYouthFocus 决定（青训型俱乐部 ≥3，其他 ≥1）
     */
    suspend fun generateSeasonTargets(saveId: Int, clubId: Int, seasonId: Int, currentDate: LocalDate) {
        val profile = clubProfileService.getProfile(clubId)
            ?: error("Club AI profile not found for club $clubId")
        val lastSeasonPosition = getLastSeasonPosition(saveId, clubId, seasonId - 1)
        val financial = economyService.getClubFinancialState(clubId, saveId)

        // 联赛排名目标
        val leagueTarget = calculateLeaguePositionTarget(profile, lastSeasonPosition)
        val leagueImportance = "CORE"

        // 杯赛目标
        val cupTarget = when {
            profile.ambition > 75 -> "WIN"
            profile.ambition > 60 -> "FINAL"
            profile.ambition > 45 -> "SEMI_FINAL"
            else -> "QUARTER_FINAL"
        }
        val cupImportance = if (profile.ambition > 60) "CORE" else "SECONDARY"

        // 欧战目标（基于上赛季是否获得欧战资格）
        val hasEuropeanQualification = checkEuropeanQualification(saveId, clubId, seasonId - 1)
        val europeanTarget = if (!hasEuropeanQualification) "NONE"
            else when {
                profile.ambition > 80 -> "QUARTER_FINAL"
                profile.ambition > 65 -> "ROUND_OF_16"
                else -> "GROUP_STAGE"
            }
        val europeanImportance = if (hasEuropeanQualification && profile.ambition > 65) "CORE" else "SECONDARY"

        // 财政目标
        val wageRatioTarget = when (profile.clubType) {
            ClubType.ELITE_CONTENDER -> 0.70
            ClubType.MONEY_EXPANSION -> 0.80
            ClubType.YOUTH_DEVELOPER -> 0.75
            ClubType.PROFIT_MAKER -> 0.65
            ClubType.RELEGATION_FIGHTER -> 0.90
        }
        val financialImportance = if (profile.wageStrictness > 60) "CORE" else "SECONDARY"

        // 青训目标
        val youthPromotionTarget = when {
            profile.youthPreference > 70 -> 3
            profile.youthPreference > 50 -> 2
            else -> 1
        }

        val target = SeasonTargetEntity(
            saveId = saveId,
            clubId = clubId,
            seasonId = seasonId,
            setAt = currentDate.format(ISO_LOCAL_DATE),
            leaguePositionTarget = leagueTarget,
            leaguePositionImportance = leagueImportance,
            cupTarget = cupTarget,
            cupImportance = cupImportance,
            europeanTarget = europeanTarget,
            europeanImportance = europeanImportance,
            financialWageRatioTarget = wageRatioTarget,
            financialImportance = financialImportance,
            youthPromotionTarget = youthPromotionTarget,
            evaluationStatus = "PENDING"
        )
        seasonTargetDao.upsert(target)

        // 生成 3 年长期目标（每 3 年生成一次）
        ensureLongTermGoals(saveId, clubId, currentDate, profile)
    }

    /**
     * 联赛排名目标计算
     * | 上赛季排名 | 野心 >75 | 野心 50-75 | 野心 <50 |
     * | 1-4       | 提升 1 位 | 维持     | 维持    |
     * | 5-10      | 前 4    | 前 6     | 前 10   |
     * | 11-17     | 前 10   | 前 12    | 保级（17）|
     * | 18-20     | 前 12   | 前 15    | 保级（17）|
     */
    private fun calculateLeaguePositionTarget(profile: ClubAiProfile, lastPosition: Int?): Int {
        val ambition = profile.ambition
        return when {
            lastPosition == null -> when {                 // 新赛季首季
                ambition > 75 -> 4
                ambition > 50 -> 8
                else -> 15
            }
            lastPosition <= 4 -> when {
                ambition > 75 -> (lastPosition - 1).coerceAtLeast(1)
                ambition > 50 -> lastPosition
                else -> lastPosition
            }
            lastPosition in 5..10 -> when {
                ambition > 75 -> 4
                ambition > 50 -> 6
                else -> 10
            }
            lastPosition in 11..17 -> when {
                ambition > 75 -> 10
                ambition > 50 -> 12
                else -> 17
            }
            else -> when {                                  // 降级队（升降级机制下可能仍在新赛季）
                ambition > 75 -> 12
                ambition > 50 -> 15
                else -> 17
            }
        }
    }

    /**
     * 赛季末评估目标达成
     */
    suspend fun evaluateSeasonTargets(
        saveId: Int, clubId: Int, seasonId: Int, currentDate: LocalDate
    ): SeasonTargetEvaluation {
        val target = seasonTargetDao.getBySeason(saveId, clubId, seasonId)
            ?: return SeasonTargetEvaluation.empty()

        val actualLeaguePosition = getActualLeaguePosition(saveId, clubId, seasonId)
        val actualCupResult = getActualCupResult(saveId, clubId, seasonId)
        val actualEuropeanResult = getActualEuropeanResult(saveId, clubId, seasonId)
        val actualWageRatio = economyService.getClubFinancialState(clubId, saveId).wageToIncomeRatio
        val actualYouthPromotions = countYouthPromotions(saveId, clubId, seasonId)

        val leagueScore = evaluateLeaguePosition(target.leaguePositionTarget, actualLeaguePosition)
        val cupScore = evaluateCupResult(target.cupTarget, actualCupResult)
        val europeanScore = evaluateEuropeanResult(target.europeanTarget, actualEuropeanResult)
        val financialScore = evaluateFinancial(target.financialWageRatioTarget, actualWageRatio)
        val youthScore = evaluateYouth(target.youthPromotionTarget, actualYouthPromotions)

        // 综合分（核心目标权重 0.7，次要目标权重 0.3）
        val coreWeight = 0.7
        val secondaryWeight = 0.3
        val coreScores = mutableListOf<Pair<Double, Double>>()  // (score, weight)
        val secondaryScores = mutableListOf<Pair<Double, Double>>()

        if (target.leaguePositionImportance == "CORE") coreScores.add(leagueScore to 0.35)
        else secondaryScores.add(leagueScore to 0.30)

        if (target.cupImportance == "CORE") coreScores.add(cupScore to 0.20)
        else secondaryScores.add(cupScore to 0.20)

        if (target.europeanImportance == "CORE") coreScores.add(europeanScore to 0.20)
        else secondaryScores.add(europeanScore to 0.15)

        if (target.financialImportance == "CORE") coreScores.add(financialScore to 0.15)
        else secondaryScores.add(financialScore to 0.20)

        secondaryScores.add(youthScore to 0.15)

        val coreTotal = if (coreScores.isNotEmpty())
            coreScores.sumOf { it.first * it.second } / coreScores.sumOf { it.second }
        else 0.0
        val secondaryTotal = if (secondaryScores.isNotEmpty())
            secondaryScores.sumOf { it.first * it.second } / secondaryScores.sumOf { it.second }
        else 0.0

        val overallScore = (coreTotal * coreWeight + secondaryTotal * secondaryWeight).coerceIn(0.0, 100.0)

        val status = when {
            overallScore >= 85 -> "ACHIEVED"
            overallScore >= 60 -> "PARTIALLY"
            else -> "FAILED"
        }

        seasonTargetDao.updateEvaluation(target.id, status, overallScore, currentDate.format(ISO_LOCAL_DATE))

        return SeasonTargetEvaluation(
            seasonId = seasonId,
            overallScore = overallScore,
            status = status,
            leagueScore = leagueScore,
            cupScore = cupScore,
            europeanScore = europeanScore,
            financialScore = financialScore,
            youthScore = youthScore,
            coreGoalFailed = listOf(
                target.leaguePositionImportance to leagueScore,
                target.cupImportance to cupScore,
                target.europeanImportance to europeanScore,
                target.financialImportance to financialScore
            ).filter { it.first == "CORE" && it.second < 50 }.isNotEmpty()
        )
    }

    private fun evaluateLeaguePosition(target: Int, actual: Int?): Double {
        if (actual == null) return 50.0
        return when {
            actual <= target -> 100.0                              // 达成或更好
            actual == target + 1 -> 70.0                           // 差 1 名
            actual == target + 2 -> 50.0
            actual <= target + 4 -> 30.0
            else -> 10.0
        }
    }

    private fun evaluateCupResult(target: String, actual: String?): Double {
        if (actual == null) return 50.0
        val order = listOf("GROUP_STAGE", "ROUND_OF_16", "QUARTER_FINAL", "SEMI_FINAL", "FINAL", "WIN")
        val targetIdx = order.indexOf(target)
        val actualIdx = order.indexOf(actual)
        return when {
            actualIdx >= targetIdx -> 100.0
            actualIdx == targetIdx - 1 -> 60.0
            actualIdx == targetIdx - 2 -> 40.0
            else -> 20.0
        }
    }

    private fun evaluateEuropeanResult(target: String, actual: String?): Double {
        if (target == "NONE") return 80.0  // 无欧战目标默认良好
        return evaluateCupResult(target, actual)
    }

    private fun evaluateFinancial(target: Double, actual: Double): Double {

            actual <= target + 0.05 -> 70.0
            actual <= target + 0.10 -> 50.0
            actual <= target + 0.20 -> 30.0

        }
```

```kotlin
// satisfaction/SatisfactionCalculator.kt
class SatisfactionCalculator(
    private val saveDb: SaveDatabase,
    private val satisfactionDao: BoardSatisfactionDao,
    private val economyService: EconomyService,
    private val clubProfileService: ClubProfileService,
    private val seasonTargetDao: SeasonTargetDao,
    private val config: BoardroomConfig
) {

    /**
     * 8 因子满意度计算
     *
     * overall_satisfaction =
     *   league_performance_score      * 0.20
     * + cup_performance_score         * 0.10
     * + financial_score               * 0.15
     * + fan_satisfaction_score        * 0.15
     * + transfer_market_score         * 0.10
     * + youth_development_score       * 0.10
     * + dressing_room_stability_score * 0.10
     * + manager_personal_reputation   * 0.10
     *
     * 权重可根据董事会性格微调（如 AMBITIOUS 董事会联赛权重 +0.05、青训 -0.05）
     */
    suspend fun calculateAndSnapshot(saveId: Int, clubId: Int, currentDate: LocalDate) {
        val profile = clubProfileService.getProfile(clubId) ?: return
        val board = saveDb.boardMemberDao().getByClub(clubId)
        val financial = economyService.getClubFinancialState(clubId, saveId)
        val clubState = saveDb.saveClubStateDao().getByClub(saveId, clubId)!!

        // 1. 联赛成绩（与赛季目标对比）
        val leagueScore = calculateLeaguePerformanceScore(saveId, clubId, profile)

        // 2. 杯赛成绩
        val cupScore = calculateCupPerformanceScore(saveId, clubId)

        // 3. 财政状况
        val financialScore = calculateFinancialScore(financial, profile)

        // 4. 球迷满意度（来自 save_club_state.fan_satisfaction）
        val fanScore = clubState.fanSatisfaction.toDouble()

        // 5. 转会市场表现（窗内净投入 / 预算执行率）
        val transferScore = calculateTransferMarketScore(saveId, clubId, currentDate)

        // 6. 青训发展（本季青训提拔数 / 目标）
        val youthScore = calculateYouthDevelopmentScore(saveId, clubId)

        // 7. 更衣室稳定（来自 save_club_state.dressing_room_morale）
        val dressingRoomScore = clubState.dressingRoomMorale.toDouble()

        // 8. 经理个人声望（玩家经理声望，V1 用俱乐部声望近似）
        val managerReputationScore = calculateManagerReputationScore(saveId, clubId)

        // 按董事会性格微调权重
        val adjustedWeights = adjustWeightsByBoardPersonality(config.satisfactionWeights, board?.chairmanPersonality)

        val overall = (
            leagueScore * adjustedWeights.league +
            cupScore * adjustedWeights.cup +
            financialScore * adjustedWeights.financial +
            fanScore * adjustedWeights.fan +
            transferScore * adjustedWeights.transfer +
            youthScore * adjustedWeights.youth +
            dressingRoomScore * adjustedWeights.dressingRoom +
            managerReputationScore * adjustedWeights.managerReputation
        ).coerceIn(0.0, 100.0)

        val level = when {
            overall >= 80 -> "EXCELLENT"
            overall >= 65 -> "GOOD"
            overall >= 50 -> "ACCEPTABLE"
            overall >= 35 -> "POOR"
            else -> "CRITICAL"
        }

        val lastSnapshot = satisfactionDao.getLatest(saveId, clubId, 1).firstOrNull()
        val trend = when {
            lastSnapshot == null -> "STABLE"
            overall > lastSnapshot.overallSatisfaction + 3 -> "RISING"
            overall < lastSnapshot.overallSatisfaction - 3 -> "FALLING"
            else -> "STABLE"
        }

        val snapshot = BoardSatisfactionEntity(
            saveId = saveId,
            clubId = clubId,
            snapshotDate = currentDate.format(ISO_LOCAL_DATE),
            leaguePerformanceScore = leagueScore,
            cupPerformanceScore = cupScore,
            financialScore = financialScore,
            fanSatisfactionScore = fanScore,
            transferMarketScore = transferScore,
            youthDevelopmentScore = youthScore,
            dressingRoomStabilityScore = dressingRoomScore,
            managerPersonalReputationScore = managerReputationScore,
            overallSatisfaction = overall,
            satisfactionLevel = level,
            trendDirection = trend
        )
        satisfactionDao.insert(snapshot)

        // 同步更新 save_club_state.board_satisfaction 冗余字段
        saveDb.saveClubStateDao().updateBoardSatisfaction(saveId, clubId, overall.toInt())

        // 同步缓存
        cacheDb.boardSatisfactionCacheDao().upsert(snapshot.toCache())
    }

    /**
     * 联赛成绩评分
     * 基于当前排名 vs 目标排名
     */
    private suspend fun calculateLeaguePerformanceScore(saveId: Int, clubId: Int, profile: ClubAiProfile): Double {
        val currentPos = getCurrentLeaguePosition(saveId, clubId) ?: return 50.0
        val targetPos = getCurrentSeasonTarget(saveId, clubId)?.leaguePositionTarget ?: return 50.0
        return when {
            currentPos <= targetPos -> 100.0
            currentPos == targetPos + 1 -> 80.0
            currentPos == targetPos + 2 -> 60.0
            currentPos <= targetPos + 4 -> 40.0
            else -> 20.0
        }
    }

    /**
     * 财政评分
     * 工资/收入比 + 盈利状况
     */
    private fun calculateFinancialScore(financial: ClubFinancialState, profile: ClubAiProfile): Double {
        val ratioScore = when {
            financial.wageToIncomeRatio < 0.55 -> 100.0
            financial.wageToIncomeRatio < 0.70 -> 80.0
            financial.wageToIncomeRatio < 0.85 -> 50.0
            else -> 20.0
        }
        val balanceScore = if (financial.balance > 0) 80.0 else 30.0
        // 工资控制严格的董事会更看重财政
        val weight = if (profile.wageStrictness > 60) 0.7 else 0.5
        return ratioScore * weight + balanceScore * (1 - weight)
    }

    /**
     * 转会市场评分
     * - 预算执行率（用了多少预算）
     * - 是否补强了短板位置
     */
    private suspend fun calculateTransferMarketScore(saveId: Int, clubId: Int, currentDate: LocalDate): Double {
        val lastWindow = getLastTransferWindowSummary(saveId, clubId, currentDate) ?: return 60.0
        val budgetUsageRatio = lastWindow.spent.toDouble() / lastWindow.budget.coerceAtLeast(1)
        val usageScore = when {
            budgetUsageRatio in 0.7..1.0 -> 100.0    // 合理使用
            budgetUsageRatio in 0.5..0.7 -> 80.0
            budgetUsageRatio < 0.5 -> 60.0           // 太保守
            else -> 50.0                              // 超支
        }
        val positionFilledScore = if (lastWindow.topNeedPositionFilled) 100.0 else 40.0
        return usageScore * 0.5 + positionFilledScore * 0.5
    }

    private suspend fun calculateYouthDevelopmentScore(saveId: Int, clubId: Int): Double {
        val target = getCurrentSeasonTarget(saveId, clubId)?.youthPromotionTarget ?: 1
        val actual = countYouthPromotionsThisSeason(saveId, clubId)
        return when {
            actual >= target -> 100.0
            actual == target - 1 -> 60.0
            actual == 0 && target > 0 -> 20.0
            else -> 40.0
        }
    }

    private fun calculateManagerReputationScore(saveId: Int, clubId: Int): Double {
        // V1 简化：经理声望 = 俱乐部声望（玩家经理跟随俱乐部声望）
        val clubState = saveDb.saveClubStateDao().getByClub(saveId, clubId)!!
        return clubState.reputation.toDouble()
    }

    /**
     * 按董事会性格调整权重
     * - PATIENT：青训/更衣室权重 +0.05，联赛 -0.05
     * - AMBITIOUS：联赛 +0.05，青训 -0.05
     * - RUTHLESS：联赛 +0.05，更衣室 -0.03
     * - HANDS_OFF：财政 +0.05，联赛 -0.03
     */
    private fun adjustWeightsByBoardPersonality(base: SatisfactionWeights, personality: String?): SatisfactionWeights {
        if (personality == null) return base
        return when (personality) {
            "PATIENT" -> base.copy(
                league = base.league - 0.05,
                youth = base.youth + 0.03,
                dressingRoom = base.dressingRoom + 0.02
            )
            "AMBITIOUS" -> base.copy(
                league = base.league + 0.05,
                youth = base.youth - 0.03,
                cup = base.cup + 0.02,
                financial = base.financial - 0.04
            )
            "RUTHLESS" -> base.copy(
                league = base.league + 0.05,
                dressingRoom = base.dressingRoom - 0.03,
                fan = base.fan + 0.02,
                financial = base.financial - 0.04
            )
            "HANDS_OFF" -> base.copy(
                league = base.league - 0.03,
                financial = base.financial + 0.05,
                cup = base.cup - 0.02
            )
            else -> base
        }
    }
}
```

### 4. BudgetRequestHandler（预算申请处理）

```kotlin
// budget/BudgetRequestHandler.kt
class BudgetRequestHandler(
    private val saveDb: SaveDatabase,
    private val budgetRequestDao: BudgetRequestDao,
    private val economyService: EconomyService,
    private val clubProfileService: ClubProfileService,
    private val satisfactionDao: BoardSatisfactionDao,
    private val config: BoardroomConfig
) {

    /**
     * 预算申请审批
     *
     * 审批策略：
     * | 满意度 | 财政健康 | 审批结果 |
     * | ≥80    | HEALTHY  | 100% 批准 |
     * | ≥80    | ACCEPTABLE | 80% 批准 |
     * | ≥65    | HEALTHY  | 80% 批准 |
     * | ≥65    | ACCEPTABLE | 60% 批准 |
     * | ≥50    | -        | 50% 批准 + 警告 |
     * | <50    | -        | 拒绝 + 冷却 30 天 |
     *
     * 设施类申请：额外校验俱乐部设施当前等级上限
     */
    suspend fun submitAndReview(
        saveId: Int, clubId: Int, currentDate: LocalDate,
        requestType: BudgetRequestType, amount: Int, justification: String
    ): BudgetRequestResult {
        // 1. 冷却期检查
        val cooldownDays = config.budgetRequestCooldownDays  // 默认 30
        val lastReviewed = budgetRequestDao.getLatestReviewed(
            saveId, clubId, requestType.name,
            currentDate.minusDays(cooldownDays.toLong()).format(ISO_LOCAL_DATE)
        )
        if (lastReviewed != null && lastReviewed.status == "REJECTED" &&
            lastReviewed.cooldownDaysRemaining > 0) {
            return BudgetRequestResult.rejectedDueToCooldown(lastReviewed.cooldownDaysRemaining)
        }

        // 2. 限制申请频率（同类型 30 天内不可重复申请）
        if (lastReviewed != null) {
            val daysSince = java.time.Period.between(
                LocalDate.parse(lastReviewed.reviewedAt), currentDate
            ).days
            if (daysSince < cooldownDays) {
                return BudgetRequestResult.rejectedDueToCooldown(cooldownDays - daysSince)
            }
        }

        // 3. 计算审批额度
        val satisfaction = satisfactionDao.getLatest(saveId, clubId, 1).firstOrNull()?.overallSatisfaction ?: 50.0
        val financial = economyService.getClubFinancialState(clubId, saveId)
        val healthReport = economyService.checkFinancialHealth(clubId, saveId)

        val approvalRatio = calculateApprovalRatio(satisfaction, healthReport.level, requestType)
        val approvedAmount = (amount * approvalRatio).toInt()

        // 4. 财政承受力二次校验（不超过俱乐部年收入 50%）
        val maxAffordable = (financial.totalIncome * config.maxBudgetRequestIncomeRatio).toInt()
        val finalAmount = minOf(approvedAmount, maxAffordable)

        // 5. 设施类申请：校验等级上限
        val facilityLevelCheck = if (requestType.isFacilityType()) {
            checkFacilityLevelCap(saveId, clubId, requestType)
        } else null

        val status = when {
            facilityLevelCheck?.capped == true -> "NEGOTIATED"
            finalAmount <= 0 -> "REJECTED"
            finalAmount < amount -> "NEGOTIATED"
            else -> "APPROVED"
        }

        val boardResponse = buildBoardResponse(status, finalAmount, satisfaction, healthReport.level)

        val request = BudgetRequestEntity(
            saveId = saveId, clubId = clubId,
            requestDate = currentDate.format(ISO_LOCAL_DATE),
            requestType = requestType.name,
            requestedAmount = amount,
            justification = justification,
            currentSatisfaction = satisfaction,
            currentFinancialHealth = healthReport.level.name,
            status = status,
            approvedAmount = finalAmount,
            boardResponse = boardResponse,
            reviewedAt = currentDate.format(ISO_LOCAL_DATE),
            cooldownDaysRemaining = if (status == "REJECTED") config.budgetRequestCooldownDays else 0
        )
        budgetRequestDao.insert(request)

        // 6. 执行预算变更
        if (status in listOf("APPROVED", "NEGOTIATED")) {
            applyBudgetChange(saveId, clubId, requestType, finalAmount)
        }

        return BudgetRequestResult(
            status = status, approvedAmount = finalAmount,
            boardResponse = boardResponse, satisfactionImpact = -1  // 申请即使批准也会小扣满意度（频繁要钱）
        )
    }

    private fun calculateApprovalRatio(
        satisfaction: Double, healthLevel: FinancialWarningLevel, requestType: BudgetRequestType
    ): Double {
        // 设施类申请额外严格
        val facilityPenalty = if (requestType.isFacilityType()) 0.85 else 1.0
        return when {
            satisfaction >= 80 && healthLevel == FinancialWarningLevel.HEALTHY -> 1.00 * facilityPenalty
            satisfaction >= 80 && healthLevel == FinancialWarningLevel.ACCEPTABLE -> 0.80 * facilityPenalty
            satisfaction >= 65 && healthLevel == FinancialWarningLevel.HEALTHY -> 0.80 * facilityPenalty
            satisfaction >= 65 && healthLevel == FinancialWarningLevel.ACCEPTABLE -> 0.60 * facilityPenalty
            satisfaction >= 50 && healthLevel != FinancialWarningLevel.HIGH_RISK -> 0.50 * facilityPenalty
            else -> 0.0  // 拒绝
        }
    }

    private fun applyBudgetChange(saveId: Int, clubId: Int, requestType: BudgetRequestType, amount: Int) {
        when (requestType) {
            BudgetRequestType.TRANSFER_BUDGET -> {
                saveDb.saveClubStateDao().addTransferBudget(saveId, clubId, amount)
            }
            BudgetRequestType.WAGE_BUDGET -> {
                saveDb.saveClubStateDao().addWageBudget(saveId, clubId, amount)
            }
            // 设施类投资：增加设施等级（V1 简化，每笔投资升 1 级，需达阈值）
```

```kotlin
// dismissal/DismissalManager.kt
class DismissalManager(
    private val saveDb: SaveDatabase,
    private val seasonTargetDao: SeasonTargetDao,
    private val satisfactionDao: BoardSatisfactionDao,
    private val config: BoardroomConfig,
    private val clubProfileService: ClubProfileService
) {

    /**
     * 解雇判定主流程
     *
     * 4 档解雇机制：
     * 1. NONE         - 无解雇风险
     * 2. WARNING      - 连续 2 赛季核心目标未达成 → 警告
     * 3. ULTIMATUM    - 连续 3 赛季核心目标未达成 → 最后通牒
     * 4. DISMISS      - 连续 4 赛季核心目标未达成 OR 最终通牒赛季仍未达成 → 解雇
     *
     * 紧急解雇（不受缓冲约束，立即触发）：
     * - 严重更衣室叛乱（dressing_room_morale < 20 持续 1 月以上）
     * - 严重财政危机（wageToIncomeRatio > 1.0 持续 1 赛季）
     * - 球迷大规模抗议（fan_satisfaction < 15 持续 1 月以上）
     */
    suspend fun evaluate(
        saveId: Int, clubId: Int, seasonId: Int,
        seasonEvaluation: SeasonTargetEvaluation, currentDate: LocalDate
    ): DismissalDecision {
        val profile = clubProfileService.getProfile(clubId) ?: return DismissalDecision.none()
        val clubState = saveDb.saveClubStateDao().getByClub(saveId, clubId)!!
        val financial = economyService.getClubFinancialState(clubId, saveId)

        // 1. 紧急解雇检查（最高优先级）
        val emergencyReason = checkEmergencyDismissal(clubState, financial, profile)
        if (emergencyReason != null) {
            return DismissalDecision(
                shouldDismiss = true,
                warningLevel = DismissalLevel.DISMISS,
                reason = emergencyReason,
                isEmergency = true
            )
        }

        // 2. 计算连续未达成核心目标赛季数
        val sinceSeasonId = seasonId - config.dismissalLookbackSeasons + 1  // 默认看 4 赛季
        val consecutiveCoreFailed = countConsecutiveCoreFailedSeasons(saveId, clubId, seasonId, seasonEvaluation)

        // 3. 董事会耐心调整阈值
        // patience 越低，触发解雇越快
        val patienceAdjustment = (profile.patienceWithManager - 50) / 50.0  // -1.0 ~ +1.0
        val effectiveWarningThreshold = config.warningTriggerSeasons - patienceAdjustment.toInt()  // 默认 2，耐心低变 1
        val effectiveUltimatumThreshold = config.ultimatumTriggerSeasons - patienceAdjustment.toInt()  // 默认 3
        val effectiveDismissThreshold = config.dismissTriggerSeasons - patienceAdjustment.toInt()  // 默认 4

        // 4. 当前满意度作为额外触发条件
        val currentSatisfaction = satisfactionDao.getLatest(saveId, clubId, 1).firstOrNull()?.overallSatisfaction ?: 50.0

        val warningLevel = when {
            consecutiveCoreFailed >= effectiveDismissThreshold ||
            (consecutiveCoreFailed >= effectiveUltimatumThreshold && seasonEvaluation.coreGoalFailed) -> {
                DismissalLevel.DISMISS
            }
            consecutiveCoreFailed >= effectiveUltimatumThreshold ||
            (consecutiveCoreFailed >= effectiveWarningThreshold && seasonEvaluation.coreGoalFailed) -> {
                // 如果上季已经在 ULTIMATUM，本季仍未达成 → DISMISS
                val lastWarning = getLastWarningLevel(saveId, clubId, seasonId - 1)
                if (lastWarning == DismissalLevel.ULTIMATUM && seasonEvaluation.coreGoalFailed) {
                    DismissalLevel.DISMISS
                } else {
                    DismissalLevel.ULTIMATUM
                }
            }
            consecutiveCoreFailed >= effectiveWarningThreshold -> {
                // 警告级
                val lastWarning = getLastWarningLevel(saveId, clubId, seasonId - 1)
                if (lastWarning == DismissalLevel.WARNING) {
                    // 上季已警告，本季仍未达成 → 升级为最后通牒
                    DismissalLevel.ULTIMATUM
                } else {
                    DismissalLevel.WARNING
                }
            }
            currentSatisfaction < 25 -> DismissalLevel.WARNING  // 满意度极低也触发警告
            else -> DismissalLevel.NONE
        }

        val shouldDismiss = warningLevel == DismissalLevel.DISMISS
        val reason = buildDismissalReason(warningLevel, consecutiveCoreFailed, seasonEvaluation, currentSatisfaction)

        // 记录警告等级
        saveWarningLevel(saveId, clubId, seasonId, warningLevel)

        return DismissalDecision(
            shouldDismiss = shouldDismiss,
            warningLevel = warningLevel,
            reason = reason,
            consecutiveCoreFailedSeasons = consecutiveCoreFailed,
            isEmergency = false
        )
    }

    /**
     * 紧急解雇检查
     */
    private suspend fun checkEmergencyDismissal(
        clubState: SaveClubStateEntity,
        financial: ClubFinancialState,
        profile: ClubAiProfile
    ): String? {
        // 1. 严重更衣室叛乱
        if (clubState.dressingRoomMorale < 20) {
            return "更衣室严重失控：球员士气已跌至 ${clubState.dressingRoomMorale}，董事会认为你已失去对球队的控制。"
        }

        // 2. 严重财政危机
        if (financial.wageToIncomeRatio > 1.0) {
            return "严重财政危机：工资/收入比已达 ${(financial.wageToIncomeRatio * 100).toInt()}%，俱乐部濒临破产。"
        }

        // 3. 球迷大规模抗议
        if (clubState.fanSatisfaction < 15) {
            return "球迷大规模抗议：球迷满意度已跌至 ${clubState.fanSatisfaction}，董事会为平息民愤不得不做出解雇决定。"
        }

        return null
    }

    /**
     * 计算连续未达成核心目标赛季数
     * 从当前赛季倒推，直到遇到一个达成的核心目标赛季
     */
    private suspend fun countConsecutiveCoreFailedSeasons(
        saveId: Int, clubId: Int, currentSeasonId: Int,
        currentEvaluation: SeasonTargetEvaluation
    ): Int {
        if (!currentEvaluation.coreGoalFailed && currentEvaluation.status != "FAILED") return 0

        var count = 1  // 当前赛季算 1
        var seasonId = currentSeasonId - 1
        while (seasonId > 0) {
            val target = seasonTargetDao.getBySeason(saveId, clubId, seasonId) ?: break
            if (target.evaluationStatus == "FAILED" || hasCoreGoalFailed(target)) {
                count++
                seasonId--
            } else {
                break
            }
        }
        return count
    }

    private fun hasCoreGoalFailed(target: SeasonTargetEntity): Boolean {
        // 简化：若 evaluationStatus = FAILED 视为核心未达成
        return target.evaluationStatus == "FAILED"
    }

    private fun buildDismissalReason(
        level: DismissalLevel, consecutiveFailed: Int,
        evaluation: SeasonTargetEvaluation, satisfaction: Double
    ): String {
        return when (level) {
            DismissalLevel.NONE -> "董事会暂时对你的工作表示满意。"
            DismissalLevel.WARNING -> "你已经连续 $consecutiveFailed 赛季未能达成核心目标，董事会对你表达了明确的不满。"
            DismissalLevel.ULTIMATUM -> "连续 $consecutiveFailed 赛季未达成核心目标，董事会下达最后通牒：下赛季必须达成。"
            DismissalLevel.DISMISS -> "连续 $consecutiveFailed 赛季核心目标未达成（综合满意度 $satisfaction），董事会决定解除你的主帅职务。"
        }
    }
}

enum class DismissalLevel {
    NONE, WARNING, ULTIMATUM, DISMISS
}

data class DismissalDecision(
    val shouldDismiss: Boolean,
    val warningLevel: DismissalLevel,
    val reason: String,
    val consecutiveCoreFailedSeasons: Int = 0,
    val isEmergency: Boolean = false
) {
    companion object {
        fun none() = DismissalDecision(false, DismissalLevel.NONE, "", 0, false)
    }
}
```

```
function calculate_satisfaction(club_id, current_date):
    profile = get_club_ai_profile(club_id)
    board = get_board_member(club_id)
    financial = get_financial_state(club_id)
    club_state = get_save_club_state(club_id)

    # 8 因子评分（每项 0-100）
    s1 = league_performance_score(current_pos, target_pos)         # 联赛成绩
    s2 = cup_performance_score(current_cup_round, target_cup_round) # 杯赛成绩
    s3 = financial_score(wage_ratio, balance)                       # 财政
    s4 = club_state.fan_satisfaction                                # 球迷满意度
    s5 = transfer_market_score(budget_usage, top_need_filled)       # 转会表现
    s6 = youth_development_score(actual_promotions, target)         # 青训
    s7 = club_state.dressing_room_morale                            # 更衣室稳定
    s8 = manager_reputation_score(club_state.reputation)            # 经理声望

    # 基础权重
    weights = {league:0.20, cup:0.10, financial:0.15, fan:0.15,
               transfer:0.10, youth:0.10, dressingRoom:0.10, manager:0.10}

    # 董事会性格调整
    if board.personality == "PATIENT":
        weights.league -= 0.05; weights.youth += 0.03; weights.dressingRoom += 0.02
    elif board.personality == "AMBITIOUS":
        weights.league += 0.05; weights.youth -= 0.03; weights.cup += 0.02; weights.financial -= 0.04
    elif board.personality == "RUTHLESS":
        weights.league += 0.05; weights.dressingRoom -= 0.03; weights.fan += 0.02; weights.financial -= 0.04
    elif board.personality == "HANDS_OFF":
        weights.league -= 0.03; weights.financial += 0.05; weights.cup -= 0.02

    overall = s1*w.league + s2*w.cup + s3*w.financial + s4*w.fan +
              s5*w.transfer + s6*w.youth + s7*w.dressingRoom + s8*w.manager
    overall = clamp(overall, 0, 100)

    level = if overall >= 80: "EXCELLENT"
            elif overall >= 65: "GOOD"
            elif overall >= 50: "ACCEPTABLE"
            elif overall >= 35: "POOR"
            else: "CRITICAL"

    save_snapshot(club_id, current_date, s1-s8, overall, level)
    update_save_club_state(club_id, board_satisfaction=overall)
```

```
        return reject_due_to_cooldown(last_reviewed.cooldown_remaining)

    # 2. 30 天内同类型不可重复申请
    if last_reviewed:
        days_since = current_date - last_reviewed.reviewed_at
        if days_since < 30: return reject_due_to_cooldown(30 - days_since)

    # 3. 计算审批比例
    satisfaction = get_latest_satisfaction(club_id)
    financial_health = check_financial_health(club_id)

    if satisfaction >= 80 and financial_health == HEALTHY: ratio = 1.00
    elif satisfaction >= 80 and financial_health == ACCEPTABLE: ratio = 0.80
    elif satisfaction >= 65 and financial_health == HEALTHY: ratio = 0.80
    elif satisfaction >= 65 and financial_health == ACCEPTABLE: ratio = 0.60
    elif satisfaction >= 50 and financial_health != HIGH_RISK: ratio = 0.50
    else: ratio = 0.0

    # 设施类申请额外 15% 惩罚
    if request_type.is_facility_type(): ratio *= 0.85

    approved_amount = amount * ratio

    # 4. 财政承受力校验（不超过年收入 50%）
    max_affordable = financial.total_income * 0.50
    final_amount = min(approved_amount, max_affordable)

    # 5. 设施等级上限校验
    if request_type.is_facility_type():
        facility_level = get_facility_level(club_id, request_type)
        if facility_level >= MAX_LEVEL: return reject_capped()

    # 6. 写入申请记录
    status = if final_amount == 0: "REJECTED"
            elif final_amount < amount: "NEGOTIATED"
            else: "APPROVED"

    save_budget_request(club_id, request_type, amount, final_amount, status, current_date)

    # 7. 执行预算变更
    if status in ["APPROVED", "NEGOTIATED"]:
        apply_budget_change(club_id, request_type, final_amount)

    return BudgetRequestResult(status, final_amount, board_response, satisfaction_impact=-1)
```

### 4. 解雇判定

```
function evaluate_dismissal(club_id, season_id, season_evaluation, current_date):
    profile = get_club_ai_profile(club_id)
    club_state = get_save_club_state(club_id)
    financial = get_financial_state(club_id)

    # 1. 紧急解雇检查（最高优先级）
    if club_state.dressing_room_morale < 20:
        return DISMISS(emergency=true, reason="更衣室失控")
    if financial.wage_to_income_ratio > 1.0:
        return DISMISS(emergency=true, reason="严重财政危机")
    if club_state.fan_satisfaction < 15:
        return DISMISS(emergency=true, reason="球迷大规模抗议")

    # 2. 计算连续未达成核心目标赛季数
    consecutive_failed = 1 if season_evaluation.core_goal_failed else 0
    season = season_id - 1
    while season > 0:
        target = get_season_target(club_id, season)
        if target == null: break
        if target.evaluation_status == "FAILED":
            consecutive_failed += 1
            season -= 1
        else: break

    # 3. 董事会耐心调整阈值
    patience_adj = (profile.patience_with_manager - 50) / 50  # -1 ~ +1
    warning_threshold = 2 - patience_adj         # 默认 2
    ultimatum_threshold = 3 - patience_adj       # 默认 3
    dismiss_threshold = 4 - patience_adj         # 默认 4

    # 4. 当前满意度
    satisfaction = get_latest_satisfaction(club_id)

    # 5. 判定层级
    last_warning = get_last_warning_level(club_id, season_id - 1)

    if consecutive_failed >= dismiss_threshold:
        level = DISMISS
    elif consecutive_failed >= ultimatum_threshold and season_evaluation.core_goal_failed:
        if last_warning == ULTIMATUM and season_evaluation.core_goal_failed:
            level = DISMISS
        else:
            level = ULTIMATUM
    elif consecutive_failed >= warning_threshold:
        if last_warning == WARNING:
            level = ULTIMATUM  # 上季已警告，本季仍未达成 → 升级
        else:
            level = WARNING
    elif satisfaction < 25:
        level = WARNING  # 满意度极低也触发警告
    else:
        level = NONE

    # 6. 写入警告等级
    save_warning_level(club_id, season_id, level)

    should_dismiss = (level == DISMISS)
    return DismissalDecision(should_dismiss, level, reason, consecutive_failed)
```

## 七、交互流程

### 7.1 赛季初目标设定流程

```
玩家 -> T07 赛季开始推进 -> BoardroomService.generateSeasonTargets()
    -> GoalManager 基于俱乐部画像 + 上赛季排名生成 5 类目标
    -> BoardEventTrigger.triggerSeasonStartMeeting() 生成事件
    -> 推送通知到玩家首页（"董事会会议：新赛季目标已设定"）
    -> 玩家进入董事会页查看目标
    -> 玩家可选择"请求降低目标"（仅赛季前 1/3 阶段）
        -> GoalManager.requestTargetLowering()
        -> 满意度 -5
        -> 目标降低 1 档（如前 4 → 前 6）
        -> 事件标记为已响应
```

### 7.2 预算申请流程

```
玩家 -> 董事会页 -> 预算申请 Tab
    -> 选择类型（6 选 1）
    -> 填写金额 + 理由
    -> 点击"提交申请"
    -> BudgetRequestHandler.submitAndReview()
        -> 冷却期校验（30 天）
        -> 计算审批比例（基于满意度 + 财政）
        -> 财政承受力校验（≤ 年收入 50%）
        -> 设施等级上限校验
        -> 写入 BudgetRequestEntity
        -> 执行预算变更（save_club_state.transfer_budget += approvedAmount）
    -> 返回审批结果
        -> APPROVED: "申请已批准，转会预算 +X"
        -> NEGOTIATED: "董事会部分批准，转会预算 +Y（申请 X）"
        -> REJECTED: "董事会拒绝了你的请求，30 天内不可重复申请"
```

### 7.3 赛季评估流程

```
T19 赛季归档 -> BoardroomService.processSeasonEnd()
    -> GoalManager.evaluateSeasonTargets()
        -> 5 类目标逐一评估
        -> 综合分 = 核心 70% + 次要 30%
        -> 状态：ACHIEVED / PARTIALLY / FAILED
    -> DismissalManager.evaluate()
        -> 紧急解雇检查
        -> 连续未达成计数
        -> 董事会耐心调整阈值
        -> 判定层级：NONE / WARNING / ULTIMATUM / DISMISS
    -> BoardEventTrigger.triggerSeasonEndSummary()
        -> 生成赛季末总结事件
    -> 若 WARNING: 触发 TARGET_MISSED_WARNING 事件
    -> 若 ULTIMATUM: 触发 DISMISSAL_THREAT 事件
    -> 若 DISMISS:
        -> 玩家俱乐部：触发 Game Over 画面（"你被解雇了"）
            -> 提供选项：1) 接受失业，开始求职 2) 加载最近存档
        -> AI 俱乐部：触发 T18 ManagerSackDecider 关联换帅
```

### 7.4 解雇警告响应流程

```
玩家 -> 收到 TARGET_MISSED_WARNING 事件通知
    -> 进入董事会页
    -> 查看警告内容
    -> 选项：
        1. "接受警告，承诺下赛季达成"（默认，无额外惩罚）
        2. "反驳：上赛季转会预算不足"（需要证据，否则满意度 -5）
        3. "申请降低目标"（如还在赛季前 1/3 阶段）
    -> 事件标记为已响应
    -> 警告等级写入DismissalWarningTracker
    -> 下赛季评估时若仍未达成 → 升级为 ULTIMATUM
```

### 性能优化策略

1. **满意度缓存**：cache.db 维护 `board_satisfaction_cache` 表，UI 查询走缓存，月度计算时刷新
2. **AI 俱乐部简化**：AI 俱乐部不生成事件、不处理预算申请，仅月度满意度快照 + 赛季末评估
3. **事件批量插入**：赛季末归档时一次性写入所有事件，避免多次 IO
4. **历史赛季目标归档**：T19 归档时将 5 赛季前的 season_target 移至 history 归档表
5. **冷启动优化**：开局仅生成 40 队董事会成员模板，目标随赛季生成

## 十四、风险与应对

### 1. 满意度计算过严苛导致玩家频繁被解雇

- **表现**：玩家 5 赛季内被解雇 ≥2 次，体验差
- **应对**：
  - 调低 `satisfaction_weights` 中联赛权重（0.20 → 0.18）
  - 提高 `dismissal.warning_trigger_seasons`（2 → 3）
  - `dismissal_lookback_seasons` 限制为 4 赛季（避免历史包袱过长）
  - 满意度极低（<25）触发 WARNING 而非直接 DISMISS
- **红线**：5 赛季内玩家被解雇 ≤1 次（正常游戏行为下）

### 2. 满意度计算过宽松导致解雇机制形同虚设

- **表现**：玩家故意摆烂也不会被解雇
- **应对**：
  - 紧急解雇 3 类触发不受缓冲约束，确保极端情况立即响应
  - 满意度 <25 时强制触发 WARNING
  - 董事会耐心 <30 时阈值降低 1 档
- **红线**：连续 4 赛季核心目标 FAILED 必须触发 DISMISS

### 3. 预算申请被滥用刷钱

- **表现**：玩家反复申请预算导致俱乐部财政崩溃
- **应对**：
  - 30 天冷却期 + 同类型限制
  - 财政承受力上限（≤ 年收入 50%）
  - 申请即使批准也扣满意度 -1（频繁要钱惹恼董事会）
  - 财政 HIGH_RISK 时直接拒绝
- **红线**：单赛季预算申请总额不超过年收入 100%

### 4. AI 俱乐部董事会计算拖累性能

- **表现**：40 队月度推进 P95 > 1 秒
- **应对**：
  - AI 俱乐部仅月度满意度快照 + 赛季末评估，不生成事件
  - 满意度缓存写入 cache.db，UI 查询走缓存
  - 赛季末批量处理 40 队评估，并行化（CoroutineScope）
- **红线**：40 队月度推进 P95 ≤ 800ms

### 5. 与 T18 俱乐部画像野心不一致

- **表现**：T18 俱乐部画像 ambition=85 但 T22 董事会野心=60，行为矛盾
- **应对**：
  - **铁律**：T22 董事会野心 = T18 俱乐部画像野心（误差 ≤ 10）
  - `BoardMemberEntity.boardAmbition` 在初始化时从 `ClubAiProfile.ambition` 同步
  - 董事会换届事件触发时同步更新两边
- **红线**：自动化测试 `assert abs(board.boardAmbition - profile.ambition) <= 10`

### 6. 赛季目标与实际赛制脱节

- **表现**：欧战目标设定为 QUARTER_FINAL 但俱乐部未获欧战资格
- **应对**：
  - `GoalManager.generateSeasonTargets()` 严格校验上赛季是否获欧战资格
  - 未获资格时 `europeanTarget = "NONE"` 且 `europeanImportance = "SECONDARY"`
  - 评估时若 target = "NONE" 默认 80 分
- **红线**：未获欧战资格的俱乐部不会被设定欧战核心目标

### 7. 解雇后玩家失业流程未实现

- **表现**：玩家被解雇后游戏卡死
- **应对**：
  - W85 预留 Game Over 画面 + "加载最近存档"选项
  - V1 简化：失业后不实现求职流程，仅提供"接受失业 → 加载存档"和"回退到上赛季 checkpoint"两个选项
  - 求职流程推迟到 M7
- **红线**：解雇事件触发后游戏不卡死，玩家可恢复

### 8. 长期目标进度卡在 0%

- **表现**：声望提升目标 3 年后进度仍 0%
- **应对**：
  - `LongTermGoalTracker` 月度更新 `currentMetric`
  - 进度计算公式：`(current - start) / (target - start) * 100`
  - 长期目标到期未达成不直接解雇，仅影响满意度 -10
- **红线**：长期目标进度每月更新，不会卡 0%
