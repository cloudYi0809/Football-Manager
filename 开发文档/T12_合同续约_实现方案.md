> 定位：C 轨合同管理的核心任务，承接 T11 报价谈判，完成现有球员合同续约的完整闭环，复用 T11 的合同条款与经纪人谈判机制

## 一、业务目标与范围

### 1. 业务目标

完成 V0.1 09 §六-§七 中关于现有球员合同续约的全部场景，让玩家能：

- 在合同到期前 12/6/1 个月收到续约提醒（由 T07 每日推进触发）。
- 主动从阵容页（T04）/球员详情页对任意本队球员发起续约。
- 应对球员主动提出的续约/涨薪要求（事件驱动）。
- 通过多轮谈判（复用 T11 ContractNegotiator + AgentNegotiator）达成新合同。
- 处理合同到期：自由释放、预合同接触、自由球员进入 T11 自由签约池。
- 完成青年球员转职业合同（与 T16 青训学院衔接）。
- 完成外租球员续约（保持所有权）。

### 2. V1 做的范围

- 7 种续约触发场景：到期前 12/6/1 个月提醒、玩家主动续约、球员不满要求涨薪、青年转职业、外租续约。
- 续约意愿完整 8 因子公式（俱乐部声望 × 0.20 + 当前工资满意度 × 0.20 + 上场机会 × 0.15 + 俱乐部前景 × 0.15 + 联赛吸引力 × 0.10 + 球员野心 × 0.10 + 经纪人关系 × 0.05 + 语言/国家适应 × 0.05）。
- 续约要求多因子计算（当前工资 / CA 提升 / 表现 / 年龄 / 重要性 / 合同剩余 / 同位置对比 / 俱乐部财政 / 经纪人贪婪 / 球员野心）。
- 合同条款复用 T11：基础 9 项 + 特殊 7 项 + 角色承诺 5 档。
- 续约特有条款 3 项：涨薪条款（表现达标自动涨薪）、退役条款（老将一年一签）、青训保护条款（青训球员特殊解约金）。
- 多轮续约谈判（玩家出价 → 经纪人反应 → 球员决定 → 接受/还价/破裂）。
- 合同到期处理：到期前 6 个月可被其他俱乐部接触（预合同）、到期变自由球员、自由球员进入 T11 自由签约。
- 续约事件 8 种：球员主动要求续约、威胁不续约就离队、接受低薪续约（忠诚型）、经纪人推动续约、续约破裂挂牌、合同到期自由转会、青年转职业、外租续约。
- 全部参数配置化（独立 `renewal_config.json`）。

### 3. V1 不做（推迟）

- 合同重新谈判中的球员交换条款（V0.1 09 §二.3 明确"后期可做"）。
- 续约后的新闻发布多选项互动（仅生成固定新闻条目）。
- 球员联合施压（多名球员同时要求涨薪的群体事件）。
- 合同奖金的细化触发追踪（仅保留条款定义，触发结算在 T07 月度推进中简化处理）。
- 续约后立即调整工资预算的精细分摊（V1 按年化简单扣减）。
- 球员自购断条款（球员主动买断合同离队，V0.1 未定义）。

## 四、核心类设计

### 1. RenewalManager（续约管理器）

```kotlin
// negotiation/RenewalManager.kt
class RenewalManager(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val demandsCalculator: RenewalDemandsCalculator,
    private val willingnessCalculator: RenewalWillingnessCalculator,
    private val renegotiator: ContractRenegotiator,
    private val eventEngine: RenewalEventEngine,
    private val economyService: EconomyService,        // T17
    private val config: RenewalConfig
) {

    /**
     * 发起续约（玩家主动 / 球员主动 / 系统提醒触发）
     */
    suspend fun initiateRenewal(
        ctx: RenewalContext,
        playerId: Int,
        initiationType: InitiationType,
        triggerReason: String? = null
    ): RenewalInitiateResult {
        // 1. 校验球员属于本俱乐部
        val player = saveDb.savePlayerStateDao().getByIdWithAttributes(ctx.saveId, playerId)
            ?: return RenewalInitiateResult.Failed("球员不存在")
        if (player.currentClubId != ctx.clubId) {
            return RenewalInitiateResult.Failed("球员不属于本俱乐部")
        }

        // 2. 校验是否有进行中的续约
        val activeRenewals = saveDb.contractRenewalOfferDao().getByClubAndStatus(
            ctx.saveId, ctx.clubId, listOf(RenewalStatus.SUBMITTED.name, RenewalStatus.AGENT_EVALUATING.name)
        )
        if (activeRenewals.any { it.playerId == playerId }) {
            return RenewalInitiateResult.Failed("该球员已有进行中的续约谈判")
        }

        // 3. 计算续约要求（球员/经纪人提出）
        val demands = demandsCalculator.calculate(ctx, player)

        // 4. 计算续约意愿
        val willingness = willingnessCalculator.calculate(ctx, player, demands)

        // 5. 校验意愿是否足以谈判
        if (willingness.score < config.minWillingnessToNegotiate) {
            // 球员拒绝续约
            eventEngine.tryTriggerEvent(
                ctx, player, RenewalEventType.PLAYER_REJECT_RENEWAL,
                mapOf("willingness" to willingness.score)
            )
            return RenewalInitiateResult.PlayerRejected(
                "球员无意续约（意愿 ${"%.2f".format(willingness.score)}）",
                willingness
            )
        }

        // 6. 创建续约报价（初始为 DRAFT，玩家填条款后 SUBMITTED）
        val renewalId = saveDb.contractRenewalOfferDao().insert(ContractRenewalOfferEntity(
            renewalId = 0,
            saveId = ctx.saveId,
            playerId = playerId,
            clubId = ctx.clubId,
            initiationType = initiationType.name,
            triggerReason = triggerReason,
            currentWage = player.wage,
            currentContractUntil = player.contractUntil,
            proposedWage = 0,
            proposedContractYears = 0,
            proposedSigningBonus = 0,
            proposedAgentCommission = 0,
            rolePromise = RolePromise.STARTER.name,
            demandsWage = demands.demandsWage,
            demandsMaxYears = demands.demandsMaxYears,
            willingnessScore = willingness.score,
            status = RenewalStatus.DRAFT.name,
            createdDate = ctx.currentDate.toString(),
            expiresDate = ctx.currentDate.plusDays(config.offerValidityDays).toString(),
            currentRound = 0
        )).toInt()

        // 7. 创建续约谈判会话
        val agentId = resolveAgentId(playerId)
        saveDb.renewalSessionDao().insert(RenewalSessionEntity(
            sessionId = 0,
            saveId = ctx.saveId,
            renewalId = renewalId,
            playerId = playerId,
            agentId = agentId,
            currentRound = 0,
            maxRounds = config.maxNegotiationRounds,
            clubPatience = 100,
            playerPatience = 100,
            relationshipImpact = 0,
            startedDate = ctx.currentDate.toString(),
            lastUpdatedDate = ctx.currentDate.toString()
        ))

        // 8. 触发"球员主动要求续约"事件（如适用）
        if (initiationType == InitiationType.PLAYER_DEMAND) {
            eventEngine.tryTriggerEvent(
                ctx, player, RenewalEventType.PLAYER_DEMAND_RENEWAL,
                mapOf("demands" to demands)
            )
        }

        return RenewalInitiateResult.Success(renewalId, demands, willingness)
    }

    /**
     * 玩家提交续约报价（含合同条款）
     */
    suspend fun submitRenewalOffer(
        ctx: RenewalContext,
        renewalId: Int,
        terms: ContractTermsEntity,
        renewalSpecialTerms: RenewalSpecialTerms? = null
    ): RenewalSubmitResult {
        val offer = saveDb.contractRenewalOfferDao().getById(ctx.saveId, renewalId)
            ?: return RenewalSubmitResult.Failed("续约报价不存在")

        // 1. 校验工资预算（年化增量）
        val wageIncrease = (terms.weeklyWage - offer.currentWage).coerceAtLeast(0)
        val annualWageImpact = wageIncrease * 52
        val club = saveDb.saveClubStateDao().getByClub(ctx.saveId, ctx.clubId)!!
        if (annualWageImpact > club.wageBudgetRemaining) {
            return RenewalSubmitResult.Failed("工资预算不足（年化增量 $annualWageImpact）")
        }

        // 2. 校验签字费 + 经纪人佣金预算
        val upfrontCost = terms.signingBonus + terms.agentCommission
        if (upfrontCost > club.transferBudget) {
            return RenewalSubmitResult.Failed("预算不足（签字费+佣金 $upfrontCost）")
        }

        // 3. 持久化合同条款（复用 T11 ContractTerms，但关联 renewalId 而非 offerId）
        val termsId = saveDb.contractTermsDao().insert(terms.copy(
            termsId = 0,
            saveId = ctx.saveId,
            offerId = -1  // 标记为续约条款（负数或独立字段）
        )).toInt()

        // 4. 更新续约报价
        saveDb.contractRenewalOfferDao().update(offer.copy(
            proposedWage = terms.weeklyWage,
            proposedContractYears = terms.contractYears,
            proposedSigningBonus = terms.signingBonus,
            proposedAgentCommission = terms.agentCommission,
            rolePromise = terms.rolePromise,
            status = RenewalStatus.SUBMITTED.name,
            currentRound = 1,
            relatedTermsId = termsId
        ))

        // 5. 进入第一轮谈判
        val result = renegotiator.negotiateRound(ctx, offer.copy(relatedTermsId = termsId), terms, renewalSpecialTerms)

        return when (result) {
            is ContractRoundResult.Accepted -> {
                RenewalSubmitResult.Accepted(terms)
            }
            is ContractRoundResult.Rejected -> {
                saveDb.contractRenewalOfferDao().update(offer.copy(status = RenewalStatus.COLLAPSED.name))
                RenewalSubmitResult.Rejected(result.message)
            }
            is ContractRoundResult.Counter -> {
                saveDb.contractRenewalOfferDao().update(offer.copy(status = RenewalStatus.AGENT_COUNTERED.name))
                RenewalSubmitResult.Counter(result.terms, result.message)
            }
        }
    }

    private suspend fun resolveAgentId(playerId: Int): Int? {
        // 通过 history.db 球员表查找经纪人
        return historyDb.playerDao().getById(playerId)?.agentId
    }
}
```

### 2. RenewalDemandsCalculator（续约要求计算器）

```kotlin
// demands/RenewalDemandsCalculator.kt
class RenewalDemandsCalculator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val economyService: EconomyService,        // T17
    private val config: RenewalConfig
) {

    /**
     * 续约要求计算（多因子）
     * 球员/经纪人提出的续约要求基于：
     * 1. 当前工资（通常要求涨薪）
     * 2. 球员能力（CA 提升 → 要求匹配工资）
     * 3. 球员表现（赛季数据优异 → 要求涨薪）
     * 4. 球员年龄（老将可能要求长合同）
     * 5. 球员重要性（关键球员要求高工资）
     * 6. 合同剩余（剩余少 → 议价权高）
     * 7. 同位置球员工资对比
     * 8. 俱乐部财政（俱乐部富 → 要求高）
     * 9. 经纪人贪婪程度（贪婪型要求高）
     * 10. 球员野心（高野心可能要求解约金/豪门条款）
     */
    suspend fun calculate(
        ctx: RenewalContext,
        player: SavePlayerStateWithAttributes
    ): RenewalDemands {
        // 1. 基础期望工资（T17 WageCalculator）
        val expectedWage = economyService.calculateExpectedWage(player)

        // 2. 当前工资对比（涨薪诉求）
        val currentWage = player.wage
        val wageGap = expectedWage - currentWage
        val baseDemandWage = if (wageGap > 0) {
            // 期望工资高于当前 → 要求涨薪
            (expectedWage * (1 + config.defaultRaiseExpectation)).toInt()
        } else {
            // 当前工资已高 → 维持或小幅涨薪
            (currentWage * (1 + config.minRaisePercent)).toInt()
        }

        // 3. CA 提升修正（自上次签约以来 CA 提升）
        val caImprovement = calculateCaImprovementSinceLastContract(ctx, player)
        val caImprovementBonus = (caImprovement * config.caImprovementWageMultiplier).toInt()
        val wageAfterCa = baseDemandWage + caImprovementBonus

        // 4. 表现修正（赛季数据优异 → 要求涨薪）
        val performanceBonus = calculatePerformanceBonus(ctx, player)

        // 5. 年龄修正（老将要求长合同）
        val age = calculateAge(player.birthDate)
        val demandedYears = when {
            age >= 32 -> 1                                          // 老将一年一签
            age >= 28 -> (2..3).random()                            // 老将短约
            age >= 24 -> (3..4).random()                            // 当打之年
            else -> (4..5).random()                                 // 年轻球员长约
        }

        // 6. 重要性修正（关键球员要求高工资）
        val club = saveDb.saveClubStateDao().getByClub(ctx.saveId, ctx.clubId)!!
        val importance = PlayerImportanceResolver.resolve(player, club)  // 复用 T11
        val importanceMultiplier = config.demandsImportanceMultiplier[importance]!!
        val wageAfterImportance = (wageAfterCa * importanceMultiplier).toInt()

        // 7. 合同剩余修正（剩余少 → 议价权高）
        val monthsLeft = monthsUntilContractEnd(player.contractUntil, ctx.currentDate)
        val contractRemainMultiplier = when {
            monthsLeft <= 6 -> 1.15     // 半年到期，议价权高（可接触其他俱乐部）
            monthsLeft <= 12 -> 1.08
            monthsLeft <= 24 -> 1.0
            else -> 0.95                // 长合同议价权低
        }

        // 8. 同位置球员工资对比
        val samePosPlayers = saveDb.savePlayerStateDao()
            .getByClubAndPosition(ctx.saveId, ctx.clubId, player.primaryPosition)
        val samePosAvgWage = samePosPlayers.map { it.wage }.average().toInt()
        val samePosMultiplier = if (currentWage < samePosAvgWage * 0.8) {
            1.10  // 工资低于同位置平均 → 要求追平
        } else {
            1.0
        }

        // 9. 俱乐部财政修正（俱乐部富 → 要求高）
        val clubFinancialPower = economyService.getClubFinancialPower(ctx.clubId, ctx.saveId)
        val financeMultiplier = 1.0 + (clubFinancialPower - 50) / 200.0  // 0.75 - 1.25

        // 10. 经纪人贪婪修正
        val agent = player.agentId?.let { historyDb.agentDao().getById(it) }
        val greedMultiplier = agent?.let {
            1.0 + config.agentGreedDemandAdjustment[AgentType.valueOf(it.agentType)]!! +
                (it.greed - 50) / 200.0
        } ?: 1.0

        // 11. 球员野心修正（高野心可能要求解约金/豪门条款）
        val ambition = player.ambition
        val ambitionWageMultiplier = 1.0 + (ambition - 50) / 400.0  // 0.875 - 1.125
        val demandReleaseClause = ambition > 70 && importance == PlayerImportance.KEY

        // 12. 综合计算
        val finalDemandWage = (wageAfterImportance *
            contractRemainMultiplier *
            samePosMultiplier *
            financeMultiplier *
            greedMultiplier *
            ambitionWageMultiplier).toInt().coerceAtLeast(currentWage)

        val finalDemandWageWithPerf = finalDemandWage + performanceBonus

        // 13. 特殊要求（解约金/豪门条款等）
        val specialDemands = buildSpecialDemands(player, importance, ambition, age)

        return RenewalDemands(
            demandsWage = finalDemandWageWithPerf,
            demandsMaxYears = demandedYears,
            expectedWage = expectedWage,
            wageIncreasePercent = (finalDemandWageWithPerf - currentWage).toDouble() / currentWage.coerceAtLeast(1),
            importance = importance,
            monthsRemaining = monthsLeft,
            demandReleaseClause = demandReleaseClause,
            specialDemands = specialDemands,
            breakdown = DemandsBreakdown(
                baseDemandWage = baseDemandWage,
                caImprovementBonus = caImprovementBonus,
                performanceBonus = performanceBonus,
                importanceMultiplier = importanceMultiplier,
                contractRemainMultiplier = contractRemainMultiplier,
                samePosMultiplier = samePosMultiplier,
                financeMultiplier = financeMultiplier,
                greedMultiplier = greedMultiplier,
                ambitionWageMultiplier = ambitionWageMultiplier
            )
        )
    }

    private suspend fun calculateCaImprovementSinceLastContract(
        ctx: RenewalContext,
        player: SavePlayerStateWithAttributes
    ): Int {
        // 查询上次续约/签约时的 CA
        val lastExtension = saveDb.contractExtensionDao()
            .getByPlayer(ctx.saveId, player.playerId).firstOrNull()
        val caAtLastContract = lastExtension?.let {
            // 通过历史 CA 记录查询（T09 成长记录）
            saveDb.playerGrowthHistoryDao().getCaAtDate(ctx.saveId, player.playerId, it.extensionDate)
        } ?: player.currentCa  // 无历史记录则不修正
        return (player.currentCa - caAtLastContract).coerceAtLeast(0)
    }

    private suspend fun calculatePerformanceBonus(
        ctx: RenewalContext,
        player: SavePlayerStateWithAttributes
    ): Int {
        // 查询本赛季数据
        val seasonStats = saveDb.playerSeasonStatsDao()
            .getByPlayerAndSeason(ctx.saveId, player.playerId, ctx.seasonId) ?: return 0
        // 赛季评分 > 7.5 → 涨薪奖金
        return when {
            seasonStats.avgRating >= 8.0 -> (player.wage * 0.20).toInt()  // 顶级表现 +20%
            seasonStats.avgRating >= 7.5 -> (player.wage * 0.10).toInt()  // 优异表现 +10%
            seasonStats.avgRating >= 7.0 -> 0
            else -> 0
        }
    }

    private fun buildSpecialDemands(
        player: SavePlayerStateWithAttributes,
        importance: PlayerImportance,
        ambition: Int,
        age: Int
    ): List<String> {
        val demands = mutableListOf<String>()
        if (ambition > 70 && importance == PlayerImportance.KEY) {
            demands.add("解约金条款")
        }
        if (age >= 32) {
            demands.add("一年一签（退役条款）")
        }
        if (player.isHomeGrown) {
            demands.add("青训保护条款")
        }
        return demands
    }
}
```

### 3. RenewalWillingnessCalculator（续约意愿计算器）

```kotlin
// willingness/RenewalWillingnessCalculator.kt
class RenewalWillingnessCalculator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val economyService: EconomyService,
    private val config: RenewalConfig
) {

    /**
     * 续约意愿 8 因子公式（严格按 V0.2 文档）
     * 续约意愿 =
     *   俱乐部声望 × 0.20
     * + 当前工资满意度 × 0.20
     * + 上场机会 × 0.15
     * + 俱乐部前景 × 0.15
     * + 联赛吸引力 × 0.10
     * + 球员野心 × 0.10
     * + 经纪人关系 × 0.05
     * + 语言/国家适应 × 0.05
     */
    suspend fun calculate(
        ctx: RenewalContext,
        player: SavePlayerStateWithAttributes,
        demands: RenewalDemands
    ): RenewalWillingness {
        val club = saveDb.saveClubStateDao().getByClub(ctx.saveId, ctx.clubId)!!
        val clubEntity = historyDb.clubDao().getById(ctx.clubId)!!

        // 因子 1：俱乐部声望（0-1）
        val clubReputation = clubEntity.reputation / 100.0

        // 因子 2：当前工资满意度（0-1，基于当前工资 vs 期望工资）
        val expectedWage = demands.expectedWage
        val wageSatisfaction = (player.wage.toDouble() / expectedWage.coerceAtLeast(1))
            .coerceIn(0.0, 1.5) / 1.5  // 归一化 0-1

        // 因子 3：上场机会（基于本赛季出场时间 + 角色承诺）
        val playingChance = calculatePlayingChance(ctx, player)

        // 因子 4：俱乐部前景（基于近期战绩 + 阵容实力 + 财政）
        val clubProspect = calculateClubProspect(ctx, club)

        // 因子 5：联赛吸引力
        val leagueAttractiveness = calculateLeagueAttractiveness(clubEntity.leagueId)

        // 因子 6：球员野心（高野心球员续约意愿低，因想转会豪门）
        // 注：野心高 → 续约意愿降低（除非俱乐部声望高）
        val ambition = 1.0 - (player.ambition / 200.0)  // 0.5-1.0

        // 因子 7：经纪人关系
        val agentRelation = calculateAgentRelation(ctx, player.agentId)

        // 因子 8：语言/国家适应
        val adaptation = calculateAdaptation(player, clubEntity)

        val willingness = (
            clubReputation * 0.20 +
            wageSatisfaction * 0.20 +
            playingChance * 0.15 +
            clubProspect * 0.15 +
            leagueAttractiveness * 0.10 +
            ambition * 0.10 +
            agentRelation * 0.05 +
            adaptation * 0.05
        ).coerceIn(0.0, 1.0)

        return RenewalWillingness(
            score = willingness,
            clubReputation = clubReputation,
            wageSatisfaction = wageSatisfaction,
            playingChance = playingChance,
            clubProspect = clubProspect,
            leagueAttractiveness = leagueAttractiveness,
            ambition = ambition,
            agentRelation = agentRelation,
            adaptation = adaptation
        )
    }

    private suspend fun calculatePlayingChance(
        ctx: RenewalContext,
        player: SavePlayerStateWithAttributes
    ): Double {
        // 基于本赛季出场时间比例
        val seasonStats = saveDb.playerSeasonStatsDao()
            .getByPlayerAndSeason(ctx.saveId, player.playerId, ctx.seasonId)
        val playingTimeRatio = seasonStats?.let {
            it.appearances.toDouble() / 38.0  // 赛季 38 场
        } ?: 0.5
        // 角色修正
        val roleBonus = when (player.squadRole) {
            "key_player" -> 1.0
            "starter" -> 0.9
            "rotation" -> 0.6
            "backup" -> 0.3
            "prospect" -> 0.4
            else -> 0.5
        }
        return (playingTimeRatio * 0.6 + roleBonus * 0.4).coerceIn(0.0, 1.0)
    }

    private suspend fun calculateClubProspect(
        ctx: RenewalContext,
        club: SaveClubStateEntity
    ): Double {
        // 近期战绩（联赛排名）
        val leaguePosition = club.currentLeaguePosition
        val positionScore = when {
            leaguePosition <= 4 -> 1.0           // 欧冠区
            leaguePosition <= 6 -> 0.8           // 欧联区
            leaguePosition <= 10 -> 0.6          // 中游
            leaguePosition <= 17 -> 0.4          // 保级
            else -> 0.2                          // 降级区
        }
        // 财政前景
        val financialPower = economyService.getClubFinancialPower(club.clubId, ctx.saveId)
        val financeScore = financialPower / 100.0
        // 欧战资格
        val euroBonus = when {
            club.hasChampionsLeague -> 0.15
            club.hasEuropaLeague -> 0.08
            else -> 0.0
        }
        return (positionScore * 0.5 + financeScore * 0.5 + euroBonus).coerceIn(0.0, 1.0)
    }

    private fun calculateLeagueAttractiveness(leagueId: String): Double {
        return config.leagueAttractiveness[leagueId] ?: 0.5
    }

    private suspend fun calculateAgentRelation(
        ctx: RenewalContext,
        agentId: Int?
    ): Double {
        if (agentId == null) return 0.5
        return saveDb.agentPlayerRelationDao()
            .get(ctx.saveId, agentId, ctx.clubId)?.let {
                (it.relationshipValue / 100.0).coerceIn(0.0, 1.0)
            } ?: 0.5
    }

    private fun calculateAdaptation(
        player: SavePlayerStateWithAttributes,
        club: ClubEntity
    ): Double {
        if (player.nationality == club.country) return 1.0
        val sameLanguageGroup = isSameLanguageGroup(player.nationality, club.country)
        return if (sameLanguageGroup) 0.7 else 0.4
    }

    private fun isSameLanguageGroup(nationality1: String, nationality2: String): Boolean {
        // V1 简化：按语言大区分组
        val groups = listOf(
            setOf("England", "Scotland", "Wales", "Ireland", "USA", "Canada", "Australia"),
            setOf("Spain", "Mexico", "Argentina", "Colombia", "Chile", "Uruguay", "Portugal", "Brazil"),
            setOf("France", "Belgium", "Switzerland", "Senegal", "Ivory Coast", "Morocco", "Algeria", "Tunisia"),
            setOf("Germany", "Austria", "Netherlands", "Belgium"),
            setOf("Italy", "San Marino"),
            setOf("Croatia", "Serbia", "Bosnia", "Slovenia", "Montenegro", "North Macedonia")
        )
        return groups.any { nationality1 in it && nationality2 in it }
    }
}

    val playingChance: Double,
    val clubProspect: Double,
    val leagueAttractiveness: Double,
```

### 4. ContractRenegotiator（合同重新谈判，复用 T11）

```kotlin
// negotiation/ContractRenegotiator.kt
class ContractRenegotiator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val t11ContractNegotiator: ContractNegotiator,    // 复用 T11
    private val t11AgentNegotiator: AgentNegotiator,          // 复用 T11
    private val config: RenewalConfig
) {

    /**
     * 续约谈判回合（复用 T11 ContractNegotiator，但有差异）
     * 差异：
     * 1. 无卖方俱乐部（球员已在俱乐部）
     * 2. 谈判焦点在工资/年限/特殊条款，无转会费
     * 3. 球员议价权基于合同剩余（剩少 → 议价权高）
     * 4. 续约特有条款（涨薪/退役/青训保护）
     */
    suspend fun negotiateRound(
        ctx: RenewalContext,
        offer: ContractRenewalOfferEntity,
        playerTerms: ContractTermsEntity,
        renewalSpecialTerms: RenewalSpecialTerms? = null
    ): ContractRoundResult {
        val player = saveDb.savePlayerStateDao().getByIdWithAttributes(ctx.saveId, offer.playerId)

        // 1. 经纪人评估条款（复用 T11 AgentNegotiator）
        val agentReaction = t11AgentNegotiator.evaluateTerms(
            ctx.toNegotiationContext(),
            buildT11OfferForRenewal(offer),
            playerTerms,
            player
        )

        // 2. 续约特有评估（球员额外考虑续约特有条款）
        val renewalSpecificScore = scoreRenewalSpecialTerms(renewalSpecialTerms, player, offer)

        // 3. 球员综合评估（复用 T11 评分逻辑 + 续约修正）
        val playerScore = scoreTermsForRenewal(playerTerms, player, offer, renewalSpecificScore, ctx)

        // 4. 合同剩余议价权修正
        val monthsLeft = monthsUntilContractEnd(player.contractUntil, ctx.currentDate)
        val bargainingPower = when {
            monthsLeft <= 6 -> 1.15     // 可接触其他俱乐部，议价权高
            monthsLeft <= 12 -> 1.08
            else -> 1.0
        }
        val adjustedScore = playerScore * bargainingPower

        // 5. 综合决策
        return when {
            agentReaction.accepted && adjustedScore >= config.playerAcceptThreshold -> {
                ContractRoundResult.Accepted(playerTerms)
            }
            agentReaction.walkAway || adjustedScore < config.playerRejectThreshold -> {
                ContractRoundResult.Rejected(agentReaction.message)
            }
            else -> {
                val counterTerms = generateRenewalCounter(playerTerms, agentReaction, player, offer, ctx)
                ContractRoundResult.Counter(counterTerms, agentReaction.message)
            }
        }
    }

    /**
     * 续约特有条款评分（涨薪条款对球员有利，退役条款对俱乐部有利）
     */
    private fun scoreRenewalSpecialTerms(
        specialTerms: RenewalSpecialTerms?,
        player: SavePlayerStateWithAttributes,
        offer: ContractRenewalOfferEntity
    ): Double {
        if (specialTerms == null) return 0.0
        var score = 0.0
        // 涨薪条款（表现达标自动涨薪）—— 对球员有利
        specialTerms.performanceRaiseClause?.let {
            score += 0.20
        }
        // 退役条款（老将一年一签）—— 对俱乐部有利，对球员略不利
        specialTerms.veteranClause?.let {
            score -= 0.10
        }
        // 青训保护条款（特殊解约金）—— 对球员有利（易离队）
        specialTerms.academyProtectionClause?.let {
            score += 0.15
        }
        return score.coerceIn(-0.20, 0.35)
    }

    private fun scoreTermsForRenewal(
        terms: ContractTermsEntity,
        player: SavePlayerStateWithAttributes,
        offer: ContractRenewalOfferEntity,
        renewalSpecificScore: Double,
        ctx: RenewalContext
    ): Double {
        // 复用 T11 的评分逻辑（工资/年限/角色/奖金/特殊条款）
        // 续约场景下额外考虑：工资涨幅（vs 当前工资）
        val wageIncrease = terms.weeklyWage - offer.currentWage
        val wageIncreaseRatio = wageIncrease.toDouble() / offer.currentWage.coerceAtLeast(1)
        val wageIncreaseBonus = when {
            wageIncreaseRatio >= 0.30 -> 0.20   // 涨薪 30%+
            wageIncreaseRatio >= 0.15 -> 0.10   // 涨薪 15%+
            wageIncreaseRatio >= 0.0 -> 0.0     // 平薪或微涨
            else -> -0.15                        // 降薪
        }
        return (renewalSpecificScore + wageIncreaseBonus).coerceIn(-0.35, 0.55)
    }

    /**
     * 生成续约还价（球员/经纪人还价）
     */
    private suspend fun generateRenewalCounter(
        playerTerms: ContractTermsEntity,
        agentReaction: AgentReaction,
        player: SavePlayerStateWithAttributes,
        offer: ContractRenewalOfferEntity,
        ctx: RenewalContext
    ): ContractTermsEntity {
        // 还价工资：在玩家出价和球员要求之间，偏向球员要求
        val targetWage = offer.demandsWage
        val counterWage = ((playerTerms.weeklyWage + targetWage * (1 + agentReaction.greedAdjustment)) / 2)
            .toInt().coerceAtLeast(playerTerms.weeklyWage)

        // 还价年限：球员偏好的年限
        val counterYears = offer.demandsMaxYears.coerceAtLeast(playerTerms.contractYears)

        return playerTerms.copy(
            weeklyWage = counterWage,
            contractYears = counterYears,
            signingBonus = (playerTerms.signingBonus * 1.2).toInt(),
            agentCommission = (playerTerms.agentCommission * 1.15).toInt()
        )
    }

    private fun buildT11OfferForRenewal(renewal: ContractRenewalOfferEntity): TransferOfferEntity {
        // 构造一个虚拟的 T11 TransferOfferEntity 以复用 T11 AgentNegotiator
        return TransferOfferEntity(
            offerId = -renewal.renewalId,  // 负数标记为续约
            saveId = renewal.saveId,
            playerId = renewal.playerId,
            fromClubId = renewal.clubId,
            toClubId = renewal.clubId,  // 续约无卖方
            offerType = OfferType.PERMANENT.name,
            transferFee = 0,  // 续约无转会费
            wageOffer = renewal.proposedWage,
            contractYears = renewal.proposedContractYears,
            signingBonus = renewal.proposedSigningBonus,
            agentCommission = renewal.proposedAgentCommission,
            rolePromise = renewal.rolePromise,
            status = OfferStatus.PLAYER_NEGOTIATING.name,
            createdDate = renewal.createdDate,
            expiresDate = renewal.expiresDate,
            currentRound = renewal.currentRound
        )
    }
}
```

```kotlin
// reminder/RenewalReminderService.kt
class RenewalReminderService(
    private val saveDb: SaveDatabase,
    private val eventEngine: RenewalEventEngine,
    private val config: RenewalConfig
) {

    /**
     * 续约提醒检查（由 T07 每日推进调用）
     * 检查全队球员合同剩余，触发提醒
     */
    suspend fun checkReminders(ctx: RenewalContext): List<ContractReminderEntity> {
        val newReminders = mutableListOf<ContractReminderEntity>()

        // 1. 获取本俱乐部所有球员
        val players = saveDb.savePlayerStateDao().getByClub(ctx.saveId, ctx.clubId)

        for (player in players) {
            // 跳过已处理的球员
            val existingReminder = saveDb.contractReminderDao()
                .getLatestByPlayer(ctx.saveId, player.playerId)
            if (existingReminder != null && !existingReminder.isHandled) continue

            // 2. 计算合同剩余月数
            val monthsLeft = monthsUntilContractEnd(player.contractUntil, ctx.currentDate)
            if (monthsLeft > 12) continue  // 12 个月以上不提醒

            // 3. 确定提醒级别
            val level = when {
                monthsLeft <= 1 -> ReminderLevel.FINAL
                monthsLeft <= 6 -> ReminderLevel.URGENT
                monthsLeft <= 12 -> ReminderLevel.EARLY_WARNING
                else -> continue
            }

            // 4. 检查是否已存在同级别提醒
            val existingSameLevel = saveDb.contractReminderDao()
                .getActiveByClub(ctx.saveId, ctx.clubId)
                .find { it.playerId == player.playerId && it.level == level.name }
            if (existingSameLevel != null) continue

            // 5. 生成建议动作
            val recommendedAction = generateRecommendedAction(player, monthsLeft, ctx)

            // 6. 创建提醒
            val reminder = ContractReminderEntity(
                reminderId = 0,
                saveId = ctx.saveId,
                playerId = player.playerId,
                clubId = ctx.clubId,
                level = level.name,
                monthsRemaining = monthsLeft,
                triggerDate = ctx.currentDate.toString(),
                isHandled = false,
                playerDemandTriggered = false,
                recommendedAction = recommendedAction
            )
            val reminderId = saveDb.contractReminderDao().insert(reminder).toInt()
            newReminders.add(reminder.copy(reminderId = reminderId))

            // 7. 6 个月提醒触发球员主动续约概率
            if (level == ReminderLevel.URGENT) {
                val playerDemandProbability = calculatePlayerDemandProbability(player, ctx)
                if (Math.random() < playerDemandProbability) {
                    eventEngine.tryTriggerEvent(
                        ctx, player, RenewalEventType.PLAYER_DEMAND_RENEWAL,
                        mapOf("reminder" to reminder, "monthsLeft" to monthsLeft)
                    )
                    saveDb.contractReminderDao().update(reminder.copy(
                        reminderId = reminderId,
                        playerDemandTriggered = true
                    ))
                }
            }

            // 8. 1 个月提醒触发威胁离队概率
            if (level == ReminderLevel.FINAL) {
                val threatProbability = calculateThreatProbability(player, ctx)
                if (Math.random() < threatProbability) {
                    eventEngine.tryTriggerEvent(
                        ctx, player, RenewalEventType.THREAT_TO_LEAVE,
                        mapOf("reminder" to reminder, "monthsLeft" to monthsLeft)
                    )
                }
            }
        }

        return newReminders
    }

    private fun generateRecommendedAction(
        player: SavePlayerStateWithAttributes,
        monthsLeft: Int,
        ctx: RenewalContext
    ): String {
        val importance = player.squadRole
        val age = calculateAge(player.birthDate)
        return when {
            importance == "key_player" && age < 30 -> "RENEW_IMMEDIATELY"
            importance == "starter" && age < 32 -> "RENEW_IMMEDIATELY"
            monthsLeft <= 1 -> "RENEW_OR_RELEASE"
            age >= 34 -> "RELEASE_OR_SHORT_TERM"
            importance in listOf("backup", "prospect") -> "EVALUATE"
            else -> "RENEW"
        }
    }

    private fun calculatePlayerDemandProbability(
        player: SavePlayerStateWithAttributes,
        ctx: RenewalContext
    ): Double {
        // 关键球员 + 高野心 → 更可能主动要求续约
        val baseProb = 0.30
        val roleBonus = when (player.squadRole) {
            "key_player" -> 0.20
            "starter" -> 0.10
            else -> 0.0
        }
        val ambitionBonus = (player.ambition - 50) / 200.0
        return (baseProb + roleBonus + ambitionBonus).coerceIn(0.10, 0.80)
    }

    private fun calculateThreatProbability(
        player: SavePlayerStateWithAttributes,
        ctx: RenewalContext
    ): Double {
        // 高野心 + 高能力 → 更可能威胁离队
        val baseProb = 0.20
        val ambitionBonus = (player.ambition - 50) / 200.0
        val caBonus = (player.currentCa - 70) / 200.0
        return (baseProb + ambitionBonus + caBonus).coerceIn(0.05, 0.60)
    }
}
```

```kotlin
// expiry/ContractExpiryHandler.kt
class ContractExpiryHandler(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val preContractHandler: PreContractHandler,
    private val freeAgentReleaser: FreeAgentReleaser,
    private val eventEngine: RenewalEventEngine,
    private val config: RenewalConfig
) {

    /**
     * 合同到期处理（由 T07 每日推进调用）
     * 处理流程：
     * 1. 合同到期前 6 个月：其他俱乐部可接触（预合同）
     * 2. 合同到期：球员变为自由球员
     * 3. 自由球员可被任何俱乐部签约（衔接 T11 自由签约）
     */
    suspend fun processExpiry(ctx: RenewalContext): List<ExpiryResult> {
        val results = mutableListOf<ExpiryResult>()

        // 1. 处理预合同接触（合同剩 6 个月的球员）
        val preContractResults = preContractHandler.processPreContractContacts(ctx)
        results.addAll(preContractResults)

        // 2. 处理合同到期（合同已到期的球员）
        val expiryResults = processExpiredContracts(ctx)
        results.addAll(expiryResults)

        return results
    }

    private suspend fun processExpiredContracts(ctx: RenewalContext): List<ExpiryResult> {
        val results = mutableListOf<ExpiryResult>()
        val players = saveDb.savePlayerStateDao().getByClub(ctx.saveId, ctx.clubId)

        for (player in players) {
            val contractUntil = LocalDate.parse(player.contractUntil)
            if (contractUntil.isAfter(ctx.currentDate)) continue  // 未到期

            // 1. 检查是否有续约完成（合同已更新）
            val lastExtension = saveDb.contractExtensionDao()
                .getLastCompletedByPlayer(ctx.saveId, player.playerId)
            if (lastExtension != null && LocalDate.parse(lastExtension.newContractUntil).isAfter(ctx.currentDate)) {
                continue  // 已续约
            }

            // 2. 球员变为自由球员
            val releaseResult = freeAgentReleaser.release(ctx, player.playerId)
            results.add(releaseResult)

            // 3. 触发"合同到期自由转会"事件
            eventEngine.tryTriggerEvent(
                ctx, player, RenewalEventType.CONTRACT_EXPIRED_FREE_TRANSFER,
                mapOf("player" to player)
            )

            // 4. 生成新闻
            saveDb.newsDao().insert(NewsEntity(
                newsId = 0,
                saveId = ctx.saveId,
                type = "CONTRACT_EXPIRED",
                title = "${getPlayerName(player.playerId)} 合同到期离队",
                content = "${getPlayerName(player.playerId)} 与 ${getClubName(ctx.clubId)} 的合同到期，成为自由球员。",
                date = ctx.currentDate.toString(),
                relatedPlayerId = player.playerId,
                relatedClubId = ctx.clubId
            ))
        }

        return results
    }
}

// expiry/PreContractHandler.kt
class PreContractHandler(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val config: RenewalConfig
) {

    /**
     * 预合同接触处理（Bosman 规则）
     * 合同到期前 6 个月，其他俱乐部可接触球员签订预合同
     */
    suspend fun processPreContractContacts(ctx: RenewalContext): List<ExpiryResult> {
        val results = mutableListOf<ExpiryResult>()
        val players = saveDb.savePlayerStateDao().getByClub(ctx.saveId, ctx.clubId)

        for (player in players) {
            val monthsLeft = monthsUntilContractEnd(player.contractUntil, ctx.currentDate)
            // 仅在 6 个月时触发一次（避免重复）
            if (monthsLeft != 6) continue

            // 1. 检查球员是否已有预合同接触标记
            if (saveDb.preContractContactDao().hasActiveContact(ctx.saveId, player.playerId)) continue

            // 2. 计算被其他俱乐部接触的概率
            val contactProbability = calculateContactProbability(player, ctx)
            if (Math.random() > contactProbability) continue

            // 3. 查找感兴趣的俱乐部（V1 简化：随机选 1-3 家）
            val interestedClubs = findInterestedClubs(player, ctx)
            if (interestedClubs.isEmpty()) continue

            // 4. 记录预合同接触
            for (club in interestedClubs) {
                saveDb.preContractContactDao().insert(PreContractContactEntity(
                    contactId = 0,
                    saveId = ctx.saveId,
                    playerId = player.playerId,
                    currentClubId = ctx.clubId,
                    contactingClubId = club.clubId,
                    contactDate = ctx.currentDate.toString(),
                    status = "CONTACTED"
                ))
            }

            // 5. 生成新闻（提醒玩家）
            saveDb.newsDao().insert(NewsEntity(
                newsId = 0,
                saveId = ctx.saveId,
                type = "PRE_CONTRACT_CONTACT",
                title = "${getPlayerName(player.playerId)} 被其他俱乐部接触",
                content = "${getPlayerName(player.playerId)} 合同即将到期，已收到来自其他俱乐部的预合同接触。",
                date = ctx.currentDate.toString(),
                relatedPlayerId = player.playerId,
                relatedClubId = ctx.clubId
            ))

            results.add(ExpiryResult.PreContractContact(
                playerId = player.playerId,
                contactingClubs = interestedClubs.map { it.clubId }
            ))
        }

        return results
    }

    private fun calculateContactProbability(
        player: SavePlayerStateWithAttributes,
        ctx: RenewalContext
    ): Double {
        // CA 越高、年龄越当打 → 越可能被接触
        val baseProb = 0.40
        val caBonus = (player.currentCa - 70) / 100.0
        val age = calculateAge(player.birthDate)
        val ageBonus = when (age) {
            in 24..29 -> 0.20
            in 30..32 -> 0.10
            else -> 0.0
        }
        return (baseProb + caBonus + ageBonus).coerceIn(0.10, 0.85)
    }

    private suspend fun findInterestedClubs(
        player: SavePlayerStateWithAttributes,
        ctx: RenewalContext
    ): List<ClubEntity> {
        // V1 简化：按声望筛选，随机选 1-3 家
        val allClubs = historyDb.clubDao().getByLeague(player.preferredLeague ?: "EPL")
        return allClubs
            .filter { it.clubId != ctx.clubId && it.reputation > 50 }
            .sortedByDescending { it.reputation }
            .take(5)
            .shuffled()
            .take((1..3).random())
    }
}

class FreeAgentReleaser(

            saveId = ctx.saveId,
```

### 7. RenewalCompleter（续约完成）

```kotlin
// complete/RenewalCompleter.kt
class RenewalCompleter(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val economyService: EconomyService,
    private val config: RenewalConfig
) {

    /**
     * 续约完成
     * 1. 更新球员合同（工资/到期日/角色）
     * 2. 扣减工资预算（年化增量）+ 转会预算（签字费+佣金）
     * 3. 触发经纪人关系提升
     * 4. 生成续约记录
     * 5. 生成新闻
     * 6. 标记提醒已处理
     */
    suspend fun complete(
        ctx: RenewalContext,
        offer: ContractRenewalOfferEntity,
        terms: ContractTermsEntity
    ): RenewalCompleteResult {
        val player = saveDb.savePlayerStateDao().getByIdWithAttributes(ctx.saveId, offer.playerId)

        // 1. 计算新合同到期日
        val newContractUntil = ctx.currentDate.plusYears(terms.contractYears.toLong()).toString()

        // 2. 更新球员合同
        saveDb.savePlayerStateDao().updateClubAndContract(
            saveId = ctx.saveId,
            playerId = offer.playerId,
            newClubId = ctx.clubId,
            newWage = terms.weeklyWage,
            newContractUntil = newContractUntil,
            newSquadRole = terms.rolePromise.toSquadRole()
        )

        // 3. 扣减预算
        applyBudgetImpact(ctx, offer, terms)

        // 4. 经纪人关系提升（成交后 +关系）
        val session = saveDb.renewalSessionDao().getByRenewal(ctx.saveId, offer.renewalId)
        session?.agentId?.let { agentId ->
            saveDb.agentPlayerRelationDao().adjustRelationship(
                ctx.saveId, agentId, ctx.clubId, config.agentRelationGainOnRenewal
            )
        }

        // 5. 计算工资变化百分比
        val wageChangePercent = (terms.weeklyWage - offer.currentWage).toDouble() /
            offer.currentWage.coerceAtLeast(1) * 100

        // 6. 写入续约记录
        val extensionId = saveDb.contractExtensionDao().insert(ContractExtensionEntity(
            extensionId = 0,
            saveId = ctx.saveId,
            playerId = offer.playerId,
            clubId = ctx.clubId,
            renewalId = offer.renewalId,
            previousWage = offer.currentWage,
            newWage = terms.weeklyWage,
            previousContractUntil = offer.currentContractUntil,
            newContractUntil = newContractUntil,
            extensionYears = terms.contractYears,
            signingBonus = terms.signingBonus,
            agentCommission = terms.agentCommission,
            wageChangePercent = wageChangePercent,
            extensionDate = ctx.currentDate.toString(),
            seasonId = ctx.seasonId,
            specialTermsSnapshot = null  // V1 简化
        )).toInt()

        // 7. 更新续约报价状态
        saveDb.contractRenewalOfferDao().update(offer.copy(
            status = RenewalStatus.COMPLETED.name
        ))

        // 8. 标记续约提醒已处理
        saveDb.contractReminderDao().markHandledByPlayer(ctx.saveId, offer.playerId)

        // 9. 生成新闻
        generateRenewalNews(ctx, offer, player, terms, wageChangePercent)

        return RenewalCompleteResult(
            extensionId = extensionId,
            playerId = offer.playerId,
            newWage = terms.weeklyWage,
            newContractUntil = newContractUntil,
            wageChangePercent = wageChangePercent
        )
    }

    private suspend fun applyBudgetImpact(
        ctx: RenewalContext,
        offer: ContractRenewalOfferEntity,
        terms: ContractTermsEntity
    ) {
        val club = saveDb.saveClubStateDao().getByClub(ctx.saveId, ctx.clubId)!!

        // 1. 工资预算扣减（年化增量）
        val wageIncrease = (terms.weeklyWage - offer.currentWage).coerceAtLeast(0)
        val annualWageImpact = wageIncrease * 52
        saveDb.saveClubStateDao().updateWageBudget(
            ctx.clubId, ctx.saveId, club.wageBudget - annualWageImpact
        )

        // 2. 转会预算扣减（签字费 + 经纪人佣金）
        val upfrontCost = terms.signingBonus + terms.agentCommission
        if (upfrontCost > 0) {
            saveDb.saveClubStateDao().updateTransferBudget(
                ctx.clubId, ctx.saveId, club.transferBudget - upfrontCost
            )
        }
    }

    private suspend fun generateRenewalNews(
        ctx: RenewalContext,
        offer: ContractRenewalOfferEntity,
        player: SavePlayerStateWithAttributes,
        terms: ContractTermsEntity,
        wageChangePercent: Double
    ) {
        val playerName = getPlayerName(player.playerId)
        val clubName = getClubName(ctx.clubId)
        val changeText = when {
            wageChangePercent >= 30 -> "大幅涨薪续约"
            wageChangePercent >= 10 -> "涨薪续约"
            wageChangePercent >= 0 -> "续约"
            else -> "降薪续约"
        }
        saveDb.newsDao().insert(NewsEntity(
            newsId = 0,
            saveId = ctx.saveId,
            type = "CONTRACT_RENEWAL",
            title = "$playerName 与 $clubName $changeText",
            content = "$playerName 与 $clubName 续约 ${terms.contractYears} 年，新合同至 ${LocalDate.parse(offer.currentContractUntil).plusYears(terms.contractYears.toLong()).year} 年。",
            date = ctx.currentDate.toString(),
            relatedPlayerId = offer.playerId,
            relatedClubId = ctx.clubId
        ))
    }
}
```

```kotlin
            RenewalEventType.CONTRACT_EXPIRED_FREE_TRANSFER -> triggerContractExpiredFreeTransfer(ctx, player, payload)
            RenewalEventType.YOUTH_TO_PRO -> triggerYouthToPro(ctx, player, payload)
```

### 1. 续约要求计算（10 因子）

```text
function calcRenewalDemands(ctx, player):
    # 1. 基础期望工资（T17）
    expectedWage = economyService.calculateExpectedWage(player)

    # 2. 当前工资对比
    currentWage = player.wage
    if expectedWage > currentWage:
        baseDemandWage = expectedWage * (1 + DEFAULT_RAISE_EXPECTATION)  # 默认 +10%
    else:
        baseDemandWage = currentWage * (1 + MIN_RAISE_PERCENT)            # 默认 +3%

    # 3. CA 提升修正
    caImprovement = player.currentCa - getCaAtLastContract(player)
    caImprovementBonus = caImprovement * CA_IMPROVEMENT_WAGE_MULTIPLIER  # 每点 CA +5000
    wageAfterCa = baseDemandWage + caImprovementBonus

    # 4. 表现修正
    performanceBonus = 0
    if player.seasonStats.avgRating >= 8.0:
        performanceBonus = currentWage * 0.20
    elif player.seasonStats.avgRating >= 7.5:
        performanceBonus = currentWage * 0.10

    # 5. 年龄修正（要求年限）
    age = calcAge(player.birthDate)
    demandedYears =
        if age >= 32: 1                          # 老将一年一签
        elif age >= 28: random(2, 3)
        elif age >= 24: random(3, 4)
        else: random(4, 5)

    # 6. 重要性修正
    importance = resolveImportance(player, club)  # 复用 T11 5 档
    importanceMultiplier = DEMANDS_IMPORTANCE_MULTIPLIER[importance]
        # KEY=1.30, STARTER=1.15, ROTATION=1.00, BACKUP=0.85, LISTED=0.70
    wageAfterImportance = wageAfterCa * importanceMultiplier

    # 7. 合同剩余修正
    monthsLeft = monthsUntilContractEnd(player.contractUntil, currentDate)
    contractRemainMultiplier =
        if monthsLeft <= 6: 1.15     # 半年到期，议价权高
        elif monthsLeft <= 12: 1.08
        elif monthsLeft <= 24: 1.00
        else: 0.95

    # 8. 同位置球员工资对比
    samePosAvgWage = avg(samePositionPlayers.wage)
    samePosMultiplier =
        if currentWage < samePosAvgWage * 0.8: 1.10  # 工资低于同位置 → 追平
        else: 1.0

    # 9. 俱乐部财政修正
    clubFinancialPower = economyService.getClubFinancialPower(clubId)  # 0-100
    financeMultiplier = 1.0 + (clubFinancialPower - 50) / 200.0        # 0.75 - 1.25

    # 10. 经纪人贪婪修正
    agent = getAgent(player.agentId)
    greedMultiplier = 1.0 + AGENT_GREED_DEMAND[agent.type] + (agent.greed - 50) / 200
        # GREEDY=+0.20, STAR=+0.15, DISRUPTIVE=+0.10, RELATIONAL=-0.05, ACADEMY=-0.05

    # 11. 球员野心修正
    ambition = player.ambition
    ambitionWageMultiplier = 1.0 + (ambition - 50) / 400.0  # 0.875 - 1.125
    demandReleaseClause = ambition > 70 and importance == KEY

    # 12. 综合计算
    finalDemandWage = wageAfterImportance * contractRemainMultiplier * samePosMultiplier *
                      financeMultiplier * greedMultiplier * ambitionWageMultiplier
    finalDemandWage = max(finalDemandWage + performanceBonus, currentWage)

    return RenewalDemands(
        demandsWage = finalDemandWage,
        demandsMaxYears = demandedYears,
        demandReleaseClause = demandReleaseClause,
        specialDemands = buildSpecialDemands(player, importance, ambition, age)
    )
```

```text
function calcRenewalWillingness(ctx, player, demands):
    club = getClub(ctx.clubId)
    clubEntity = getClubEntity(ctx.clubId)

    # 因子 1：俱乐部声望（0-1）
    clubReputation = clubEntity.reputation / 100.0

    # 因子 2：当前工资满意度（0-1）
    wageSatisfaction = clamp(player.wage / demands.expectedWage, 0.0, 1.5) / 1.5

    # 因子 3：上场机会（0-1）
    playingChance = calcPlayingChance(ctx, player)
        # = seasonStats.appearances / 38 * 0.6 + roleBonus * 0.4

    # 因子 4：俱乐部前景（0-1）
    clubProspect = calcClubProspect(ctx, club)
        # = leaguePositionScore * 0.5 + financialPower * 0.5 + euroBonus

    # 因子 5：联赛吸引力（0-1）
    leagueAttractiveness = LEAGUE_ATTRACTIVENESS[clubEntity.leagueId]

    # 因子 6：球员野心（高野心 → 续约意愿降低）
    ambition = 1.0 - player.ambition / 200.0  # 0.5-1.0

    # 因子 7：经纪人关系（0-1）
    agentRelation = getAgentRelation(ctx, player.agentId)

    # 因子 8：语言/国家适应（0-1）
    adaptation = calcAdaptation(player.nationality, clubEntity.country)

    return clamp(
        clubReputation * 0.20 +
        wageSatisfaction * 0.20 +
        playingChance * 0.15 +
        clubProspect * 0.15 +
        leagueAttractiveness * 0.10 +
        ambition * 0.10 +
        agentRelation * 0.05 +
        adaptation * 0.05,
        0.0, 1.0
    )
```

```text
function renewalNegotiationRound(ctx, offer, playerTerms, renewalSpecialTerms):
    player = getPlayer(offer.playerId)

    # 1. 经纪人评估（复用 T11 AgentNegotiator）
    agentReaction = t11AgentNegotiator.evaluateTerms(
        ctx, buildT11OfferForRenewal(offer), playerTerms, player
    )
        # 返回: accepted / walkAway / counter(greedAdj)

    # 2. 续约特有评估
    renewalSpecificScore = scoreRenewalSpecialTerms(renewalSpecialTerms, player)
        # performanceRaiseClause: +0.20
        # veteranClause: -0.10
        # academyProtectionClause: +0.15

    # 3. 球员综合评估
    playerScore = scoreTermsForRenewal(playerTerms, player, offer, renewalSpecificScore)
        # = renewalSpecificScore + wageIncreaseBonus
        # wageIncreaseBonus:
        #   涨薪 >= 30%: +0.20
        #   涨薪 >= 15%: +0.10
        #   平薪: 0
        #   降薪: -0.15

    # 4. 合同剩余议价权修正
    monthsLeft = monthsUntilContractEnd(player.contractUntil, currentDate)
    bargainingPower =
        if monthsLeft <= 6: 1.15     # 可接触其他俱乐部
        elif monthsLeft <= 12: 1.08
        else: 1.0
    adjustedScore = playerScore * bargainingPower

    # 5. 综合决策
    if agentReaction.accepted and adjustedScore >= PLAYER_ACCEPT_THRESHOLD (0.60):
        return ACCEPTED(playerTerms)

    if agentReaction.walkAway or adjustedScore < PLAYER_REJECT_THRESHOLD (0.25):
        return REJECTED(agentReaction.message)

    # 还价
    counterTerms = generateRenewalCounter(playerTerms, agentReaction, player, offer)
        # counterWage = (playerTerms.wage + demandsWage * (1 + greedAdj)) / 2
        # counterYears = max(demandsMaxYears, playerTerms.years)
    return COUNTER(counterTerms, agentReaction.message)
```

```text
function processExpiry(ctx):
    results = []

    # 1. 预合同接触（合同剩 6 个月）
    for player in clubPlayers:
        monthsLeft = monthsUntilContractEnd(player.contractUntil, currentDate)
        if monthsLeft != 6: continue
        if hasActivePreContractContact(player): continue

        contactProb = calcContactProbability(player)
            # base 0.40 + caBonus + ageBonus
        if random() > contactProb: continue

        interestedClubs = findInterestedClubs(player)  # 1-3 家
        for club in interestedClubs:
            recordPreContractContact(player, club)

        generateNews(player, "被其他俱乐部接触")
        results.add(PRE_CONTRACT_CONTACT(player, interestedClubs))

    # 2. 合同到期（已到期球员）
    for player in clubPlayers:
        if not isContractExpired(player): continue
        if hasRenewedContract(player): continue  # 已续约

        # 球员变为自由球员
        releasePlayerAsFreeAgent(player)
            # player.currentClubId = null
            # player.wage = 0
            # 加入 free_agent_pool 表（T11 可搜索）

        triggerEvent(CONTRACT_EXPIRED_FREE_TRANSFER)
        generateNews(player, "合同到期离队")
        results.add(RELEASED(player))

    return results
```

```text
function tryTriggerRenewalEvent(ctx, player, eventType, payload):
    probability = EVENT_TRIGGER_RATE[eventType]
    if random() > probability: return false

    switch eventType:
        case PLAYER_DEMAND_RENEWAL:
            # 球员主动要求续约
            generateNews(player, "要求续约")
            addToTodoList(player, "续约要求")

            # 球员威胁离队
            player.morale -= 10
            generateNews(player, "威胁离队")

        case ACCEPT_LOW_SALARY:
            # 仅高忠诚度球员触发
            if player.loyalty < 70: return false
            generateNews(player, "愿意降薪续约")

        case AGENT_PUSH_RENEWAL:
            # 经纪人推动续约
            adjustAgentRelation(player.agentId, +5)

        case RENEWAL_COLLAPSE_LISTED:
            # 续约破裂球员挂牌
            player.isListedForTransfer = true
            generateNews(player, "续约失败被挂牌")

        case CONTRACT_EXPIRED_FREE_TRANSFER:

        case LOANED_PLAYER_RENEWAL:
            # 外租球员续约
            generateNews(player, "外租球员续约")

    return true
```

### 2. 关键交互

- **续约提醒入口**：首页待办事项"续约提醒" → RenewalListScreen；阵容页长按球员"续约" → 直接进入 RenewalNegotiationScreen。
- **要求建议值**：ContractTermsEditorScreen 默认填入球员要求工资（demandsWage），玩家可修改；低于要求则触发还价。
- **意愿条**：RenewalNegotiationScreen 顶部显示续约意愿条，意愿低则提示"球员可能拒绝续约"。
- **多轮谈判**：每轮显示经纪人/球员反应，玩家选择"接受还价/修改后重报/撤回"。
- **预算预览**：BudgetPreviewBar 实时显示工资预算剩余 + 转会预算剩余，超预算时禁用提交按钮。
- **撤回惩罚**：撤回续约时弹窗提示"球员意愿高时撤回将导致士气下降"。

## 七、交互流程

### 1. 续约提醒流程（T07 每日推进触发）

```text
T07 每日推进
    ↓
调用 RenewalService.processDailyCheck(ctx)
    ↓
RenewalReminderService.checkReminders(ctx)
    ↓
遍历本俱乐部所有球员
    ↓
计算合同剩余月数
    ↓
┌─ 剩余 12 个月 ──→ 创建 EARLY_WARNING 提醒
│                   ↓
│                   生成新闻"球员合同将到期"
│
├─ 剩余 6 个月 ───→ 创建 URGENT 提醒
│                   ↓
│                   概率触发球员主动要求续约（PLAYER_DEMAND_RENEWAL）
│                   ↓
│                   生成待办事项 + 新闻
│
├─ 剩余 1 个月 ───→ 创建 FINAL 提醒
│                   ↓
│                   概率触发威胁离队（THREAT_TO_LEAVE）
│                   ↓
│                   球员士气 -10
│
└─ 已到期 ────────→ 调用 ContractExpiryHandler.processExpiry()
                    ↓
                    球员变自由球员 + 加入自由球员池
                    ↓
                    生成新闻"合同到期离队"
```

### 2. 玩家发起续约流程

```text
阵容页长按球员 → 选择"续约"
    ↓
调用 RenewalService.initiateRenewal(ctx, playerId, PLAYER_INITIATED)
    ↓
RenewalManager.initiateRenewal()
    ↓
校验球员属于本俱乐部 + 无进行中续约
    ↓
计算续约要求（RenewalDemandsCalculator）
    ↓
计算续约意愿（RenewalWillingnessCalculator）
    ↓
┌─ 意愿 < 0.30 ──→ 球员拒绝续约（生成新闻）
│                  ↓
│                  玩家可选择挂牌或等待
│
└─ 意愿 >= 0.30 ─→ 创建续约报价（DRAFT）+ 谈判会话
                   ↓
                   进入 ContractTermsEditorScreen
                   ↓
                   玩家编辑合同条款（9 基础 + 7 特殊 + 3 续约特有）
                   ↓
                   提交报价 → RenewalManager.submitRenewalOffer()
                   ↓
                   进入多轮谈判（复用 T11）
```

### 3. 球员主动要求续约流程

```text
T07 每日推进触发 PLAYER_DEMAND_RENEWAL 事件
    ↓
生成待办事项"球员要求续约"
    ↓
玩家点击待办 → 进入 RenewalNegotiationScreen
    ↓
（同玩家发起续约流程，但 initiationType = PLAYER_DEMAND）
    ↓
玩家选择
    ├─ 提交报价 → 多轮谈判
    ├─ 拒绝续约 → 球员士气 -15，可能要求挂牌
    └─ 忽略 → 球员士气 -5，到期前可能威胁离队
```

### 4. 续约谈判流程（多轮）

```text
玩家提交续约报价
    ↓
RenewalManager.submitRenewalOffer()
    ↓
ContractRenegotiator.negotiateRound()
    ↓
经纪人评估（复用 T11 AgentNegotiator）
    ├─ 接受 → 球员评估
    ├─ 不满意且搅局 → 谈判破裂
    └─ 不满意 → 生成经纪人还价
    ↓
球员评估（续约特有评分）
    ├─ 满意 → 续约达成 → RenewalCompleter.complete()
    ├─ 极不满 → 拒绝（结束）
    └─ 中间 → 生成球员还价
    ↓
玩家选择
    ├─ 接受还价 → RenewalCompleter.complete()
    ├─ 修改后重报 → 重新评估（轮次+1，耐心-10）
    └─ 撤回 → 谈判破裂（关系受损）
    ↓
达到最大轮次（5 轮）→ 强制破裂或玩家接受最后还价
```

### 5. 谈判破裂处理流程

```text
触发条件：
- 任一方耐心归零
- 经纪人 walkAway（搅局型 + 满意度极低）
- 玩家主动撤回
- 达到最大轮次未达成

破裂后果：
1. 续约报价状态置为 COLLAPSED 或 WITHDRAWN
2. 球员士气 -10（如玩家主动撤回且球员意愿高）
3. 经纪人关系 -5（如玩家多次撤回）
4. 触发 RENEWAL_COLLAPSE_LISTED 事件（概率 30%）
   → 球员标记为挂牌
5. 球员可能在下一转会窗降低对俱乐部意愿
6. 生成"续约谈判破裂"新闻
7. 如合同剩 < 6 个月，球员可被其他俱乐部接触（预合同）
```

### 6. 合同到期处理流程

```text
T07 每日推进
    ↓
调用 RenewalService.processExpiryCheck(ctx)
    ↓
ContractExpiryHandler.processExpiry()
    ↓
┌─ 合同剩 6 个月 ──→ PreContractHandler.processPreContractContacts()
│                    ↓
│                    计算被接触概率
│                    ↓
│                    概率性触发预合同接触
│                    ↓
│                    生成新闻"球员被其他俱乐部接触"
│                    ↓
│                    玩家收到提醒（需尽快续约或放走）
│
└─ 合同已到期 ────→ FreeAgentReleaser.release()
                    ↓
                    球员 currentClubId = null
                    ↓
                    加入 free_agent_pool 表
                    ↓
                    标记提醒已处理
                    ↓
                    生成新闻"球员合同到期离队"
                    ↓
                    球员可被任何俱乐部签约（T11 自由签约）
```

## 九、性能要求

| 指标 | 目标 | 说明 |
|------|------|------|
| 续约要求计算 | ≤50ms | 单球员 10 因子计算（含 T17 工资查询 + 同位置查询） |
| 续约意愿计算 | ≤100ms | 单球员 8 因子计算（含出场数据 + 经纪人关系查询） |
| 续约谈判回合响应 | ≤500ms | 单轮谈判（经纪人评估 + 球员评估 + 还价生成） |
| 续约提醒检查 | ≤100ms | 全队球员合同检查（T07 每日调用，假设 30 人阵容） |
| 合同到期处理 | ≤200ms | 全队到期检查 + 自由球员释放 |
| 续约完成（含新闻） | ≤500ms | 球员合同更新 + 预算扣减 + 记录写入 + 新闻生成 |

**性能保障措施**：
- 续约要求/意愿在发起续约时计算并缓存到 `ContractRenewalOfferEntity`，避免每轮重算。
- 续约提醒检查走索引（`contract_reminder` 表的 `saveId + clubId + level` 索引）。
- 同位置球员工资查询结果在会话内缓存。
- T17 经济服务查询（expectedWage / financialPower）走 T17 内部缓存。
- 续约完成走单事务，确保球员合同更新 + 预算扣减 + 记录写入原子性。
- 合同到期处理按月度批量执行（T07 月度推进），避免每日全量扫描。

## 十四、风险与应对

### 1. 续约谈判死循环

- **表现**：玩家与球员/经纪人无限还价，无法达成
- **应对**：
  - maxNegotiationRounds = 5，达到上限后强制破裂
  - 每轮扣减耐心（patience -10），耐心归零自动破裂
  - 报价有效期 7 天，过期自动撤回

### 2. 续约要求过高导致工资失控

- **表现**：球员要求工资远超俱乐部承受能力
- **应对**：
  - 续约要求基于 T17 期望工资，受财政修正限制（financeMultiplier 0.75-1.25）
  - 经纪人贪婪修正有上限（GREEDY 型 +0.20）
  - 工资预算校验：年化增量超预算时拒绝提交
  - 与 T17 财政安全线联动：高危俱乐部（>85%）球员要求更高

### 3. 合同到期处理衔接失败

- **表现**：合同到期球员未正确进入自由球员池，T11 无法签约
- **应对**：
  - ContractExpiryHandler 在 T07 每日推进中强制执行
  - FreeAgentReleaser 释放后写入 free_agent_pool 表（T11 可查询）
  - 月度推进批量检查，避免遗漏

### 4. 预合同接触规则错误

- **表现**：合同剩 6 个月但未触发预合同接触，或提前触发
- **应对**：
  - 严格按 V0.1 09 §三 Bosman 规则：合同剩 ≤6 个月才可接触
  - 仅在 monthsLeft == 6 时触发一次（避免重复）
  - 检查 hasActivePreContractContact 避免重复接触

### 5. T11 复用集成阻塞

- **表现**：T11 ContractNegotiator/AgentNegotiator 接口变更导致 T12 失败
- **应对**：
  - 通过 `ContractRenegotiator` 适配层封装 T11 调用，T11 接口变更只影响适配层
  - 定义 `RenewalNegotiationAdapter` 接口隔离，T12 业务代码面向适配层编程
  - T11 联调期间锁定 T11 接口版本（git tag），T12 仅依赖锁定版本
  - 单元测试用 Mock 模拟 T11 行为，T11 未完成时 T12 仍可独立测试
  - 集成阶段开启契约测试（Contract Test），T11 接口破坏性变更立即告警

### 6. 续约意愿公式权重失衡

- **表现**：8 因子公式权重不合理，导致球员普遍拒续约或普遍接受
- **应对**：
  - 权重配置化（renewal_config.json `willingnessWeights`），可热重载调参
  - 单元测试覆盖极端场景（声望 0/100、工资满意度 0/100 等 16 组边界）
  - 回归测试：100 名球员续约意愿分布应在 0.3-0.9 区间内 ≥70%
  - 灰度发布后采集意愿分布数据，偏差 >15% 触发权重校准

### 7. 多轮谈判状态丢失

- **表现**：续约谈判中途中断（玩家退出 App、应用崩溃）后状态丢失
- **应对**：
  - `RenewalSessionEntity` 持久化每轮报价与玩家/经纪人回应
  - 每轮操作后立即写入 save.db（事务保证一致性）
  - 应用重启后 `RenewalManager.resumeSession(sessionId)` 恢复中断点
  - 会话有效期 7 天，超时自动关闭并标记为 `EXPIRED`

### 8. 青训保护条款误判

- **表现**：非青训球员被错误标记为青训，触发保护条款；或青训球员被漏标
- **应对**：
  - 青训身份由 T16 青训学院统一维护（`isHomeGrown` 字段）
  - 续约时仅读取标识，不二次判断
  - 数据校验：青训球员俱乐部 ID 必须与首次注册俱乐部一致
  - 单元测试覆盖纯青训 / 外租回归 / 转会青训 3 类场景

### 9. 性能瓶颈（每日推进卡顿）

- **表现**：合同到期检查导致每日推进 P95 超 3 秒
- **应对**：
  - `ContractExpiryHandler` 仅在月度推进执行（非每日）
  - 索引 `idx_contract_end_date` 加速到期查询
  - AI 俱乐部续约批量异步执行，不阻塞玩家推进
  - 性能监控：续约相关耗时占比每日推进 <10%

### 10. 退役条款误触发

- **表现**：球员状态良好但被强制退役，或退役后仍出现在阵容
- **应对**：
  - 退役条款仅在 `RetirementClauseEntity.enabled == true` 且球员年龄 ≥34 时校验
  - 退役判定综合 CA 下降趋势 + 出场率 + 伤病历史（不只看年龄）
  - 退役后 `FreeAgentReleaser` 立即移出阵容，写入 `retired_players` 表
  - 退役球员不可被任何俱乐部签约（T11 自由签约校验）

## 十六、总结

T12 合同续约是 C 轨合同管理的关键收尾任务，承担"现有球员合同生命周期管理"职责。本方案严格复用 T11 的合同条款系统（9 基础 + 7 特殊）和经纪人系统（5 型 8 属性 7 事件），通过 `ContractRenegotiator` 适配层隔离 T11 接口变更风险。

核心交付物：

1. **续约意愿 8 因子公式**（俱乐部声望 × 0.20 + 当前工资满意度 × 0.20 + 上场机会 × 0.15 + 俱乐部前景 × 0.15 + 联赛吸引力 × 0.10 + 球员野心 × 0.10 + 经纪人关系 × 0.05 + 语言/国家适应 × 0.05）
2. **续约要求 10 因子计算**（期望工资 + 年限 + 签字费 + 经纪人佣金 + 解约金 + 角色承诺 + 特殊条款 + 涨薪条款 + 退役条款 + 青训保护条款）
3. **续约特有 3 项条款**（涨薪条款 / 退役条款 / 青训保护条款）
4. **多轮谈判机制**（≤5 轮，复用 T11 ContractNegotiator + AgentNegotiator）
5. **合同到期处理**（自由球员释放 + Bosman 预合同 + T11 自由签约衔接）

技术约束：

- 包结构 `com.greenDynasty.contract.renewal`
- Room 三库分离（history.db 只读 / save.db 存档 / cache.db 缓存）
- 性能要求：续约意愿计算 ≤50ms / 续约要求计算 ≤50ms / 谈判单轮 ≤500ms / 提醒查询 ≤100ms
- 配置化：所有权重 / 阈值 / 上限 / 修正系数走 renewal_config.json，不写死代码

集成约束：

- T11 复用通过适配层，避免接口变更阻塞
- T17 经济模型通过 EconomyService 接口调用
- T07 每日推进仅在月度触发到期检查
- T16 青训身份由青训学院统一维护

本方案在 W47-W49（3 周）内完成，交付后玩家可完整管理现有球员合同生命周期，并与 T11 自由签约、T13 AI 转会、T16 青训转职业形成完整的合同管理闭环。
