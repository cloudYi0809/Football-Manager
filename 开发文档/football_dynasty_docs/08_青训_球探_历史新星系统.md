# 08. 青训、球探与历史新星系统

这是本游戏最关键的长期玩法系统。

核心目标：

> 让玩家可以在真实球星成名前发现他们、培养他们，并改变他们的职业轨迹。

---

# 一、青训与球探的区别

## 1. 青训学院

来源：俱乐部内部。  
重点：培养自家小将。  
特点：低成本、长期成长、受俱乐部青训设施影响。

## 2. 全球球探

来源：世界各地。  
重点：发现隐藏天才、历史新星、低估球员。  
特点：需要预算、地区知识、球探能力。

## 3. 历史新星池

来源：真实历史球员。  
重点：让罗纳尔多、卡卡、C 罗、梅西等在少年时期按历史时间进入世界。

# 六、真实路径与打断逻辑

如果真实转会开启，球员会按默认路径发展。

但如果玩家提前签下该球员：

1. 标记原真实转会为 interrupted。
2. 取消后续强制转会。
3. 触发相关俱乐部替代转会需求。
4. 触发蝴蝶效应事件。

示例：玩家提前签下 C 罗。

结果：

- Sporting CP 获得不同资金。
- Manchester United 缺少未来边锋。
- Manchester United 可能追求 Robben 或其他边锋。
- Portugal 国家队核心成长路径变化。
- Ballon d'Or 竞争变化。

# 八、验收标准

1. 球探可以派遣到指定地区。
2. 球探任务能随日期推进。
3. 球探报告能逐步解锁信息。
4. 历史新星按年份出现。
5. 玩家提前签下历史新星后，真实路径被打断。
6. 青训能产出随机球员。
7. 青训设施能影响产出质量。
8. 青训球员可以提拔、外租、签约、放弃。
