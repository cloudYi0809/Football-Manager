# 《绿茵王朝：2002》开发交付包说明

本交付包用于安排 Android 单机足球经理模拟游戏开发。项目定位为：**真实历史数据库 + Android 单机 + 足球经理模拟 + 青训球探挖宝 + 历史改写沙盒**。

## 二、开发方式建议

不要一开始做完整 UI。建议顺序：

1. 先把 SQLite 数据库和存档跑通。
2. 再做球员、俱乐部、阵容、赛季推进。
3. 再做比赛模拟。
4. 再做转会、青训、球探。
5. 最后做历史事件、蝴蝶效应、挑战剧本、编辑器。

## 三、项目边界

本项目默认用于个人本地单机娱乐，不做公开上架、不做联网服务、不做商业分发。真实球员、真实队伍、真实图片、真实队徽等素材如使用，应只在个人本地资源包中使用。

## 四、核心一句话
