# 10. 历史事件、蝴蝶效应与挑战剧本

---

# 一、系统目标

本系统让游戏不只是赛程模拟，而是模拟整个足球历史。玩家改变关键节点后，世界应产生连锁反应。

---

# 二、历史事件系统

## 1. 事件类型

- 资本入主
- 豪门衰落
- 财政危机
- 新球场建设
- 青训黄金一代
- 名帅更替
- 巨星伤病
- 联赛规则变化
- 欧战改革
- 转会市场通胀
- 国家队黄金一代
- 更衣室危机

## 2. 事件结构

每个事件包括：

- event_id
- title
- event_type
- trigger_date
- trigger_conditions
- choices
- effects
- related_club_id
- related_player_id
- is_historical

## 3. 触发条件

事件可以通过以下条件触发：

- 日期到达
- 球队财政低于阈值
- 球队排名低于预期
- 球员未转会
- 球员已转会
- 教练下课
- 关键比赛结果改变
- 董事会满意度低

---

# 三、典型历史事件

## 1. 资本入主事件

效果：

- 俱乐部资金暴涨
- 转会预算增加
- 工资预算增加
- AI 引援变激进
- 转会市场通胀

## 2. 豪门老化事件

效果：

- 老将能力下降
- 工资压力上升
- 青训或重建需求上升
- 董事会要求降低平均年龄

## 3. 新球场财政压力

效果：

- 长期收入提升
- 短期债务增加
- 转会预算下降
- 工资控制增强

## 4. 青训黄金一代

效果：

- 青训池生成高潜力球员
- 媒体关注提升
- 豪门挖角风险提升

# 四、蝴蝶效应系统

## 1. 触发条件

- 玩家提前签下历史新星
- 玩家阻止关键转会
- 玩家卖掉某豪门未来核心
- 玩家让某队无法夺冠
- 玩家改变重大伤病结果
- 玩家让名帅提前下课
- 玩家改变国家队核心阵容

## 2. 影响范围

- 转会市场
- 豪门阵容
- 国家队
- 欧冠冠军
- 金球奖
- 球员成长
- 俱乐部财政
- 教练命运
- 媒体评价

## 3. 示例：提前签下 C 罗

原历史路径：Sporting CP → Manchester United。

如果玩家提前签下：

- Manchester United 触发“边锋替代需求”。
- AI 可能转向 Robben、Quaresma 或其他边锋。
- Sporting CP 获得不同转会收入。
- Portugal 国家队核心形成时间改变。
- 未来 Ballon d'Or 候选权重改变。

## 4. 示例：梅西离开巴萨

可能影响：

- 巴萨梦三无法按原历史形成。
- 巴萨加强其他青训核心。
- 阿根廷国家队战术提前改变。
- 玩家俱乐部商业价值暴涨。
- 其他豪门加大南美青训挖掘。

# 五、替代转会逻辑

当真实转会被打断，AI 俱乐部需要寻找替代球员。

替代标准：

- 位置相同
- 年龄接近
- 能力接近
- 潜力接近
- 预算可承受
- 联赛吸引力匹配
- 历史候选名单优先

# 六、挑战剧本系统

## 1. 目标

提供短中期高目标玩法，提高重开率。

## 2. 剧本结构

每个剧本包含：

- 剧本名称
- 起始日期
- 可选球队
- 初始状态
- 限制条件
- 主目标
- 次目标
- 失败条件
- 奖励

## 3. 剧本列表

### 1992：发现外星人

目标：在欧洲球队中提前发现并培养罗纳尔多。

### 2002：银河战舰重建

目标：三年内拿欧冠，同时解决更衣室和老化问题。

### 2003：阿森纳不败挑战

目标：保持联赛不败，并延续王朝。

### 2005：伊斯坦布尔之夜改写

目标：改变欧冠决赛历史。

### 2008：巴萨梦三起点

目标：围绕青训核心建立王朝。

### 2013：后弗格森曼联

目标：避免长期衰落，三年内重夺联赛。

### 2020：疫情财政危机

目标：控制债务，保留核心阵容。

# 七、金球奖与历史荣誉

历史改变后，金球奖也应变化。

金球评分：

```text
金球评分 =
俱乐部冠军 × 0.25
+ 欧冠表现 × 0.20
+ 国家队表现 × 0.15
+ 个人数据 × 0.20
+ 媒体声望 × 0.10
+ 历史影响力 × 0.10
```

---

# 八、验收标准

1. 历史事件能按日期或条件触发。
2. 玩家可选择事件处理方式。
3. 事件能改变预算、士气、转会、阵容等数据。
4. 关键真实转会被打断后，AI 有替代行为。
5. 挑战剧本能开始、进行、成功、失败。
6. 蝴蝶效应不会导致世界模拟中断。
