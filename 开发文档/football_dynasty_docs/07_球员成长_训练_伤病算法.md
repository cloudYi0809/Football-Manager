# 07. 球员成长、训练、伤病与退役算法

---

# 一、球员生命周期

球员生命周期分为：

| 年龄 | 阶段 | 说明 |
|---:|---|---|
| 14-16 | 青训苗子 | 属性波动大，潜力判断不稳定 |
| 17-19 | 青年突破 | 成长最快，需要比赛时间 |
| 20-23 | 一线队成型 | 需要稳定出场与训练 |
| 24-28 | 巅峰形成 | 能力稳定上升 |
| 29-32 | 巅峰维持 | 经验提升，身体开始下降 |
| 33+ | 下滑期 | 身体下降，精神属性保留 |

---

# 二、成长总公式

```text
月度成长值 =
潜力空间 × 0.20
+ 职业态度 × 0.20
+ 出场时间 × 0.20
+ 训练质量 × 0.15
+ 教练能力 × 0.10
+ 联赛强度 × 0.05
+ 士气 × 0.05
+ 导师影响 × 0.05
- 伤病影响 × 0.15
```

其中：

```text
潜力空间 = 当前潜力 - 当前能力
```

潜力空间越大，成长可能性越高。

---

# 三、成长影响因素

## 1. 潜力

潜力不是绝对保证，只代表上限。球员需要通过训练、比赛、职业态度兑现潜力。

## 2. 职业态度

职业态度影响长期成长。

高职业态度：

- 成长稳定
- 不容易迷失
- 伤后恢复更好
- 巅峰期更长

低职业态度：

- 容易训练懈怠
- 成长断档
- 媒体吹捧后下滑
- 更衣室风险高

## 3. 出场时间

年轻球员需要比赛。

- 17-19 岁：适量一线队或租借最重要。
- 20-23 岁：需要稳定首发或轮换。
- 长期坐板凳会浪费潜力。

## 4. 联赛强度

联赛强度越高，成长经验越好，但压力和伤病风险也更高。

## 5. 战术适配

球员在适合的战术中更容易成长。

例如：

- 技术型中场适合控球体系。
- 速度型边锋适合反击体系。
- 高中锋适合边路传中或长传冲吊。

# 四、训练系统

## 1. 全队训练

类型：

- 平衡训练
- 体能训练
- 战术训练
- 进攻训练
- 防守训练
- 控球训练
- 定位球训练
- 恢复训练

## 2. 个人训练

专项：

- 射门
- 盘带
- 速度
- 身体对抗
- 弱脚
- 新位置
- 防守意识
- 传球视野
- 任意球
- 体能恢复

## 3. 训练强度

- 低强度：成长慢，伤病低，体能恢复快。
- 标准：平衡。
- 高强度：成长快，伤病风险高，体能消耗大。
- 恢复型：用于密集赛程。

# 五、导师系统

## 1. 导师条件

导师需要：

- 年龄较大
- 声望较高
- 职业态度较好
- 领导力较高
- 同位置或相近位置最佳

## 2. 导师影响

正面影响：

- 职业态度提升
- 稳定性提升
- 领导力提升
- 战术理解提升
- 更衣室融入提升

负面影响：

- 坏性格传染
- 年轻人不满
- 老将影响派系

## 3. 导师组合例子

- 马尔蒂尼指导年轻中卫。
- 皮尔洛指导年轻中场。
- 劳尔指导年轻前锋。
- 卡恩指导年轻门将。

# 六、伤病系统

## 1. 伤病类型

| 类型 | 时间 | 影响 |
|---|---:|---|
| 轻伤 | 3-7 天 | 轻微体能下降 |
| 肌肉伤 | 2-6 周 | 速度、爆发短期下降 |
| 脚踝伤 | 2-8 周 | 灵活、平衡下降 |
| 膝盖伤 | 2-9 个月 | 长期身体风险 |
| 韧带伤 | 4-12 个月 | 可能影响职业生涯 |
| 疲劳伤 | 1-4 周 | 赛程密集触发 |
| 复发伤 | 不固定 | 与旧伤相关 |

## 2. 伤病概率

```text
伤病概率 =
伤病倾向 × 0.25
+ 当前体能反向值 × 0.20
+ 比赛强度 × 0.15
+ 训练强度 × 0.15
+ 赛程密度 × 0.10
+ 年龄因素 × 0.10
+ 随机因素 × 0.05
```

## 3. 强行复出

强行复出效果：

- 短期可上场。
- 当前能力降低。
- 复发风险上升。
- 球员可能不满或感谢。
- 长期伤病倾向可能上升。

## 4. 手术

手术效果：

- 恢复时间更长。
- 复发风险降低。
- 可能永久损失部分身体属性。

---

# 七、能力下滑

## 1. 身体属性下滑

从 29-32 岁开始根据球员类型逐步下滑。

受影响属性：

- 速度
- 爆发
- 耐力
- 灵活
- 弹跳

## 2. 精神属性保留或提升

老将可能提升：

- 冷静
- 决断
- 领导力
- 站位
- 视野

## 3. 转型机制

球员可以转型：

- 边锋转前腰
- 前锋转影锋
- 中场转后腰
- 边后卫转中卫
- 快速前锋转机会型射手

---

# 八、退役系统

## 1. 退役概率

```text
退役概率 =
年龄因素 × 0.35
+ 当前能力下滑 × 0.20
+ 伤病历史 × 0.20
+ 出场时间不足 × 0.10
+ 职业态度 × -0.10
+ 声望与合同 × 0.05
```

## 2. 退役后去向

- 退隐
- 助教
- 主教练
- 青训教练
- 球探
- 经纪人
- 媒体评论员

## 3. 留队任职

如果玩家与球员关系好，可邀请其进入教练组。

---

# 九、球员状态

每名球员每月更新：

- 当前能力
- 体能
- 士气
- 伤病状态
- 市场价值
- 续约意愿
- 离队倾向
- 成长报告

---

# 十、验收标准

1. 年轻球员在有比赛和训练时能成长。
2. 长期坐板凳的小妖成长受限。
3. 伤病会影响能力和体能。
4. 老将身体下降但精神属性保留。
5. 高职业态度球员成长更稳定。
6. 导师能影响年轻球员隐藏属性。
7. 退役球员可以进入员工系统。
