# T08 伤病系统 - 详细实现方案

> 任务编号：T08
> 周期：W32-W34（3 周）
> 优先级：P0（B 轨医疗系统核心，Gate 2 验收硬指标"赛季伤病 8-15 人次"的承担者）
> 依据：V0.1 `07_球员成长_训练_伤病算法.md` §六 + V0.2 `08_球员成长_训练_伤病细化算法.md` §九/§十 + V0.2 `04_比赛模拟算法详细设计.md` §十
> 铁律：伤病概率 7 因子公式 + 4 类重伤永久影响 + 强行复出风险代价 + 伤病历史持久化
> 定位：决定伤病是否真实影响球员命运，串联比赛引擎、每日推进、球员成长三系统

## 一、业务目标与范围

### 1. 业务目标

- 让伤病成为球员生涯的"真实变量"：天才可能因 ACL 改变命运，老将易伤且恢复慢。
- 让玩家面对真实抉择：强行复出换短期战力 vs. 耐心康复保长期上限。
- 让医疗投入有可见收益：高等级医疗设施缩短恢复期、降低复发概率。
- 串联赛季循环：伤病发生（比赛/训练）→ 每日恢复 → 月度成长跳过 → 长期 CA/PA 影响。

### 2. V1 做清单

- 4 档伤病类型：轻伤 / 中等伤 / 重伤 / 职业威胁伤（覆盖 V0.1 T0Y 文档 4 类重伤 ACL/跟腱/半月板/骨折）。
- 伤病发生机制：比赛中受伤（接触性 + 非接触性）、训练中受伤（过度训练 + 意外）、疲劳累积伤病、旧伤复发、年龄因素。
- 伤病风险计算：7 因子公式（伤病倾向/体能/比赛强度/训练强度/赛程密度/年龄/位置）+ 伤病历史 + 体质属性。
- 伤病恢复机制：自然恢复、医疗设施加速、外部专家治疗、强行复出、年龄/体质影响。
- 伤病影响：能力临时下降、成长停滞（T0Y 月结跳过）、长期伤病永久 CA/PA 下降、职业威胁伤触发退役评估、士气下降、财政影响。
- 医疗中心页：伤病列表、康复进度、疲劳风险、医疗设施、伤病预防 5 模块。
- 4 项操作：选择治疗方案、强行复出（二次确认）、外部专家治疗、调整训练负荷。
- 伤病历史持久化：用于复发评估和球员详情页展示。

### 3. V1 不做清单（推迟）

- 详细康复训练计划定制（V1 仅按治疗方案类型走标准流程）。
- 心理康复辅导（伤病期间心理状态）。
- 营养师/理疗师员工系统对接（V1 用医疗设施等级抽象）。
- 国家队比赛日伤病征召冲突处理。
- 球员伤病倾向的动态隐藏成长（V1 仅在重伤后按规则提升 injury_proneness）。
- 伤病视频回放/动画演出。

### 3. 三库分离策略

| 数据 | 库 | 说明 |
|------|------|------|
| injury_type（伤病类型定义表） | history.db | 静态数据，所有存档共享，含 30+ 种伤病类型 |
| save_injury（当前/历史伤病记录） | save.db | 每存档独立，含已恢复记录用于历史查询 |
| save_medical_facility（医疗设施） | save.db | 每存档独立，玩家可升级 |
| save_treatment_plan（治疗方案） | save.db | 每存档独立，每条伤病对应一条方案 |
| injury_risk_cache（风险缓存） | cache.db | 每日推进前批量计算的伤病风险分数，可重建 |

```kotlin
// model/InjuryTypeEntity.kt
@Entity(tableName = "injury_type")
data class InjuryTypeEntity(
    @PrimaryKey val typeCode: String,            // "STRAIN_HAMSTRING"、"ACL_TEAR"
    val nameCn: String,                           // 中文显示名"腘绳肌拉伤"
    val nameEn: String,                           // 英文名
    val severity: String,                         // MINOR / MODERATE / MAJOR / CAREER_THREATENING
    val category: String,                         // MUSCLE / LIGAMENT / FRACTURE / CONCUSSION / TENDON
    val baseRecoveryDaysMin: Int,                 // 基础恢复天数下限
    val baseRecoveryDaysMax: Int,                 // 基础恢复天数上限
    val recurrenceBaseRisk: Double,               // 基础复发概率 0-1
    val permanentImpactRisk: Double,              // 永久影响概率 0-1（仅重伤及以上）
    val bodyPart: String,                         // 受影响部位 leg/knee/ankle/foot/head/upper_body
    val affectedAttributes: String,               // JSON: ["pace","acceleration"]
    val isContactType: Boolean,                   // true=接触性伤，false=非接触性

```

`injury_type` 表预置 30+ 条记录（见 §九配置表），覆盖 V0.1/T0Y 的 4 类重伤：

| 严重度 | 代表类型 | 恢复天数 | 来源文档 |
|--------|----------|----------|----------|
| MINOR（轻伤 1-7 天） | 拉伤、挫伤、轻微扭伤 | 1-7 | T08 任务书 |
| MODERATE（中等伤 1-4 周） | 肌肉拉伤、中度扭伤、脑震荡 | 7-28 | T08 任务书 |
| MAJOR（重伤 1-6 个月） | 十字韧带撕裂(ACL)、骨折、严重脚踝扭伤、半月板 | 30-180 | T0Y 4 类重伤 |
| CAREER_THREATENING（职业威胁 6 月+） | 跟腱断裂、多次重伤复发 | 180-365+ | T08 任务书 |

## 四、核心类设计

### 1. InjuryService（对外入口）

```kotlin
// api/InjuryService.kt
class InjuryService(
    private val injuryManager: InjuryManager,
    private val injuryGenerator: InjuryGenerator,
    private val riskCalculator: InjuryRiskCalculator,
    private val recoveryCalculator: RecoveryCalculator,
    private val treatmentSelector: TreatmentSelector,
    private val facilityManager: MedicalFacilityManager,
    private val historyService: InjuryHistoryService,
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val config: InjuryConfig
) {

    /**
     * 比赛中受伤判定（T02 比赛引擎回调）
     * 每个比赛事件 tick 后调用，对参与对抗的球员判定
     */
    suspend fun evaluateMatchInjury(
        saveId: Int,
        playerId: Int,
        matchContext: MatchInjuryContext
    ): InjuryEvent? {
        return injuryGenerator.evaluateMatchInjury(saveId, playerId, matchContext)
    }

    /**
     * 训练中受伤判定（T07 TrainingTask 每日调用）
     */
    suspend fun evaluateTrainingInjury(
        saveId: Int,
        playerId: Int,
        trainingIntensity: Int,
        currentDate: LocalDate
    ): InjuryEvent? {
        return injuryGenerator.evaluateTrainingInjury(saveId, playerId, trainingIntensity, currentDate)
    }

    /**
     * 每日伤病恢复推进（T07 InjuryRecoveryTask 调用）
     */
    suspend fun advanceDailyRecovery(
        saveId: Int,
        clubId: Int,
        currentDate: LocalDate
    ): List<InjuryEvent> {
        return injuryManager.advanceDailyRecovery(saveId, clubId, currentDate)
    }

    /**
     * 选择治疗方案（玩家操作）
     */
    suspend fun selectTreatment(
        saveId: Int,
        injuryId: Long,
        treatmentType: TreatmentType,
        currentDate: LocalDate
    ): TreatmentSelectionResult {
        return treatmentSelector.select(saveId, injuryId, treatmentType, currentDate)
    }

    /**
     * 强行复出（玩家操作，需二次确认）
     */
    suspend fun forceReturn(
        saveId: Int,
        injuryId: Long,
        currentDate: LocalDate
    ): ForceReturnResult {
        return injuryManager.forceReturn(saveId, injuryId, currentDate)
    }

    /**
     * 雇佣外部专家（玩家操作）
     */
    suspend fun hireExternalExpert(
        saveId: Int,
        injuryId: Long,
        expertType: String,
        currentDate: LocalDate
    ): TreatmentSelectionResult {
        return treatmentSelector.hireExternalExpert(saveId, injuryId, expertType, currentDate)
    }

    /**
     * 升级医疗设施（玩家操作，T22 董事会预算批准后）
     */
    suspend fun upgradeMedicalFacility(
        saveId: Int,
        clubId: Int,
        targetLevel: Int,
        currentDate: LocalDate
    ): FacilityUpgradeResult {
        return facilityManager.upgrade(saveId, clubId, targetLevel, currentDate)
    }

    /**
     * 查询球员伤病历史（球员详情页/复发评估调用）
     */
    suspend fun getPlayerInjuryHistory(saveId: Int, playerId: Int): List<InjuryEntity> {
        return historyService.getPlayerHistory(saveId, playerId)
```

```kotlin
// generator/InjuryGenerator.kt
class InjuryGenerator(
    private val riskCalculator: InjuryRiskCalculator,
    private val injuryTypeDao: InjuryTypeDao,
    private val saveInjuryDao: SaveInjuryDao,
    private val treatmentPlanDao: TreatmentPlanDao,
    private val historyService: InjuryHistoryService,
    private val playerStateDao: SavePlayerStateDao,
    private val config: InjuryConfig
) {

    /**
     * V0.1 07 §六.2 + V0.2 04 §十 比赛中伤病判定
     * 由 T02 比赛引擎在每个对抗事件后回调
     */
    suspend fun evaluateMatchInjury(
        saveId: Int,
        playerId: Int,
        ctx: MatchInjuryContext
    ): InjuryEvent? {
        val player = playerStateDao.getById(saveId, playerId) ?: return null
        if (player.injuryStatus != "healthy") return null  // 已受伤不再判新伤

        // 1. 计算基础风险分数（0-1）
        val baseRisk = riskCalculator.calculateMatchRisk(saveId, playerId, ctx)

        // 2. 比赛事件强度修正（铲断/头球/冲突）
        val eventModifier = when (ctx.eventType) {
            MatchEventType.TACKLE_HARD -> config.matchEventRiskMultiplier.tackleHard   // 2.0
            MatchEventType.HEADER_DUEL -> config.matchEventRiskMultiplier.headerDuel    // 1.5
            MatchEventType.COLLISION -> config.matchEventRiskMultiplier.collision       // 1.8
            MatchEventType.FATIGUE_LATE -> config.matchEventRiskMultiplier.fatigueLate  // 1.3
            else -> 1.0
        }

        // 3. 强行复出球员比赛中受伤风险翻倍
        val forcedReturnModifier = if (player.isForcedReturn) config.matchEventRiskMultiplier.forcedReturnActive else 1.0

        // 4. 最终概率 = baseRisk * eventModifier * forcedReturnModifier
        val finalProbability = (baseRisk * eventModifier * forcedReturnModifier).coerceIn(0.0, 0.95)

        // 5. 抽样
        if (Random.nextDouble() > finalProbability) return null

        // 6. 决定伤病类型
        val isContact = ctx.eventType != MatchEventType.FATIGUE_LATE
        val injuryType = selectInjuryType(isContact, ctx.matchIntensity)

        // 7. 创建伤病记录
        val injury = createInjuryRecord(
            saveId, playerId, player.currentClubId, injuryType,
            source = if (isContact) InjurySource.MATCH_CONTACT else InjurySource.MATCH_NON_CONTACT,
            currentDate = ctx.matchDate,
            matchId = ctx.matchId,
            matchMinute = ctx.minute
        )

        // 8. 应用到球员状态
        applyInjuryToPlayer(player, injury)

        // 9. 默认创建标准治疗方案
        createDefaultTreatmentPlan(saveId, injury)

        return InjuryEvent(
            injuryId = injury.injuryId,
            playerId = playerId,
            typeCode = injury.typeCode,
            severity = injury.severity,
            source = injury.source,
            expectedReturnDate = injury.expectedReturnDate,
            matchId = ctx.matchId,
            matchMinute = ctx.minute
        )
    }

    /**
     * 训练中受伤判定（T07 TrainingTask 调用）
     * 训练伤病概率低于比赛，但高强度训练显著提升
     */
    suspend fun evaluateTrainingInjury(
        saveId: Int,
        playerId: Int,
        trainingIntensity: Int,   // 1-10
        currentDate: LocalDate
    ): InjuryEvent? {
        val player = playerStateDao.getById(saveId, playerId) ?: return null
        if (player.injuryStatus != "healthy") return null

        // 1. 基础风险（训练日基础概率远低于比赛）
        val baseRisk = riskCalculator.calculateTrainingRisk(saveId, playerId, trainingIntensity, currentDate)

        // 2. 高强度训练翻倍
        val intensityModifier = when {
            trainingIntensity >= 8 -> config.trainingRiskMultiplier.highIntensity   // 2.5
            trainingIntensity >= 6 -> config.trainingRiskMultiplier.mediumIntensity // 1.5
            else -> 1.0
        }

        val finalProbability = (baseRisk * intensityModifier).coerceIn(0.0, 0.30)

        if (Random.nextDouble() > finalProbability) return null

        // 训练伤以非接触性肌肉伤为主
        val injuryType = selectInjuryType(isContact = false, intensity = trainingIntensity)
        val source = if (player.condition < 30) InjurySource.FATIGUE else InjurySource.TRAINING

        val injury = createInjuryRecord(
            saveId, playerId, player.currentClubId, injuryType,
            source = source,
            currentDate = currentDate,
            matchId = null,
            matchMinute = null
        )
        applyInjuryToPlayer(player, injury)
        createDefaultTreatmentPlan(saveId, injury)

        return InjuryEvent(
            injuryId = injury.injuryId, playerId = playerId,
            typeCode = injury.typeCode, severity = injury.severity,
            source = injury.source, expectedReturnDate = injury.expectedReturnDate,
            matchId = null, matchMinute = null
        )
    }

    /**
     * 旧伤复发判定
     * 在每次新伤病发生时，检查球员是否有同一部位的高复发风险旧伤
     */
    suspend fun evaluateRecurrence(
        saveId: Int,
        playerId: Int,
        newInjuryType: InjuryTypeEntity,
        currentDate: LocalDate
    ): Boolean {
        val history = historyService.getRelatedInjuries(saveId, playerId, newInjuryType.bodyPart)
        if (history.isEmpty()) return false

        // 同部位旧伤复发概率 = baseRecurrenceRisk * (1 + recurrenceCount * 0.15)
        val recurrenceCount = history.count { it.isRecurrence }
        val recurrenceProb = newInjuryType.recurrenceBaseRisk * (1.0 + recurrenceCount * 0.15)

        return Random.nextDouble() < recurrenceProb
    }

    /**
     * 按 contact/intensity 抽样伤病类型
     * 高强度比赛重伤概率更高
     */
    private suspend fun selectInjuryType(isContact: Boolean, intensity: Int): InjuryTypeEntity {
        val candidates = injuryTypeDao.getByContactType(isContact)
        if (candidates.isEmpty()) return injuryTypeDao.getByCode(config.defaultInjuryType)!!

        // 重伤权重在高强度下提升
        val adjustedWeights = candidates.map { type ->
            val intensityBonus = when (type.severity) {
                InjurySeverity.CAREER_THREATENING -> if (intensity >= 8) 2.0 else 0.3
                InjurySeverity.MAJOR -> if (intensity >= 7) 1.5 else 0.7
                InjurySeverity.MODERATE -> 1.0
                InjurySeverity.MINOR -> if (intensity <= 5) 1.5 else 0.8
                else -> 1.0
            }
            type to (type.weightInPool * intensityBonus)
        }

        return weightedSample(adjustedWeights)
    }

    private suspend fun createInjuryRecord(
        saveId: Int, playerId: Int, clubId: Int, injuryType: InjuryTypeEntity,
        source: InjurySource, currentDate: LocalDate,
        matchId: Long?, matchMinute: Int?
    ): InjuryEntity {
        // 基础恢复天数在 min-max 间抽样
        val baseDays = Random.nextInt(injuryType.baseRecoveryDaysMin, injuryType.baseRecoveryDaysMax + 1)

        // 检查是否为复发
        val isRecurrence = evaluateRecurrence(saveId, playerId, injuryType, currentDate)
        val sourceInjuryId = if (isRecurrence) {
            historyService.getLatestRelatedInjury(saveId, playerId, injuryType.bodyPart)?.injuryId
        } else null

        // 复发伤恢复时间延长 30%
        val adjustedDays = if (isRecurrence) (baseDays * 1.3).toInt() else baseDays
        val expectedReturn = currentDate.plusDays(adjustedDays.toLong())

        val injury = InjuryEntity(
            saveId = saveId, playerId = playerId, clubId = clubId,
            typeCode = injuryType.typeCode, severity = injuryType.severity,
            source = source.name,
            startDate = currentDate.toString(),
            expectedReturnDate = expectedReturn.toString(),
            actualReturnDate = null,
            recurrenceRisk = (injuryType.recurrenceBaseRisk * 100).toInt(),
            status = InjuryStatus.ACTIVE.name,
            treatmentPlanId = null,
            recoveryProgress = 0,
            recoveryTotalDays = adjustedDays,
            recoveryElapsedDays = 0,
            isForcedReturn = false,
            isRecurrence = isRecurrence,
            sourceInjuryId = sourceInjuryId,
            permanentImpactApplied = false,
            matchId = matchId,
            matchMinute = matchMinute,
            notes = if (isRecurrence) "复发伤（源伤病 ID: $sourceInjuryId）" else null
        )
        val id = saveInjuryDao.insert(injury)
        return injury.copy(injuryId = id)
    }

    private suspend fun applyInjuryToPlayer(player: SavePlayerStateEntity, injury: InjuryEntity) {
        // 更新球员 injury_status 和 injury_until
        playerStateDao.updateInjuryStatus(
            player.id,
            status = "injured",
            injuryUntil = injury.expectedReturnDate,
            typeCode = injury.typeCode
        )
        // 伤病期间体能立即下降
        val conditionPenalty = when (InjurySeverity.valueOf(injury.severity)) {
            InjurySeverity.MINOR -> -10
            InjurySeverity.MODERATE -> -20
            InjurySeverity.MAJOR -> -35
            InjurySeverity.CAREER_THREATENING -> -50
        }
        playerStateDao.updateCondition(player.id, (player.condition + conditionPenalty).coerceAtLeast(0))
    }

    private suspend fun createDefaultTreatmentPlan(saveId: Int, injury: InjuryEntity) {
        // 默认采用标准治疗
        val plan = TreatmentPlanEntity(
            saveId = saveId, injuryId = injury.injuryId, playerId = injury.playerId,
            treatmentType = TreatmentType.STANDARD.name,
            selectedDate = injury.startDate,
            adjustedRecoveryDays = injury.recoveryTotalDays,
            recoverySpeedMultiplier = config.treatment.standardSpeedMultiplier,
            recurrenceRiskModifier = config.treatment.standardRecurrenceModifier,
            permanentImpactRiskModifier = config.treatment.standardPermanentModifier,
            cost = 0, externalExpertName = null, isActive = true
        )
        val planId = treatmentPlanDao.insert(plan)
        saveInjuryDao.update(injury.copy(treatmentPlanId = planId))
    }
}
```

```kotlin
// risk/InjuryRiskCalculator.kt
class InjuryRiskCalculator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val historyService: InjuryHistoryService,
    private val config: InjuryConfig
) {

    /**
     * V0.1 07 §六.2 伤病概率 7 因子公式
     * 伤病概率 =
     *   伤病倾向 × 0.25
     * + 当前体能反向值 × 0.20
     * + 比赛强度 × 0.15
     * + 训练强度 × 0.15
     * + 赛程密度 × 0.10
     * + 年龄因素 × 0.10
     * + 随机因素 × 0.05
     */
    suspend fun calculateMatchRisk(saveId: Int, playerId: Int, ctx: MatchInjuryContext): Double {
        val player = saveDb.savePlayerStateDao().getById(saveId, playerId) ?: return 0.0
        val attrs = historyDb.playerAttributeDao().getLatest(playerId) ?: return 0.0

        val w = config.riskWeights

        // 1. 伤病倾向（injury_proneness 0-100 → 0-1）
        val injuryProneness = attrs.injuryProneness / 100.0

        // 2. 当前体能反向值（体能越低风险越高）
        val conditionInverse = (100 - player.condition) / 100.0

        // 3. 比赛强度（赛事重要性 + 对抗强度，0-1）
        val matchIntensity = ctx.matchIntensity / 10.0

        // 4. 训练强度（最近一次训练强度，0-1）
        val trainingIntensity = (ctx.recentTrainingIntensity ?: 5) / 10.0

        // 5. 赛程密度（最近 14 天比赛数 / 6，0-1）
        val fixtureDensity = calculateFixtureDensity(saveId, player.currentClubId, ctx.matchDate)

        // 6. 年龄因素（见 ageRiskTable）
        val age = calculateAge(playerId)
        val ageFactor = config.ageRiskTable.getRiskFactor(age)

        // 7. 随机因素
        val randomFactor = Random.nextDouble()

        // 基础概率（每场比赛每球员的"被判定机会"基础值，配置化）
        val baseProbability = config.baseMatchInjuryProbability  // 默认 0.025

        val weightedSum = (
            injuryProneness * w.injuryProneness +
            conditionInverse * w.conditionInverse +
            matchIntensity * w.matchIntensity +
            trainingIntensity * w.trainingIntensity +
            fixtureDensity * w.fixtureDensity +
            ageFactor * w.ageFactor +
            randomFactor * w.randomFactor
        )

        // 7 因子和归一化后乘以基础概率
        val normalizedRisk = (weightedSum / w.sum) * baseProbability

        // 8. 伤病历史修正（有重伤史的球员整体风险上浮）
        val historyModifier = calculateHistoryModifier(saveId, playerId)

        // 9. 位置修正（门将风险低，前锋被铲风险高）
        val positionModifier = config.positionRiskModifier[attrs.primaryPosition] ?: 1.0

        return (normalizedRisk * historyModifier * positionModifier).coerceIn(0.0, 0.5)
    }

    /**
     * 训练伤病风险（基础概率远低于比赛）
     */
    suspend fun calculateTrainingRisk(
        saveId: Int, playerId: Int, trainingIntensity: Int, currentDate: LocalDate
    ): Double {
        val player = saveDb.savePlayerStateDao().getById(saveId, playerId) ?: return 0.0
        val attrs = historyDb.playerAttributeDao().getLatest(playerId) ?: return 0.0

        val w = config.riskWeights

        val injuryProneness = attrs.injuryProneness / 100.0
        val conditionInverse = (100 - player.condition) / 100.0
        val trainingFactor = trainingIntensity / 10.0
        val fixtureDensity = calculateFixtureDensity(saveId, player.currentClubId, currentDate)
        val age = calculateAge(playerId)
        val ageFactor = config.ageRiskTable.getRiskFactor(age)
        val randomFactor = Random.nextDouble()

        val baseProbability = config.baseTrainingInjuryProbability  // 默认 0.005

        val weightedSum = (
            injuryProneness * w.injuryProneness +
            conditionInverse * w.conditionInverse +
            trainingFactor * w.trainingIntensity +
            trainingFactor * w.matchIntensity * 0.5 +  // 训练无比赛强度，半折替代
            fixtureDensity * w.fixtureDensity +
            ageFactor * w.ageFactor +
            randomFactor * w.randomFactor
        )

        val normalizedRisk = (weightedSum / w.sum) * baseProbability
        val historyModifier = calculateHistoryModifier(saveId, playerId)

        return (normalizedRisk * historyModifier).coerceIn(0.0, 0.3)
    }

    /**
     * 全队伤病风险评分（医疗中心页加载）
     * 返回每名球员 0-100 的风险评分（用于排序和高亮）
     */
    suspend fun calculateForSquad(saveId: Int, clubId: Int, currentDate: LocalDate): List<InjuryRiskScore> {
        val players = saveDb.savePlayerStateDao().getByClub(saveId, clubId)
        return players.map { player ->
            val attrs = historyDb.playerAttributeDao().getLatest(player.playerId)
            val risk = if (attrs != null && player.injuryStatus == "healthy") {
                calculateSquadRiskScore(player, attrs, currentDate)
            } else 0.0
            InjuryRiskScore(
                playerId = player.playerId,
                playerName = getPlayerName(player.playerId),
                riskScore = (risk * 100).toInt().coerceIn(0, 100),
                riskLevel = classifyRiskLevel(risk),
                mainFactors = explainRiskFactors(player, attrs)
            )
        }.sortedByDescending { it.riskScore }
    }

    private fun classifyRiskLevel(risk: Double): RiskLevel {
        return when {
            risk >= 0.15 -> RiskLevel.CRITICAL
            risk >= 0.08 -> RiskLevel.HIGH
            risk >= 0.04 -> RiskLevel.MEDIUM
            risk > 0 -> RiskLevel.LOW
            else -> RiskLevel.NONE
        }
    }
}
```

```kotlin
// recovery/RecoveryCalculator.kt
class RecoveryCalculator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val facilityManager: MedicalFacilityManager,
    private val config: InjuryConfig
) {

    /**
     * 计算指定伤病的实际恢复速度系数
     * 综合：基础类型 + 医疗设施 + 年龄 + 体质 + 治疗方案
     */
    suspend fun calculateRecoverySpeed(
        saveId: Int, injury: InjuryEntity, currentDate: LocalDate
    ): Double {
        val player = saveDb.savePlayerStateDao().getById(saveId, injury.playerId) ?: return 1.0
        val attrs = historyDb.playerAttributeDao().getLatest(injury.playerId) ?: return 1.0
        val facility = facilityManager.getFacility(saveId, injury.clubId)
        val plan = saveDb.treatmentPlanDao().getActiveByInjury(saveId, injury.injuryId)

        // 1. 医疗设施系数（0.7-1.3）
        val facilityMultiplier = facility?.recoverySpeedMultiplier ?: 1.0

        // 2. 年龄系数（老将恢复慢）
        val age = calculateAge(injury.playerId)
        val ageMultiplier = config.ageRecoveryTable.getMultiplier(age)

        // 3. 体质属性系数（stamina 高恢复快）
        val staminaMultiplier = 0.85 + (attrs.stamina / 100.0) * 0.30  // 0.85-1.15

        // 4. 治疗方案系数
        val treatmentMultiplier = plan?.recoverySpeedMultiplier ?: config.treatment.standardSpeedMultiplier

        // 5. 职业态度系数（高职业态度恢复好）
        val professionalismMultiplier = 0.90 + (attrs.professionalism / 100.0) * 0.20  // 0.90-1.10

        // 6. 复发伤恢复慢 20%
        val recurrencePenalty = if (injury.isRecurrence) 0.80 else 1.0

        return facilityMultiplier * ageMultiplier * staminaMultiplier *
               treatmentMultiplier * professionalismMultiplier * recurrencePenalty
    }

    /**
     * 每日恢复进度推进
     * progress = (elapsedDays / totalDays) * 100
     * totalDays 受恢复速度系数调整
     */
    suspend fun calculateDailyProgress(
        saveId: Int, injury: InjuryEntity, currentDate: LocalDate
    ): RecoveryProgress {
        val speedMultiplier = calculateRecoverySpeed(saveId, injury, currentDate)

        // 调整后总恢复天数（恢复快则总天数缩短）
        val adjustedTotalDays = (injury.recoveryTotalDays / speedMultiplier).toInt().coerceAtLeast(1)

        // 已经过的天数
        val elapsed = ChronoUnit.DAYS.between(
            LocalDate.parse(injury.startDate),
            currentDate
        ).toInt()

        val progress = ((elapsed.toDouble() / adjustedTotalDays) * 100).toInt().coerceIn(0, 100)

        val isReady = progress >= 100
        val remainingDays = (adjustedTotalDays - elapsed).coerceAtLeast(0)

        return RecoveryProgress(
            injuryId = injury.injuryId,
            progress = progress,
            elapsedDays = elapsed,
            totalDays = adjustedTotalDays,
            remainingDays = remainingDays,
            isReady = isReady,
            speedMultiplier = speedMultiplier
        )
    }

    /**
     * 评估恢复完成时的"完全恢复"概率
     * 医疗水平高 → 完全恢复概率高（无复发风险）
     * 医疗水平低 → 部分恢复（体能上限受限 + 复发风险提升）
     */
    suspend fun calculateFullRecoveryChance(saveId: Int, clubId: Int): Double {
        val facility = facilityManager.getFacility(saveId, clubId)
        val level = facility?.medicalLevel ?: 50
        // medicalLevel 50 → 0.85；100 → 0.97；1 → 0.60
        return (0.60 + (level / 100.0) * 0.37).coerceIn(0.60, 0.97)
    }
}
```

```kotlin
// treatment/TreatmentSelector.kt
class TreatmentSelector(
    private val saveDb: SaveDatabase,
    private val recoveryCalculator: RecoveryCalculator,
    private val forceReturnEvaluator: ForceReturnEvaluator,
    private val config: InjuryConfig
) {

    /**
     * 玩家选择治疗方案
     * 4 种方案对恢复速度/复发风险/永久影响有不同修正
     */
    suspend fun select(
        saveId: Int, injuryId: Long, type: TreatmentType, currentDate: LocalDate
    ): TreatmentSelectionResult {
        val injury = saveDb.saveInjuryDao().getById(injuryId) ?: return TreatmentSelectionResult.failed("伤病记录不存在")
        if (injury.status !in listOf("active", "recovering")) {
            return TreatmentSelectionResult.failed("伤病状态不允许选择治疗方案")
        }

        // 验证方案可用性
        val availability = checkAvailability(saveId, injury, type)
        if (!availability.available) return TreatmentSelectionResult.failed(availability.reason)

        // 失活旧方案
        val oldPlan = saveDb.treatmentPlanDao().getActiveByInjury(saveId, injuryId)
        if (oldPlan != null) saveDb.treatmentPlanDao().deactivate(oldPlan.planId)

        // 创建新方案
        val (speedMul, recurMod, permMod, cost) = getTreatmentParams(type, injury.severity)
        val newPlan = TreatmentPlanEntity(
            saveId = saveId, injuryId = injuryId, playerId = injury.playerId,
            treatmentType = type.name,
            selectedDate = currentDate.toString(),
            adjustedRecoveryDays = (injury.recoveryTotalDays / speedMul).toInt(),
            recoverySpeedMultiplier = speedMul,
            recurrenceRiskModifier = recurMod,
            permanentImpactRiskModifier = permMod,
            cost = cost, externalExpertName = null, isActive = true
        )
        val planId = saveDb.treatmentPlanDao().insert(newPlan)

        // 扣费
        if (cost > 0) {
            saveDb.saveClubStateDao().deductBalance(saveId, injury.clubId, cost)
        }

        // 重算预计复出日期
        val speedMultiplier = recoveryCalculator.calculateRecoverySpeed(saveId, injury, currentDate)
        val newTotalDays = (injury.recoveryTotalDays / speedMultiplier).toInt()
        val newReturnDate = LocalDate.parse(injury.startDate).plusDays(newTotalDays.toLong())

        saveDb.saveInjuryDao().update(
            injury.copy(
                treatmentPlanId = planId,
                recoveryTotalDays = newTotalDays,
                expectedReturnDate = newReturnDate.toString(),
                status = "recovering"
            )
        )

        return TreatmentSelectionResult(
            success = true, planId = planId,
            newExpectedReturnDate = newReturnDate.toString(),
            cost = cost, message = "已切换为 ${type.displayName}"
        )
    }

    /**
     * 雇佣外部专家（仅重伤及以上可用）
     */
    suspend fun hireExternalExpert(
        saveId: Int, injuryId: Long, expertType: String, currentDate: LocalDate
    ): TreatmentSelectionResult {
        val injury = saveDb.saveInjuryDao().getById(injuryId) ?: return TreatmentSelectionResult.failed("伤病记录不存在")
        if (injury.severity !in listOf("MAJOR", "CAREER_THREATENING")) {
            return TreatmentSelectionResult.failed("外部专家仅适用于重伤及以上")
        }
        // 外部专家相当于 EXTERNAL_EXPERT 类型，速度更快但有 5% 概率出现"治疗失败"导致永久影响加重
        return select(saveId, injuryId, TreatmentType.EXTERNAL_EXPERT, currentDate)
    }

    private fun getTreatmentParams(type: TreatmentType, severity: String): TreatmentParams {
        val t = config.treatment
        return when (type) {
            TreatmentType.CONSERVATIVE -> TreatmentParams(t.conservativeSpeedMul, t.conservativeRecurMod, t.conservativePermMod, 0)
            TreatmentType.STANDARD -> TreatmentParams(t.standardSpeedMultiplier, t.standardRecurrenceModifier, t.standardPermanentModifier, 0)
```

```kotlin
// treatment/ForceReturnEvaluator.kt
class ForceReturnEvaluator(
    private val saveDb: SaveDatabase,
    private val config: InjuryConfig
) {

    /**
     * V0.1 07 §六.3 强行复出风险评估
     * 返回强行复出的风险详情，供 UI 展示和玩家二次确认
     */
    suspend fun evaluate(saveId: Int, injuryId: Long, currentDate: LocalDate): ForceReturnRisk {
        val injury = saveDb.saveInjuryDao().getById(injuryId) ?: return ForceReturnRisk.invalid()
        if (injury.status !in listOf("active", "recovering")) return ForceReturnRisk.invalid()

        // 当前恢复进度
        val progress = injury.recoveryProgress

        // 1. 复发概率（进度越低越高）
        val recurrenceProbability = calculateRecurrenceProbability(progress, injury.severity)

        // 2. 加重概率（重伤强行复出可能升级为职业威胁伤）
        val aggravationProbability = calculateAggravationProbability(progress, injury.severity)

        // 3. 永久能力下降概率
        val permanentLossProbability = calculatePermanentLossProbability(progress, injury.severity)

        // 4. 比赛中再伤概率翻倍系数
        val matchReinjuryMultiplier = config.forceReturn.matchReinjuryMultiplier  // 默认 2.5

        // 5. 能力临时下降幅度（CA 惩罚）
        val caPenalty = calculateCaPenalty(progress, injury.severity)

        // 6. 球员反应（可能不满）
        val moraleDrop = config.forceReturn.moraleDrop  // 默认 -10

        return ForceReturnRisk(
            valid = true,
            currentProgress = progress,
            recurrenceProbability = recurrenceProbability,
            aggravationProbability = aggravationProbability,
            permanentLossProbability = permanentLossProbability,
            matchReinjuryMultiplier = matchReinjuryMultiplier,
            caPenalty = caPenalty,
            moraleDrop = moraleDrop,
            severity = injury.severity,
            warnings = generateWarnings(recurrenceProbability, aggravationProbability, permanentLossProbability, injury.severity)
        )
    }

    private fun calculateRecurrenceProbability(progress: Int, severity: String): Double {
        val base = config.forceReturn.recurrenceBaseBySeverity[severity] ?: 0.3
        // 进度 100 → 0；进度 0 → base；线性
        return base * (1.0 - progress / 100.0)
    }

    private fun calculateAggravationProbability(progress: Int, severity: String): Double {
        val base = config.forceReturn.aggravationBaseBySeverity[severity] ?: 0.15
        // 进度低于 50% 才有加重风险
        return if (progress < 50) base * (1.0 - progress / 50.0) else 0.0
    }

    private fun calculatePermanentLossProbability(progress: Int, severity: String): Double {
        if (severity !in listOf("MAJOR", "CAREER_THREATENING")) return 0.0
        val base = config.forceReturn.permanentLossBase[severity] ?: 0.1
        return if (progress < 50) base * (1.0 - progress / 50.0) else 0.0
    }

    private fun calculateCaPenalty(progress: Int, severity: String): Int {
        // 进度越低 CA 惩罚越大
        val maxPenalty = config.forceReturn.maxCaPenalty[severity] ?: 10
        return (maxPenalty * (1.0 - progress / 100.0)).toInt()
    }

    private fun generateWarnings(
        recur: Double, aggr: Double, perm: Double, severity: String
    ): List<String> {
        val warnings = mutableListOf<String>()
        if (recur > 0.3) warnings.add("复发概率高（${(recur * 100).toInt()}%），可能延长伤停")
        if (aggr > 0.15) warnings.add("加重风险存在（${(aggr * 100).toInt()}%），可能升级为更严重伤病")
        if (perm > 0.1) warnings.add("永久能力下降风险（${(perm * 100).toInt()}%），可能影响职业生涯")
        if (severity == "CAREER_THREATENING") warnings.add("职业威胁伤强行复出极不推荐")
        return warnings
    }
}
```

```kotlin
// impact/PermanentImpactCalculator.kt
class PermanentImpactCalculator(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val config: InjuryConfig
) {

    /**
     * V0.2 08 §十 重大伤病永久影响
     * ACL: pace -= random(1,4), acceleration -= random(1,4),
     *      injury_proneness += random(3,8), PA -= random(0,5) if age < 24
     * 医疗水平高可降低损失
     */
    suspend fun evaluateAndApply(
        saveId: Int, injury: InjuryEntity, currentDate: LocalDate, medicalLevel: Int
    ): PermanentImpactResult? {
        if (injury.permanentImpactApplied) return null
        if (injury.severity !in listOf("MAJOR", "CAREER_THREATENING")) return null

        // 永久影响基础概率（受治疗方案修正）
        val plan = saveDb.treatmentPlanDao().getActiveByInjury(saveId, injury.injuryId)
        val baseRisk = config.permanentImpact.baseProbability[injury.severity] ?: 0.0
        val modifiedRisk = baseRisk * (plan?.permanentImpactRiskModifier ?: 1.0)

        // 医疗水平降低永久影响概率
        val medicalReduction = (medicalLevel / 100.0) * 0.30  // 100 级降低 30%
        val finalRisk = (modifiedRisk - medicalReduction).coerceAtLeast(0.0)

        if (Random.nextDouble() > finalRisk) {
            // 标记已评估，不产生永久影响
            saveDb.saveInjuryDao().markPermanentImpactEvaluated(injury.injuryId)
            return null
        }

        // 应用永久影响（参考 T0Y MajorInjuryImpact + V0.2 08 §十）
        val impact = when (injury.typeCode) {
            "ACL_TEAR" -> applyAclImpact(saveId, injury, medicalLevel)
            "ACHILLES_RUPTURE" -> applyAchillesImpact(saveId, injury, medicalLevel)
            "MENISCUS_TEAR" -> applyMeniscusImpact(saveId, injury, medicalLevel)
            "FRACTURE" -> applyFractureImpact(saveId, injury, medicalLevel)
            else -> applyGenericMajorImpact(saveId, injury, medicalLevel)
        }

        saveDb.saveInjuryDao().markPermanentImpactApplied(injury.injuryId)
        return impact
    }

    /**
     * V0.2 08 §十 ACL 影响
     * pace -= random(1,4), acceleration -= random(1,4)
     * injury_proneness += random(3,8)
     * PA -= random(0,5) if age < 24
     * 医疗水平每 25 级降低 1 点损失
     */
    private suspend fun applyAclImpact(saveId: Int, injury: InjuryEntity, medicalLevel: Int): PermanentImpactResult {
        val reduction = medicalLevel / 25  // 100 级降低 4 点损失
        val paceDelta = -(Random.nextInt(1, 5) - reduction).coerceAtLeast(0)
        val accelDelta = -(Random.nextInt(1, 5) - reduction).coerceAtLeast(0)
        val pronenessDelta = (Random.nextInt(3, 9) - reduction).coerceAtLeast(1)

        val age = calculateAge(injury.playerId)
        val paDelta = if (age < 24) -(Random.nextInt(0, 6) - reduction).coerceAtLeast(0) else 0

        applyAttributeDeltas(injury.playerId, mapOf(
            "pace" to paceDelta, "acceleration" to accelDelta,
            "injury_proneness" to pronenessDelta
        ))
        if (paDelta < 0) saveDb.savePlayerStateDao().updatePa(injury.playerId, paDelta)

        return PermanentImpactResult(
            playerId = injury.playerId, injuryId = injury.injuryId,
            typeCode = "ACL_TEAR",
            paceDelta = paceDelta, accelerationDelta = accelDelta,
            paDelta = paDelta, injuryPronenessDelta = pronenessDelta,
            description = "ACL 撕裂导致永久影响：速度 ${paceDelta}、爆发 ${accelDelta}、伤病倾向 +${pronenessDelta}" +
                          (if (paDelta < 0) "、潜力 ${paDelta}" else "")
        )
    }

    private suspend fun applyAchillesImpact(saveId: Int, injury: InjuryEntity, medicalLevel: Int): PermanentImpactResult {
        // 跟腱断裂比 ACL 更严重
        val reduction = medicalLevel / 25
        val paceDelta = -(Random.nextInt(2, 7) - reduction).coerceAtLeast(0)
        val accelDelta = -(Random.nextInt(2, 6) - reduction).coerceAtLeast(0)
        val pronenessDelta = (Random.nextInt(5, 11) - reduction).coerceAtLeast(2)
        val age = calculateAge(injury.playerId)
        val paDelta = if (age < 26) -(Random.nextInt(0, 6) - reduction).coerceAtLeast(0) else 0

        applyAttributeDeltas(injury.playerId, mapOf(
            "pace" to paceDelta, "acceleration" to accelDelta,
            "injury_proneness" to pronenessDelta
        ))
        if (paDelta < 0) saveDb.savePlayerStateDao().updatePa(injury.playerId, paDelta)

        return PermanentImpactResult(
            playerId = injury.playerId, injuryId = injury.injuryId,
            typeCode = "ACHILLES_RUPTURE",
            paceDelta = paceDelta, accelerationDelta = accelDelta,
            paDelta = paDelta, injuryPronenessDelta = pronenessDelta,
            description = "跟腱断裂导致永久影响：速度 ${paceDelta}、爆发 ${accelDelta}、伤病倾向 +${pronenessDelta}" +
                          (if (paDelta < 0) "、潜力 ${paDelta}" else "")
        )
    }

    private suspend fun applyMeniscusImpact(saveId: Int, injury: InjuryEntity, medicalLevel: Int): PermanentImpactResult {
        val reduction = medicalLevel / 25
        val paceDelta = -(Random.nextInt(1, 4) - reduction).coerceAtLeast(0)
        val pronenessDelta = (Random.nextInt(2, 5) - reduction).coerceAtLeast(1)
        val age = calculateAge(injury.playerId)
        val paDelta = if (age < 24) -(Random.nextInt(0, 3) - reduction).coerceAtLeast(0) else 0

        applyAttributeDeltas(injury.playerId, mapOf(
            "pace" to paceDelta, "injury_proneness" to pronenessDelta
        ))
        if (paDelta < 0) saveDb.savePlayerStateDao().updatePa(injury.playerId, paDelta)

        return PermanentImpactResult(
            playerId = injury.playerId, injuryId = injury.injuryId,
            typeCode = "MENISCUS_TEAR",
            paceDelta = paceDelta, paDelta = paDelta, injuryPronenessDelta = pronenessDelta,
            description = "半月板撕裂导致永久影响：速度 ${paceDelta}、伤病倾向 +${pronenessDelta}"
        )
    }

    private suspend fun applyFractureImpact(saveId: Int, injury: InjuryEntity, medicalLevel: Int): PermanentImpactResult {
        val reduction = medicalLevel / 25
        val pronenessDelta = (Random.nextInt(1, 4) - reduction).coerceAtLeast(1)
        applyAttributeDeltas(injury.playerId, mapOf("injury_proneness" to pronenessDelta))
        return PermanentImpactResult(
            playerId = injury.playerId, injuryId = injury.injuryId,
            typeCode = "FRACTURE", injuryPronenessDelta = pronenessDelta,
            description = "骨折后伤病倾向 +${pronenessDelta}"
        )
    }

    private suspend fun applyGenericMajorImpact(saveId: Int, injury: InjuryEntity, medicalLevel: Int): PermanentImpactResult {
        val reduction = medicalLevel / 25
        val pronenessDelta = (Random.nextInt(2, 6) - reduction).coerceAtLeast(1)
        applyAttributeDeltas(injury.playerId, mapOf("injury_proneness" to pronenessDelta))
        return PermanentImpactResult(
            playerId = injury.playerId, injuryId = injury.injuryId,
            typeCode = injury.typeCode, injuryPronenessDelta = pronenessDelta,
            description = "重大伤病后伤病倾向 +${pronenessDelta}"
        )
    }

    /**
     * 职业威胁伤触发退役评估
     * 若球员 30+ 且发生职业威胁伤，触发提前退役判定
     */
    suspend fun evaluateCareerThreatenedRetirement(
        saveId: Int, injury: InjuryEntity, currentDate: LocalDate
    ): Boolean {
        if (injury.severity != "CAREER_THREATENING") return false
        val age = calculateAge(injury.playerId)
        if (age < 30) return false

        val retireProbability = config.careerThreateningRetireBaseProbability + (age - 30) * 0.05
        return Random.nextDouble() < retireProbability
    }
}
```

```kotlin
// api/InjuryManager.kt
class InjuryManager(
    private val saveDb: SaveDatabase,
    private val historyDb: HistoryDatabase,
    private val recoveryCalculator: RecoveryCalculator,
    private val permanentImpactCalculator: PermanentImpactCalculator,
    private val forceReturnEvaluator: ForceReturnEvaluator,
    private val facilityManager: MedicalFacilityManager,
    private val config: InjuryConfig
) {

    /**
     * 每日恢复推进（T07 InjuryRecoveryTask 调用）
     * 1. 更新所有进行中伤病的恢复进度
     * 2. 进度达 100% 的触发恢复完成
     * 3. 重伤及以上评估永久影响
     * 4. 触发强行复出球员的复发/加重判定
     */
    suspend fun advanceDailyRecovery(
        saveId: Int, clubId: Int, currentDate: LocalDate
    ): List<InjuryEvent> {
        val events = mutableListOf<InjuryEvent>()
        val activeInjuries = saveDb.saveInjuryDao().getActiveInjuriesByClub(saveId, clubId)

        // 性能优化：批量进度推进（先更新进度，再单独处理完成的）
        val stillActiveIds = mutableListOf<Long>()

        for (injury in activeInjuries) {
            val progress = recoveryCalculator.calculateDailyProgress(saveId, injury, currentDate)

            if (progress.isReady) {
                // 恢复完成
                handleRecoveryComplete(saveId, injury, currentDate, clubId, events)
            } else {
                // 进行中
                saveDb.saveInjuryDao().updateRecoveryProgress(
                    injury.injuryId, progress.progress, progress.elapsedDays,
                    if (injury.status == "active") "recovering" else injury.status
                )
                stillActiveIds.add(injury.injuryId)

                // 强行复出球员每日有复发概率
                if (injury.isForcedReturn && injury.status == "returned_early") {
                    evaluateForcedReturnRecurrence(saveId, injury, currentDate, events)
                }
            }
        }

        return events
    }

    /**
     * 恢复完成处理
     */
    private suspend fun handleRecoveryComplete(
        saveId: Int, injury: InjuryEntity, currentDate: LocalDate,
        clubId: Int, events: MutableList<InjuryEvent>
    ) {
        // 1. 评估永久影响（仅重伤及以上）
        var permanentImpact: PermanentImpactResult? = null
        if (injury.severity in listOf("MAJOR", "CAREER_THREATENING") && !injury.permanentImpactApplied) {
            val facility = facilityManager.getFacility(saveId, clubId)
            val medicalLevel = facility?.medicalLevel ?: 50
            permanentImpact = permanentImpactCalculator.evaluateAndApply(saveId, injury, currentDate, medicalLevel)
            if (permanentImpact != null) events.add(InjuryEvent.permanentImpact(permanentImpact))
        }

        // 2. 评估完全恢复 vs 部分恢复
        val fullRecoveryChance = recoveryCalculator.calculateFullRecoveryChance(saveId, clubId)
        val isFullRecovery = Random.nextDouble() < fullRecoveryChance

        // 3. 更新伤病记录
        saveDb.saveInjuryDao().updateStatus(injury.injuryId, "recovered", currentDate.toString())

        // 4. 更新球员状态
        val player = saveDb.savePlayerStateDao().getById(saveId, injury.playerId)!!
        saveDb.savePlayerStateDao().updateInjuryStatus(player.id, "healthy", null, null)
        val recoveredCondition = if (isFullRecovery) 85 else 70
        saveDb.savePlayerStateDao().updateCondition(player.id, recoveredCondition)

        // 5. 部分恢复标记后续复发风险（写入 recurrence_risk 字段）
        if (!isFullRecovery) {
            val newRecurrenceRisk = (injury.recurrenceRisk + 20).coerceAtMost(100)
            saveDb.saveInjuryDao().update(injury.copy(recurrenceRisk = newRecurrenceRisk))
        }

        // 6. 职业威胁伤触发退役评估
        if (injury.severity == "CAREER_THREATENING") {
            val shouldRetire = permanentImpactCalculator.evaluateCareerThreatenedRetirement(saveId, injury, currentDate)
            if (shouldRetire) {
                saveDb.savePlayerStateDao().updateCareerStatus(player.id, "retiring")
                events.add(InjuryEvent.careerThreatenedRetirement(injury.playerId, injury.injuryId))
            }
        }

        events.add(InjuryEvent.recovered(injury.playerId, injury.injuryId, injury.typeCode, isFullRecovery))
    }

    /**
     * 强行复出每日复发判定
     */
    private suspend fun evaluateForcedReturnRecurrence(
        saveId: Int, injury: InjuryEntity, currentDate: LocalDate, events: MutableList<InjuryEvent>
    ) {
        val dailyRecurrenceRisk = config.forceReturn.dailyRecurrenceRisk  // 默认 0.02
        if (Random.nextDouble() < dailyRecurrenceRisk) {
            // 复发：原伤病重新激活
            saveDb.saveInjuryDao().updateStatus(injury.injuryId, "recurred", null)
            val newInjury = injury.copy(
                injuryId = 0,
                startDate = currentDate.toString(),
                expectedReturnDate = currentDate.plusDays((injury.recoveryTotalDays * 1.5).toLong()).toString(),
                status = "active",
                isRecurrence = true,
                sourceInjuryId = injury.injuryId,
                recoveryProgress = 0,
                recoveryElapsedDays = 0,
                recoveryTotalDays = (injury.recoveryTotalDays * 1.5).toInt(),
                notes = "强行复出导致复发"
            )
            val newId = saveDb.saveInjuryDao().insert(newInjury)

            // 球员状态回到受伤
            val player = saveDb.savePlayerStateDao().getById(saveId, injury.playerId)!!
            saveDb.savePlayerStateDao().updateInjuryStatus(player.id, "injured", newInjury.expectedReturnDate, newInjury.typeCode)

            events.add(InjuryEvent.recurred(injury.playerId, newId, injury.injuryId))
        }
    }

    /**
     * 强行复出执行（玩家操作）
     */
    suspend fun forceReturn(saveId: Int, injuryId: Long, currentDate: LocalDate): ForceReturnResult {
        val injury = saveDb.saveInjuryDao().getById(injuryId) ?: return ForceReturnResult.failed("伤病记录不存在")
        if (injury.status !in listOf("active", "recovering")) {
            return ForceReturnResult.failed("伤病状态不允许强行复出")
        }

        val risk = forceReturnEvaluator.evaluate(saveId, injuryId, currentDate)
        if (!risk.valid) return ForceReturnResult.failed("无法评估风险")

        // 应用 CA 临时下降
        val player = saveDb.savePlayerStateDao().getById(saveId, injury.playerId)!!
        val newCa = (player.currentCa - risk.caPenalty).coerceAtLeast(1)
        saveDb.savePlayerStateDao().updateCa(player.id, newCa)

        // 标记伤病为"强行复出"状态
        saveDb.saveInjuryDao().update(
            injury.copy(
                status = "returned_early",
                isForcedReturn = true,
                actualReturnDate = currentDate.toString()
            )
        )

        // 球员状态回到"健康但带强行复出标记"
        saveDb.savePlayerStateDao().updateInjuryStatus(player.id, "healthy_forced", null, injury.typeCode)

        // 士气下降
        saveDb.savePlayerStateDao().updateMorale(player.id, player.morale + risk.moraleDrop)

        return ForceReturnResult(
            success = true, injuryId = injuryId,
            caPenalty = risk.caPenalty, moraleDrop = risk.moraleDrop,
            recurrenceProbability = risk.recurrenceProbability,
            aggravationProbability = risk.aggravationProbability,
            permanentLossProbability = risk.permanentLossProbability,
            warnings = risk.warnings
        )
    }
}
```

```kotlin
        saveDb.medicalFacilityDao().upgrade(

            calculateRecurrenceReduction(targetLevel),

            config.facility.upgradeCooldownDays
        )

        // 每级成本递增：基础 500k，每级差按 (target^2 - current^2) * 10k
        return (500_000 + (targetLevel * targetLevel - currentLevel * currentLevel) * 10_000).coerceAtLeast(500_000)
    }
```

```text
function calculateMatchInjuryProbability(player, attrs, matchCtx):
    # 7 因子公式
    injury_proneness   = attrs.injury_proneness / 100              # 0-1
    condition_inverse  = (100 - player.condition) / 100            # 0-1
    match_intensity    = matchCtx.intensity / 10                   # 0-1
    training_intensity = matchCtx.recentTrainingIntensity / 10     # 0-1
    fixture_density    = countMatchesLast14Days(club, matchCtx.date) / 6  # 0-1+
    age_factor         = ageRiskTable[calculateAge(player)]        # 0-1
    random_factor      = random()                                  # 0-1

    weights = {injury_proneness: 0.25, condition_inverse: 0.20,
               match_intensity: 0.15, training_intensity: 0.15,
               fixture_density: 0.10, age_factor: 0.10,
               random_factor: 0.05}
    weights_sum = 1.00

    weighted = injury_proneness*0.25 + condition_inverse*0.20 + match_intensity*0.15
             + training_intensity*0.15 + fixture_density*0.10 + age_factor*0.10
             + random_factor*0.05

    base_probability = config.baseMatchInjuryProbability   # 默认 0.025

    normalized_risk = (weighted / weights_sum) * base_probability

    # 修正因子
    history_modifier = 1.0 + majorInjuryHistoryCount * 0.10   # 重伤史每次 +10%
    position_modifier = positionRiskTable[attrs.primary_position]  # 门将 0.4 / 前锋 1.3 等

    final_probability = normalized_risk * history_modifier * position_modifier
    return clamp(final_probability, 0.0, 0.5)
```

### 2. 训练伤病发生概率

```text
function calculateTrainingInjuryProbability(player, attrs, trainingIntensity, date):
    # 同 7 因子，但 base_probability 更低（0.005），match_intensity 用训练强度替代
    base_probability = config.baseTrainingInjuryProbability   # 默认 0.005

    # 高强度训练翻倍
    intensity_modifier = 2.5 if trainingIntensity >= 8 else
                         1.5 if trainingIntensity >= 6 else 1.0

    # 体能极低时（<30）触发疲劳伤
    if player.condition < 30:
        fatigue_modifier = 3.0
        source = FATIGUE
    else:
        fatigue_modifier = 1.0
        source = TRAINING

    final_probability = base_risk * intensity_modifier * fatigue_modifier
    return final_probability, source
```

### 3. 伤病恢复进度计算（每日）

```text
function calculateDailyRecovery(injury, player, attrs, facility, plan, currentDate):
    # 1. 恢复速度系数（5 因子相乘）
    facility_mult      = 0.7 + facility.medicalLevel/100 * 0.6          # 0.7-1.3
    age_mult           = ageRecoveryTable[calculateAge(player)]          # 老将 0.7 / 新人 1.1
    stamina_mult       = 0.85 + attrs.stamina/100 * 0.30                 # 0.85-1.15
    treatment_mult     = plan.recoverySpeedMultiplier                    # 保守 0.9 / 标准 1.0 / 手术 0.7 / 专家 1.3
    professionalism_mult = 0.90 + attrs.professionalism/100 * 0.20       # 0.90-1.10
    recurrence_penalty = 0.80 if injury.isRecurrence else 1.0

    speed = facility_mult * age_mult * stamina_mult * treatment_mult *
            professionalism_mult * recurrence_penalty

    # 2. 调整后总恢复天数
    adjusted_total_days = max(1, round(injury.recoveryTotalDays / speed))

    # 3. 已恢复天数
    elapsed = daysBetween(injury.startDate, currentDate)

    # 4. 进度百分比
    progress = clamp(round(elapsed / adjusted_total_days * 100), 0, 100)

    # 5. 是否完成
    is_ready = (progress >= 100)
    remaining = max(0, adjusted_total_days - elapsed)

    return {progress, elapsed, adjusted_total_days, remaining, is_ready, speed}
```

### 4. 强行复出风险评估

```text
function evaluateForceReturn(injury, currentDate):
    progress = injury.recoveryProgress

    # 1. 复发概率（进度 0 → base，进度 100 → 0）
    recurrence_base = config.forceReturn.recurrenceBaseBySeverity[injury.severity]
    recurrence_prob = recurrence_base * (1 - progress / 100)

    # 2. 加重概率（仅进度 <50%）
    aggravation_base = config.forceReturn.aggravationBaseBySeverity[injury.severity]
    aggravation_prob = aggravation_base * (1 - progress / 50) if progress < 50 else 0

    # 3. 永久能力下降概率（仅重伤及以上，进度 <50%）
    if injury.severity in [MAJOR, CAREER_THREATENING] and progress < 50:
        perm_base = config.forceReturn.permanentLossBase[injury.severity]
        perm_prob = perm_base * (1 - progress / 50)
    else:
        perm_prob = 0

    # 4. CA 临时下降
    max_penalty = config.forceReturn.maxCaPenalty[injury.severity]
    ca_penalty = round(max_penalty * (1 - progress / 100))

    # 5. 比赛中再伤概率翻倍
    match_reinjury_mult = config.forceReturn.matchReinjuryMultiplier   # 2.5

    # 6. 士气下降
    morale_drop = config.forceReturn.moraleDrop   # -10

    return {recurrence_prob, aggravation_prob, perm_prob, ca_penalty, match_reinjury_mult, morale_drop}
```

### 5. 永久能力下降判定（V0.2 08 §十）

```text
function evaluatePermanentImpact(injury, medicalLevel, plan):
    if injury.severity not in [MAJOR, CAREER_THREATENING]:
        return null
    if injury.permanentImpactApplied:
        return null

    # 基础概率
    base_risk = config.permanentImpact.baseProbability[injury.severity]   # MAJOR 0.30 / CAREER 0.60

    # 治疗方案修正
    modified_risk = base_risk * plan.permanentImpactRiskModifier

    # 医疗水平降低概率（100 级降低 30%）
    medical_reduction = medicalLevel / 100 * 0.30
    final_risk = max(0, modified_risk - medical_reduction)

    if random() > final_risk:
        markPermanentImpactEvaluated(injury)
        return null

    # 按伤病类型应用永久影响（参考 V0.2 08 §十 + T0Y §五）
    reduction = medicalLevel / 25   # 医疗每 25 级降低 1 点损失

    switch injury.typeCode:
        case ACL_TEAR:
            pace_delta      = -max(0, randInt(1,4) - reduction)
            accel_delta     = -max(0, randInt(1,4) - reduction)
            proneness_delta = +max(1, randInt(3,8) - reduction)
            pa_delta        = -max(0, randInt(0,5) - reduction) if age < 24 else 0

        case ACHILLES_RUPTURE:
            pace_delta      = -max(0, randInt(2,6) - reduction)
            accel_delta     = -max(0, randInt(2,5) - reduction)
            proneness_delta = +max(2, randInt(5,10) - reduction)
            pa_delta        = -max(0, randInt(0,5) - reduction) if age < 26 else 0

        case MENISCUS_TEAR:
            pace_delta      = -max(0, randInt(1,3) - reduction)
            proneness_delta = +max(1, randInt(2,4) - reduction)
            pa_delta        = -max(0, randInt(0,2) - reduction) if age < 24 else 0

        case FRACTURE:
            proneness_delta = +max(1, randInt(1,3) - reduction)

    applyAttributeDeltas(player, pace_delta, accel_delta, proneness_delta, pa_delta)
    markPermanentImpactApplied(injury)

    # 职业威胁伤 + 30 岁以上 → 触发退役评估
    if injury.severity == CAREER_THREATENING and age >= 30:
        retire_prob = config.careerThreateningRetireBase + (age - 30) * 0.05
        if random() < retire_prob:
            triggerRetirement(player)
```

### 6. 伤病复发概率

```text
function evaluateRecurrence(player, newInjuryType, history):
    # 同部位旧伤列表
    related = history.filter(h -> h.bodyPart == newInjuryType.bodyPart and h.status == RECOVERED)
    if related.empty:
        return false

    # 复发概率 = baseRecurrenceRisk * (1 + recurrenceCount * 0.15)
    recurrence_count = related.count(h -> h.isRecurrence)
    recurrence_prob = newInjuryType.recurrenceBaseRisk * (1 + recurrence_count * 0.15)

    # 强行复出后未完全恢复的旧伤，复发概率翻倍
    if related.any(h -> h.isForcedReturn and not h.isFullRecovery):
        recurrence_prob *= 2.0

    # 医疗水平降低复发概率
    recurrence_prob *= (1 - facility.recurrenceReduction)

    return random() < recurrence_prob
```

## 九、性能要求

### 1. 性能指标（来自任务书 + V0.2 §03 非功能需求）

| 操作 | 目标 | 策略 |
|------|------|------|
| 比赛伤病发生判定（单球员单场） | ≤10 ms | 仅查询缓存的风险分数 + 内存抽样伤病类型 |
| 每日恢复更新（全队） | ≤100 ms | 批量 `batchIncrementProgress` + 单独处理完成项 |
| 医疗中心页加载 | ≤200 ms | Flow 异步订阅 + 风险分数缓存到 cache.db |
| 全队风险评分计算 | ≤150 ms | 仅计算健康球员，避免重复查 attribute |
| 强行复出风险评估 | ≤50 ms | 单伤病查询，纯内存计算 |

### 2. 性能优化策略

- **风险分数缓存**：每日推进前批量计算全队风险分数写入 `cache.injury_risk_cache`，医疗中心页直接读缓存，避免实时查询。cache.db 可重建。
- **批量进度更新**：`SaveInjuryDao.batchIncrementProgress` 一条 SQL 更新所有进行中伤病的进度，避免 N+1。
- **伤病类型预加载**：`InjuryTypeDao.getAll()` 在应用启动时加载到内存 `Map<String, InjuryTypeEntity>`，避免每次伤病发生都查 history.db。
- **Flow 异步订阅**：医疗中心页用 `observeActiveInjuries` 返回 Flow，数据库变更自动推送，无需轮询。
- **活跃范围裁剪**：非活跃范围俱乐部（V0.2 T07）仅做简化伤病判定（按月汇总伤病次数，不逐场判定），减少 90% 计算量。

## 十四、风险与应对

### 1. 伤病人次偏离 8-15 区间（Gate 2 硬指标）

- **表现**：1 赛季伤病人次 <8（伤病太少，玩家无感）或 >15（伤病太多，毁玩家阵容）。
- **应对**：
  - 调 `base_probabilities.baseMatchInjuryProbability`（默认 0.025，可在 0.015-0.040 间微调）。
  - 调 `position_risk_modifier`（门将/前锋风险倍数）。
  - 调 `age_risk_table`（老将风险系数）。
  - W34 D5 跑 1 赛季模拟调参，目标 8-15 人次。
- **红线**：不通过修改随机数种子造假，必须靠参数调优。

### 2. 强行复出风险过低导致玩家滥用

- **表现**：玩家普遍选择强行复出而无需承担后果。
- **应对**：
  - 强行复出每日复发概率 2%（基础），1 个月累计约 45% 复发概率。
  - 复发后恢复时间延长 50%，且复发伤标记为 `isRecurrence`，永久影响概率提升。
  - 比赛中再伤概率 2.5 倍，CA 临时下降 5-20 点。
  - 球员士气下降 10，可能触发更衣室不满（T23 衔接）。
- **红线**：强行复出后 1 个月内有可见的负面后果。

### 3. 永久能力下降过于惩罚玩家

- **表现**：ACL 后球员彻底废掉，玩家失去继续玩的动力。
- **应对**：
  - 永久影响概率受医疗水平降低（100 级降低 30%）。
  - 永久影响幅度受医疗水平降低（每 25 级降低 1 点损失）。
  - 永久影响仅作用于重伤及以上，轻伤/中等伤无永久影响。
  - 永久影响后 PA 下降仅作用于 24 岁以下球员（年轻球员 PA 还有调整空间，老将不降 PA）。
  - 职业态度高的球员恢复更好（professionalism_multiplier 0.90-1.10）。
- **红线**：单次永久影响 pace 下降不超过 8 点，PA 下降不超过 5 点。

### 4. 伤病历史数据膨胀

- **表现**：长程模拟 20 年后 save_injury 表过大，影响存档体积。
- **应对**：
  - 已恢复伤病记录保留 3 个赛季后归档到 `season_archive`（与 T19 衔接）。
  - 归档后 `save_injury.status = 'archived'`，仅保留摘要用于复发评估。
  - 复发评估仅查询同部位最近 3 年的伤病历史，避免全表扫描。
  - 索引 `idx_save_injury_player_status` 加速查询。
- **红线**：20 年存档 save_injury 表不超过 5MB。

### 5. 比赛引擎集成时机不确定

- **表现**：T02 比赛引擎的伤病回调接口未在 W34 前稳定。
- **应对**：
  - W32-W33 使用 Mock 比赛上下文（`MatchInjuryContext` 假数据）独立开发。
  - W34 D4 集成时若 T02 接口未稳定，先用适配器层隔离，T08 核心逻辑不受影响。
  - 集成测试可后置到 Gate 2 验收阶段（W37-W38）。
- **红线**：T08 核心算法（风险计算/恢复/永久影响）不依赖 T02 实现细节，仅依赖回调接口契约。

### 6. 活跃范围裁剪导致 AI 俱乐部伤病失衡

- **表现**：非活跃范围俱乐部伤病统计异常（过多或过少）。
- **应对**：
  - 非活跃范围俱乐部按月汇总伤病次数（基于俱乐部规模 + 随机），不逐场判定。
  - 月度汇总目标 0.5-1.5 人次/月，与活跃范围统计一致。
  - 长程测试（Gate 3）验证 20 年后全联赛伤病分布合理。
